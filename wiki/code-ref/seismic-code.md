# 耐震規範條文速查

**規範條文對應層 — 台灣耐震設計規範 + ACI 318 Chapter 18**
**適用版本：** 結構混凝土建築物韌性設計規範（內政部）+ ACI 318-19 Ch.18

## 耐震設計核心原則

| 原則 | 目標 | 規範依據 |
|------|------|---------|
| **能力設計法** | 確保預期破壞模式（塑鉸於梁端）先於非預期（剪力破壞） | ACI 318 §18 |
| **強柱弱梁** | 柱彎矩容量 ≥ 梁彎矩容量 × 6/5 | ACI 318 §18.7.3 |
| **實際強度放大** | 以 1.25fy（可能強度）計算 Mpr 作為設計基礎 | ACI 318 §18.6.5 |
| **韌性詳細設計** | 密箍圍束、最大配筋率限制、接頭剪力強度 | ACI 318 §18.6–18.8 |

## 特殊矩形框架梁（SMF Beam）

### 縱向配筋限制（ACI 318 §18.6.3）

| 項目 | 規定 |
|------|------|
| 最大配筋率 | $\rho \leq 0.025$（ACI 318-19 §18.6.3.1） |
| 梁端正負彎矩強度比 | $M^+_{n,end} \geq 0.5 M^-_{n,end}$ |
| 梁全長正彎矩容量 | $M^+_n \geq 0.25 \max(M^-_n)$ |
| 最少兩根縱筋貫通梁全長（頂底各） | — |

### 設計剪力 Ve（ACI 318 §18.6.5.2）

$$V_e = \frac{M_{pr,l} + M_{pr,r}}{l_n} \pm \frac{w_u l_n}{2}$$

> $M_{pr}$ 以鋼筋可能降伏強度 $1.25f_y$ 計算（不含 φ 折減）

### 箍筋加密區（ACI 318 §18.6.4）

| 項目 | 規定 |
|------|------|
| 加密區長度（梁端） | $\geq 2h$（梁深的兩倍） |
| 加密區最大間距 | $\min(d/4,\; 8d_b,\; 24d_{sw},\; 300\text{ mm})$ |
| 加密區外最大間距 | $d/2$ |
| 第一道箍筋距支承面 | $\leq 50$ mm |

## 特殊矩形框架柱（SMF Column）

### 強柱弱梁驗算（ACI 318 §18.7.3.1）

$$\sum M_{nc} \geq \frac{6}{5} \sum M_{nb}$$

- $\sum M_{nc}$：節點處各柱彎矩強度之和（考慮軸力影響，取最小 φMn）
- $\sum M_{nb}$：節點處各梁 Mpr（1.25fy 計算）之和

### 圍束箍筋 Ash（ACI 318 §18.7.5.4）

$$A_{sh} \geq \max\!\left(\frac{0.3 s b_c f'_c}{f_{yt}}\!\left[\frac{A_g}{A_{ch}}-1\right],\; \frac{0.09 s b_c f'_c}{f_{yt}}\right)$$

| 符號 | 說明 |
|------|------|
| $b_c$ | 圍束區箍筋包圍的混凝土核心寬度（至箍筋外緣） |
| $A_{ch}$ | 箍筋外緣圍成的面積 |
| $s$ | 箍筋間距 |

### 柱加密區長度（ACI 318 §18.7.5.1）

$$l_o \geq \max(h_c,\; l_u/6,\; 450\text{ mm})$$

### 柱設計剪力（ACI 318 §18.7.6.1）

$$V_e = \frac{M_{pr,top} + M_{pr,bot}}{l_u}$$

## 梁柱接頭（ACI 318 §18.8）

### 水平剪力計算

$$V_{jh} = T_{top} + T_{bot} - V_{col}$$

- 內跨（雙側梁）：$T_{bot}$ 需計入（反向地震時底筋受拉）
- 外接頭：$T_{bot}$ 不計（或需依彎矩方向判斷）
- $T = 1.25 f_y A_s$（可能強度）

### 接頭剪力強度（ACI 318 §18.8.4）

$$\phi V_{jh,n} = \phi \gamma \sqrt{f'_c} A_j, \quad \phi = 0.85$$

| 接頭類型 | $\gamma$ |
|---------|:--------:|
| 四面有梁（內部接頭） | **1.0** |
| 三面有梁（T 形） | **0.75** |
| 兩面有梁（L 形） | **0.50** |

### 有效接頭面積 $A_j$

$$A_j = h_c \times b_j$$

$$b_j = \min\!\left(b_w + h_c,\; b_w + 2 \times \frac{h_c - b_w}{4} \times 2,\; b_c\right)$$

（取梁寬和柱寬的函數，詳見 ACI 318 §18.8.4.3）

## 耐震設計常見考試陷阱速查

| 陷阱 | 正確做法 |
|------|---------|
| 用 φMn 計算 Ve | 應用 **Mpr（1.25fy，無 φ）** |
| 梁端 Vc 未歸零 | 地震主控時（Ve ≥ 0.5Vu 且 Pu < 0.05Ag f'c）$V_c = 0$ |
| γ 係數版本混用 | 確認題目引用規範版本（舊4.0/3.2/2.4 vs 新1.0/0.75/0.50） |
| Ash 只算一個公式 | 必須**取兩公式最大值** |
| 強柱弱梁 6/5 忘記 | $\sum M_{nc} \geq 1.2 \sum M_{nb}$（6/5 = 1.2） |
