# 傾角變位法：公式來源與應用

**method_id:** `slope-deflection-method`
**涵蓋單元：** [[SA-U2-3]]
**分析哲學：** [[displacement-method-philosophy]]（位移法思維）

## 理論基礎

傾角變位法（Slope-Deflection Method, SDM）以**節點轉角**和**側移**為基本未知量，直接建立桿端彎矩與這些位移的關係，聯立求解。相比彯矩分配法的迭代，SDM 直接聯立方程，計算量少。

## 核心公式

### 桿端彎矩公式（標準形）

對於樑 $ij$（$i$ 為左端，$j$ 為右端）：

$$M_{ij} = \frac{4EI}{L}\theta_i + \frac{2EI}{L}\theta_j + M_{FEM,ij} + M_{\delta,ij}$$

$$M_{ji} = \frac{2EI}{L}\theta_i + \frac{4EI}{L}\theta_j + M_{FEM,ji} + M_{\delta,ji}$$

其中：

| 項目 | 符號 | 說明 |
|------|------|------|
| 轉角勁度 | $4EI/L$ | $\theta_i = 1$ rad 時 i 端彯矩 |
| 轉角耦合項 | $2EI/L$ | $\theta_j = 1$ rad 時遠端對 i 端的彯矩貢獻 |
| 固端彯矩 | $M_{FEM}$ | 構件在載重下、假設兩端固定時的端部彯矩 |
| 側移項 | $M_{\delta,ij} = -\frac{6EI}{L^2}\Delta$ | $\Delta$ = 相對側移（j 端相對 i 端） |

### 勁度參數

| 參數 | 公式 | 說明 |
|------|------|------|
| 轉角勁度 | $k_{ij} = \frac{4EI}{L}$ | 標準梁（兩端剛接） |
| 側移勁度 | $k_\delta = \frac{6EI}{L^2}$ | 側移單位值時的彯矩變化 |
| 修正係數（遠端鉸接） | 勁度減 25% → $K = 3EI/L$ | 鉸接端不承受彯矩 |

### 節點轉角與位移的邊界條件

| 約束類型 | 轉角 $\theta$ | 位移 $\Delta$ |
|---------|-------------|------------|
| **固定端** | $\theta = 0$ | $\Delta = 0$ |
| **鉸支承** | $\theta = ?$（未知） | $\Delta = 0$（不允許竖向移動） |
| **滾支承** | $\theta = ?$ | 垂直滾面方向 = 0 |
| **遠懸臂端** | $\theta = ?$ | $\Delta = ?$（完全自由） |

## 標準求解步驟

### Step 1：識別未知數
1. 計算自由度（DOF）= 節點轉角數 + 側移數
2. 依邊界條件減少已知（固定端 $\theta = 0$）
3. 列出所有基本未知量

### Step 2：建立桿端彯矩公式
對每條梁（或柱），代入基本未知量寫出 $M_{ij}$ 與 $M_{ji}$。

### Step 3：節點平衡方程

在每個自由節點 $i$，所有連接梁的端部彯矩之和應等於該節點外加力矩（若有）：

$$\sum M_{\text{meeting at } i} = M_{\text{external at } i}$$

若節點無外加力矩，則 $\sum M = 0$（稱**轉角平衡方程**）。

### Step 4：側移剛架的額外方程（層剪力平衡）

若結構存在側移 $\Delta$，需補充側移項。第 $k$ 層的層剪力平衡：

$$\sum V_k = H_k \quad \text{(外加水平力)}$$

其中梁/柱的剪力與轉角、側移有關：
$$V_{ij} = \frac{6EI}{L^2}(\theta_i + \theta_j) + \frac{12EI}{L^3}\Delta$$

組合得側移修正方程。

### Step 5：聯立求解

將所有方程聯立，求解基本未知量（各節點轉角、側移）。

### Step 6：計算最終桿端彯矩與反力

代入步驟 2 的公式計算各桿端彯矩，再由截面法求內力。

## 常見變體

### 1. 遠端鉸接的修正

若梁 $ij$ 的 $j$ 端為鉸接（不能承受彯矩），則：
- $M_{ji} = 0$（邊界條件）
- 只需寫 $M_{ij} = \frac{3EI}{L}\theta_i + \frac{3EI}{L}\theta_j + M_{FEM,ij} + M_{\delta,ij}$
- 但仍需求 $M_{ij}$（用於 $i$ 端平衡）

### 2. 對稱結構的簡化

對稱結構可取半結構分析：
- 對稱軸上的節點：轉角 $\theta = 0$（因對稱）；側移 $\Delta = 0$
- 軸外節點：按通常方式處理
- **關鍵修正：** 軸上節點相鄰梁的勁度改為 $K = 2EI/L$（半勁度）

### 3. 懸臂梁的處理

懸臂梁端（完全自由）：
- 端部彯矩 $M = M_{\text{external}}$（已知）
- 不需建立轉角平衡方程（轉角自由）
- 直接代入相鄰節點的平衡方程

## 與其他方法的比較

| 項目 | SDM | 彯矩分配 | 矩陣法 |
|------|-----|---------|--------|
| **未知數** | 轉角 + 側移 | 迭代，無明確未知數 | 節點位移 |
| **方程類型** | 直接聯立 | 逐次逼近 | 直接聯立 |
| **側移剛架** | 需補充層剪力方程 | 複雜，多輪反覆 | 系統納入 |
| **手算難度** | 中（聯立方程多） | 易（迭代簡單） | 難（矩陣運算） |
| **電腦適用** | 優 | 差 | 優 |
