# 第一單元：靜定結構分析基礎

**命題大綱：** [[SA-U1]] 靜定結構的判定與分析
**涵蓋主題：** 靜不定度判斷、靜定結構分析、影響線

## 核心公式體系

### 1. 靜定性判斷

#### 平面結構靜定度公式

$$d = r + c - 3 - j$$

| 符號 | 說明 |
|------|------|
| $d$ | 超靜不定度（degree of indeterminacy）；$d > 0$ 靜不定；$d = 0$ 靜定；$d < 0$ 機構 |
| $r$ | 支承反力分量數 |
| $c$ | 內鉸反力分量數（每個鉸 = +1） |
| $j$ | 節點數 |

**應用規則：**
- 鉸支承：$r = 2$（反力方向任意，需水平垂直兩分量，但合力過鉸點）
- 滾支承：$r = 1$（反力垂直滾面）
- 固定端：$r = 3$（反力兩分量+彎矩）
- 內鉸（連接兩桿）：$c = 1$（允許相對轉角，彎矩=0）
- 節點中有 $m$ 根桿件連接：視為複節點，計 $j$ 時該節點貢獻 1，但內鉸數 $c$ 計為 $m-1$

#### 結構穩定性判斷

| 靜定度 $d$ | 穩定性 | 幾何判定 |
|-----------|--------|---------|
| $d > 0$ | 靜不定（穩定超靜定） | 支承充分，無無約束自由度 |
| $d = 0$ | 靜定（邊界穩定） | 支承恰當，恰有3個獨立反力 |
| $d < 0$ | 機構（不穩定） | 支承不足，存在無約束自由度 |
| 特例 | 瞬間機構 | $d = 0$ 但幾何上有瞬間位移（如三支承共線） |

### 2. 靜定結構分析：截面法與力平衡

#### 力與矩平衡方程（平面結構）

$$\sum F_x = 0, \quad \sum F_y = 0, \quad \sum M_O = 0$$

**應用：**
- 求支承反力：對整體結構取矩 → 對支承點取矩
- 求內力：對截面一側分離體平衡

#### 截面內力定義（right-hand rule）

在截面 $x$ 處向右切割，左側分離體受力：

| 內力 | 正向定義 | 物理意義 |
|------|---------|---------|
| $N(x)$ | 軸向指向右（拉） | 軸力（正=拉，負=壓） |
| $V(x)$ | 竪向向上 | 剪力（正=逆時針剪） |
| $M(x)$ | 逆時針（彎矩圖下側受拉） | 彎矩（正方向下側受拉） |

**右側分離體：** 內力方向相反

#### 內力與載重的微分關係

$$\frac{dV}{dx} = -w(x), \quad \frac{dM}{dx} = V(x)$$

**推論：**
- 無載段：$V = const$（SFD 水平），$M$ 線性
- 均布載重：$V$ 線性，$M$ 二次拋物線（拱向載重方向）
- 集中力：$V$ 跳躍（跳躍量 = 集中力大小）
- 集中力矩：$M$ 跳躍（跳躍量 = 力矩大小），$V$ 無變化

### 3. 影響線（Influence Line）

#### 定義與性質

**影響線**：移動單位力沿結構某方向移動，結構任一截面或支承的內力/反力隨單位力位置的變化規律。

| 特性 | 說明 |
|------|------|
| 應用 | 活載（如車輛）通過時的最大/最小內力或反力 |
| 縱軸 | 內力/反力數值 |
| 橫軸 | 單位力作用點位置 |
| 正向約定 | 與靜定分析相同（軸力拉為正，剪力逆時針為正，彎矩下側受拉為正） |

#### 靜定梁的影響線規律

**支承反力影響線：**
- 鉸支承 A 的反力 $R_A$：載重在 A 上時 $R_A = 1$；在 B 上時 $R_A = 0$；線性變化
- 形狀：直線（一次函數）

**內力影響線：**
- 截面 C 的剪力 $V_C$：分段直線（折點在截面位置）
- 截面 C 的彎矩 $M_C$：分段二次拋物線或折線（取決於懸臂/主梁配置）

#### 簡支梁截面C的影響線（例）

假設簡支梁 AB，長 L，求截面 C（距 A 為 $a$）的影響線

**反力影響線：**
$$R_A = \begin{cases} \frac{L-x}{L} & x \in [0, L] \end{cases}$$

$$R_B = \begin{cases} \frac{x}{L} & x \in [0, L] \end{cases}$$

**剪力影響線 $V_C$：**
- 當 $x \in [0, a]$：$V_C = -\frac{L-a}{L}$（常數）
- 當 $x \in (a, L]$：$V_C = \frac{a}{L}$（常數）
- 在 $x = a$ 處跳躍

**彎矩影響線 $M_C$：**
- 當 $x \in [0, a]$：$M_C = \frac{(L-a)x}{L}$（直線，$x=0$ 時 0，$x=a$ 時 $\frac{a(L-a)}{L}$）
- 當 $x \in (a, L]$：$M_C = \frac{a(L-x)}{L}$（直線，$x=a$ 時 $\frac{a(L-a)}{L}$，$x=L$ 時 0）
- 最大值：$x = a$ 時 $M_{C,max} = \frac{a(L-a)}{L}$
