# CNS 1480 規範條文速查

**規範條文對應層 — CNS 1480（混凝土工程設計規範與解說）**
**適用版本：** CNS 1480（2011年版，對應 ACI 318-08）

> **注意：** 台灣高考題目有時引用 CNS 1480，有時直接引用 ACI 318。兩者在公式架構上大致相同，但：
> - **單位制：** CNS 1480 使用 kgf/cm²；ACI 318 使用 MPa（N/mm²）
> - **轉換：** 1 MPa = 10.2 kgf/cm²；√f'c (MPa) ≈ √f'c (kgf/cm²) / 3.19

---

## 單位換算速查

| 公式 | CNS 1480（kgf/cm²） | ACI 318（MPa） |
|------|-------------------|---------------|
| 混凝土抗拉強度 | $f_r = 2\sqrt{f'_c}$ | $f_r = 0.623\sqrt{f'_c}$ |
| 混凝土剪力 Vc | $V_c = \frac{1}{6}\sqrt{f'_c} b_w d$（台灣慣用係數形式同 ACI） | 同左 |
| 衝剪強度（三公式） | $\frac{1}{6}$、$\frac{1}{3}$ 係數（同 ACI 換算後相同） | 同左 |
| 扭力門檻 | $T_u < \phi \frac{\sqrt{f'_c}}{12} \frac{A_{cp}^2}{p_{cp}}$ | 同左 |
| 鋼筋降伏應變 | $\varepsilon_y = f_y/E_s = f_y/2040000$ | $\varepsilon_y = f_y/E_s = f_y/200000$ |

$$E_c = 4270\sqrt{f'_c} \text{ (kgf/cm², 一般重量混凝土)} \approx 15100\sqrt{f'_c} \text{ (MPa)}$$

## 鋼筋錨定長度（CNS 1480 §12）

### 拉力錨定長度

$$l_d = \frac{f_y \psi_t \psi_e \psi_s}{1.1 \lambda \sqrt{f'_c} \cdot (c_b+K_{tr})/d_b} \cdot d_b$$

| 修正係數 | 條件 | 值 |
|---------|------|----|
| $\psi_t$ | 頂層筋（下方混凝土深度 > 300mm） | 1.3 |
| $\psi_t$ | 其他位置 | 1.0 |
| $\psi_e$ | 環氧塗層鋼筋（且 c 或 s 小） | 1.5 |
| $\psi_e$ | 環氧塗層鋼筋（其他） | 1.2 |
| $\psi_e$ | 未塗層 | 1.0 |
| $\psi_s$ | D19 及以下 | 0.8 |
| $\psi_s$ | D22 及以上 | 1.0 |

### 標準彎鉤錨定長度

$$l_{dh} = \frac{0.24 \psi_e f_y}{\lambda \sqrt{f'_c}} d_b \geq \max(8d_b, 150\text{mm})$$

## 梁柱接頭耐震設計（CNS 1480 §21，舊版係數）

> **注意：** CNS 1480 與舊版 ACI 318-95 前的 $\gamma$ 係數為面積的開根號形式，與 ACI 318-08 後不同：

| 接頭類型 | $\gamma$（舊 CNS 1480 / ACI 318-99） | $\gamma$（新 ACI 318-08 後） |
|---------|:----------------------------------:|:--------------------------:|
| 四面有梁（內部） | **4.0** | **1.0** |
| 三面有梁 | **3.2** | **0.75** |
| 兩面有梁（角柱） | **2.4** | **0.50** |

考試題目若引用台灣規範，需確認規範版本再選用正確 γ 係數。

## 預力混凝土斷面應力符號（CNS 1480 §18）

$$f = \frac{P_e}{A_g} \pm \frac{P_e \cdot e \cdot y}{I_g} \pm \frac{M \cdot y}{I_g}$$

> 符號約定：**壓力為正（+），拉力為負（−）**（此為 CNS 1480 慣例；部分題目相反，需注意）

四控制條件（使用性驗算）：
| 條件 | 公式 |
|------|------|
| 施加預力時，頂端（圍束側） | $f_{top,i} \geq -f_{ci}$（不超過容許拉應力） |
| 施加預力時，底端 | $f_{bot,i} \leq 0.60 f'_{ci}$（不超過容許壓應力） |
| 使用載重時，頂端 | $f_{top,s} \leq 0.45 f'_c$ |
| 使用載重時，底端 | $f_{bot,s} \geq -f_t$（不超過容許拉應力） |
