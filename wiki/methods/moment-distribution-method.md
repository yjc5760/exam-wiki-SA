# 方法論：彎矩分配法（Moment Distribution Method）

**method_id：** `moment-distribution-method`
**適用單元：** SA-U2-5 彎矩分配法
**適用題型：** 靜不定連續梁、無側移剛架、有側移剛架（需搭配層剪力方程）
**核心公式：**
- 勁度：$K = 4EI/L$（遠端固定）、$K = 3EI/L$（遠端鉸接）
- 分配因子：$DF_{ij} = K_{ij} / \Sigma K$，同節點各桿 $\Sigma DF = 1$
- 傳遞因子：遠端固定 $COF = 1/2$；遠端鉸接 $COF = 0$
- 桿端彎矩：$M_{final} = M_{FEM} + \Sigma(\text{分配量}) + \Sigma(\text{傳遞量})$

## 步驟摘要

1. 計算各桿固端彎矩（FEM），依載重型式查表（均布 $-wL^2/12$／$+wL^2/12$；集中力於中點 $-PL/8$／$+PL/8$）
2. 計算各節點所有桿件的相對勁度與分配因子（遠端鉸接桿件用 $3EI/L$ 修正勁度以省略後續迭代）
3. 逐節點釋放不平衡彎矩：分配 → 傳遞 → 鎖回，反覆直到不平衡彎矩收斂至可忽略
4. 加總各桿端：FEM + 所有分配量 + 所有傳遞量 = 最終桿端彎矩
5. 若為有側移剛架：須額外用層剪力方程（$\Sigma H = 0$）求側移修正項，與無側移分配結果疊加

## 常見陷阱

- 忘記將遠端鉸接桿件的勁度修正為 $3EI/L$，導致多跑一輪不必要的迭代
- 對稱結構未利用對稱簡化（半結構分析），造成計算量倍增
- 有側移框架漏列層剪力方程，誤用無側移解法

## 適用範圍與限制

僅適用於彎矩為主要未知量、可用逐點釋放方式收斂的靜不定梁/剛架分析；當自由度過多或需要精確代數解（非迭代）時，宜改用矩陣位移法（見 `matrix-displacement-method.md`）。
