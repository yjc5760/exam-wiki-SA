# 題型診斷：鋼筋錨定長度與斷點計算（RC-U2-3）

---

## 辨識關鍵字

`錨定長度` `ld` `搭接長度` `lap splice` `Class A` `Class B` `c值` `Ktr` `ψt` `ψe` `拉力搭接` `壓力搭接` `彎鉤錨定` `斷點計算`

---

## 決策樹

```
看到「錨定」或「搭接」
    │
    ├─ 拉力鋼筋錨定長度 ld（詳細公式）
    │       ld/db = (fy·ψt·ψe·ψs) / (1.1·λ·√f'c) × 1/((cb+Ktr)/db)
    │       限制：(cb+Ktr)/db ≤ 2.5
    │       cb = min(保護層至鋼筋中心, 1/2 × 鋼筋間距)
    │       Ktr = 40Atr/(sn)（橫向鋼筋圍束效果）
    │
    ├─ 拉力鋼筋搭接（Tension Lap Splice）
    │       → Class A：1.0·ld（接觸搭接面積 ≤ 50%，低應力區）
    │         Class B：1.3·ld（一般情況，耐震區幾乎都用 Class B）
    │
    ├─ 耐震搭接位置限制
    │       → 特殊矩形框架柱：搭接只能在中段 1/2 柱高
    │         全部搭接在同一截面 → Class B
    │         50% 交錯搭接 → Class A（非耐震區適用）
    │
    └─ 彎鉤錨定（標準彎鉤）
            → ldh 適用於無法提供直錨長度的情況
              梁柱接頭內縱向梁筋錨定常見
```

## 修正係數速查（kgf/cm² 單位）

| 係數 | 條件 | 值 |
|------|------|----|
| ψt（頂筋修正） | 下方澆置混凝土 > 30 cm | 1.3 |
| ψt | 其他 | 1.0 |
| ψe（塗層修正） | 環氧塗層 | 1.5 或 1.2 |
| ψe | 無塗層 | 1.0 |
| λ（輕質混凝土） | 普通混凝土 | 1.0 |

---

## 常見陷阱

- **cb 取法**：取保護層距離與鋼筋間距一半的較小值，不是只取保護層
- **(cb+Ktr)/db 上限 2.5**：超過 2.5 仍以 2.5 計，不可繼續縮短 ld
- **耐震搭接位置**：特殊矩形框架柱的搭接必須在中段 1/2，這是位置限制不是長度限制
- **Class B vs A**：耐震設計幾乎一律 Class B（1.3ld），不要誤用 Class A
- **Ktr 計算**：n 是搭接面上的鋼筋根數（同一截面），不是總根數
