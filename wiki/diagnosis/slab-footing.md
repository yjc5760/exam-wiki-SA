## 決策樹

```
看到「版設計」或「基腳設計」
    │
    ├─ 版（Slab）
    │   ├─ 單向版（One-Way Slab）
    │   │       → 依 ACI 彎矩係數法求設計彎矩
    │   │         版厚 h ≥ L/20（兩端連續）等規定（撓度控制）
    │   │         溫度收縮鋼筋：ρ ≥ 0.002（Grade 420）
    │   │
    │   ├─ 雙向版（Two-Way Slab，無梁版）
    │   │       → 衝剪（Punching Shear）控制：
    │   │           臨界周長 bo = 4(c + d) 或依柱形狀
    │   │           φVc = φ·min(1/6·√f'c·bo·d,  1/3·√f'c·bo·d,
    │   │                        (αs·d/bo + 1/6)·√f'c·bo·d)
    │   │         若 Vu > φVc → 需剪力鋼筋或加大版厚
    │   │
    │   └─ 角隅配筋（角隅翹起效應）
    │           → 上下兩層，45° 方向，延伸 ℓn/5 範圍
    │
    └─ 基腳（Footing）
        ├─ 獨立基腳（矩形 / 正方形）
        │       → ① 衝剪（雙向剪力）：臨界截面距柱面 d/2
        │             φVc = φ·(1/3)√f'c·bo·d（一般情況）
        │         ② 單向剪力：臨界截面距柱面 d
        │             φVc = φ·(1/6)√f'c·bw·d
        │         ③ 彎矩設計：臨界截面在柱面
        │             Mu = qu × B × (L-c)²/2 × 0.5
        │         → 三項取控制，決定版厚與鋼筋量
        │
        ├─ 樁帽（Pile Cap）
        │       → 軸力 + 偏心：各樁反力 P_i = N/n ± M·y_i/Σy²
        │         衝剪臨界截面：距柱面 d/2
        │         角隅基樁可能穿越臨界截面 → 需特殊處理
        │
        └─ 聯合基腳 / 連續基腳
                → 視為倒置梁，設計彎矩圖 + 剪力圖
```

## 核心公式速查

### 單向版最小厚度（ACI 318 Table 9.3.1.1）

| 端部條件 | 最小 h |
|---------|--------|
| 簡支 | L/20 |
| 一端連續 | L/24 |
| 兩端連續 | L/28 |
| 懸臂 | L/10 |

### 衝剪強度（三公式取小）

$$\phi V_c = \phi \cdot \min\!\left(\frac{\sqrt{f'_c}}{3},\; \frac{\sqrt{f'_c}}{6}\!\left(1+\frac{2}{\beta_c}\right),\; \frac{\sqrt{f'_c}}{6}\!\left(\frac{\alpha_s d}{b_o}+1\right)\right) \cdot b_o d$$

## 常見陷阱

- **衝剪 vs. 單向剪力**：版和基腳都要分別驗算，缺一不可
- **臨界截面位置**：衝剪在柱面外 d/2；單向剪力在柱面外 d；彎矩在柱面
- **基腳淨因數土壓力**：用 `qu = Pu/A`（因數化），不是地基承載力驗算用的服務載重
- **樁帽角隅樁**：偏心大時最外側基樁可能受最大壓力，需單獨驗算
- **雙向版角隅配筋**：常被忽略，ACI 13.3.6 強制要求
