# 預力混凝土題型診斷

**適用：** RC-U4-1~4 預力梁斷面應力、損失、剪力

---

## 快速診斷決策樹

```
拿到預力題
  ├─ 先拉法 or 後拉法？
  │   ├─ 先拉法：彈性縮短損失（多束需修正）、無摩擦損失
  │   └─ 後拉法：摩擦損失 + 錨定滑動損失
  │
  ├─ WSD 使用性應力驗算：
  │   f = P/A ± Pe/I × y ± M/I × y
  │   四個控制點：傳遞階段（上、下）× 使用階段（上、下）
  │
  ├─ 極限彎矩 Mn：
  │   有黏結腱：fps = 近似公式或應變相容
  │   無黏結腱：fps = fse + 70 + f'c/(100ρp)（簡支）
  │
  └─ 開裂彎矩 Mcr：
      Mcr = (fr + fpe)Sb
      fr = 0.62√f'c（MPa）或 2√f'c（kgf/cm²）
```

## 高頻陷阱

- 摩擦損失：α 用弧度；拋物線腱 α ≈ 4e/L（矢高 e，跨度 L）
- 斷面慣性矩：先拉法用淨斷面，後拉法用總斷面（灌漿後）
- fps 公式選擇：題目若未指定方法，優先用 ACI 近似式
