# 撓度/裂縫題型診斷

**適用：** RC-U3-1 梁工作性要求（含撓度、裂縫）

---

## 撓度診斷

```
拿到撓度題
  ├─ Ma > Mcr？（是否開裂）
  │   Mcr = fr·Ig/yt；fr = 2√f'c（kgf/cm²）
  │   ├─ 否（Ma ≤ Mcr）→ 用 Ig 計算，Ie = Ig
  │   └─ 是 → Ie = (Mcr/Ma)³Ig + [1-(Mcr/Ma)³]Icr
  │
  ├─ 計算 Icr（開裂斷面慣性矩）：
  │   轉換斷面法，求中性軸 kd
  │   Icr = b(kd)³/3 + nAs(d-kd)² [+ n·As'(kd-d')²]
  │
  ├─ 即時撓度 δi（載入公式依支承條件）：
  │   簡支均佈：5wL⁴/(384EcIe)
  │   懸臂均佈：wL⁴/(8EcIe)
  │
  └─ 長期附加撓度：δlt = λΔ × δ_sustained
      λΔ = ξ/(1+50ρ')
```

## 裂縫診斷

```
裂縫控制（ACI 318 §24.3）
  最大間距 s = 380(280/fs) - 2.5cc
  或      s = 300(280/fs)
  fs ≈ 2/3 fy（服務載重下鋼筋應力）
  cc = 最外層鋼筋至混凝土表面淨保護層
```
