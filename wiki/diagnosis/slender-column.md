# 題型診斷：細長柱（RC-U1-3）

---

## 辨識關鍵字

`細長柱` `長柱` `放大彎矩` `彎矩放大法` `δns` `δs` `無側移構架` `有側移構架` `細長比` `klu/r` `Cm係數` `臨界挫屈載重Pc` `二階效應` `P-Delta`

---

## 決策樹

```
看到「細長柱」或「放大彎矩」
    │
    ├─ Step 1：判斷是否為細長柱（需要放大）
    │           無側移：klu/r ≤ 34 - 12(M1/M2) → 短柱，跳過
    │           有側移：klu/r ≤ 22 → 短柱，跳過
    │           超過門檻 → 細長柱，需做放大彎矩
    │
    ├─ Step 2：判斷有側移 vs. 無側移
    │           無側移構架（無法水平位移）：使用 δns
    │           有側移構架（可水平位移）：使用 δs
    │
    ├─ 無側移（δns 法）
    │       Mc = δns · M2
    │       δns = Cm / (1 - Pu/0.75Pc) ≥ 1.0
    │       Pc = π²EI / (klu)²
    │       EI = 0.4EcIg / (1 + βd)　（βd = 持續載重彎矩/總彎矩）
    │       Cm = 0.6 + 0.4(M1/M2) ≥ 0.4
    │         ← 注意：有端部側向載重時 Cm = 1.0
    │         ← M1/M2：雙曲度為正，單曲度為負
    │
    └─ 有側移（δs 法）
            Mc = δs · M2
            δs = 1 / (1 - ΣPu / 0.75ΣPc)
            此值對整層所有柱統一計算（需整層資訊）
```

---

## k 值選取（有效長度係數）

| 端部條件 | 理論 k | 設計建議 |
|---------|--------|---------|
| 兩端固定（無側移） | 0.5 | 0.65 |
| 一端固定一端鉸（無側移） | 0.7 | 0.80 |
| 兩端鉸（無側移） | 1.0 | 1.0 |
| 一端固定一端自由（有側移） | 2.0 | 2.1 |

---

## 常見陷阱

- **M1/M2 符號**：雙曲度（反曲點存在）M1/M2 為**正**，使 Cm < 1 → 有利；單曲度為**負** → Cm 較大
- **Cm=1.0 的條件**：端部有側向集中載重（非只有端部彎矩）時一律取 1.0
- **βd 的意義**：持續載重（DL）對潛變折減 EI，不要混淆 βd 與載重組合係數
- **0.75Pc 分母**：ACI 以 0.75 為安全係數，δns < 1.0 時仍取 1.0
- **迴轉半徑 r**：矩形柱 r = 0.3h；圓形柱 r = 0.25D
