# 梁彎矩強度題型診斷

**適用：** RC-U1-1 RC 梁彎矩強度分析與設計

---

## 快速診斷決策樹

```
拿到梁彎矩題
  ├─ 壓力區在哪？
  │   ├─ 正彎矩 T 形梁 → 先假設中性軸在翼板內（a ≤ hf）
  │   │     └─ 若 a > hf → 中性軸在腹板，需分段計算
  │   └─ 負彎矩或矩形梁 → 直接用 a = Asfy/(0.85f'c b)
  │
  ├─ 有無壓力鋼筋？
  │   ├─ 無（單筋梁）→ 直接 Whitney 應力塊
  │   └─ 有（雙筋梁）→ 先驗算壓力筋是否降伏
  │         εs' = (c-d')/c × 0.003 ≥ fy/Es？
  │         ├─ 是 → fs' = fy，Cs = As'(fy - 0.85f'c)
  │         └─ 否 → fs' = Es·εs'，需二次方程求 c
  │
  └─ φ 值怎麼取？
      └─ 算 εt = (dt-c)/c × 0.003
          ├─ εt ≥ 0.005 → φ = 0.9（拉力控制）
          ├─ 0.002 ≤ εt < 0.005 → φ 線性內插
          └─ εt < 0.002 → 壓力控制，φ = 0.65
```

## 高頻陷阱

- T 形梁：翼板有效寬度取最小值（跨度/4、16hf+bw、淨間距+bw）
- 雙排鋼筋：有效深度 d 需重新計算（加權平均形心距）
- 雙筋梁 Cs：需扣除 0.85f'c（壓力筋所在位置的混凝土貢獻已計入 Cc）
