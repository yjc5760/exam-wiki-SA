# 柱 P-M 互制題型診斷

**適用：** RC-U1-2 RC 柱強度分析與設計

---

## 快速診斷決策樹

```
拿到柱設計題
  ├─ 題目給 Pu, Mu → 求 φMn 是否足夠？
  │   └─ 以應變相容法求 c → 驗算各鋼筋應變 → 求 φPn, φMn
  │
  ├─ 題目給斷面 → 畫 P-M 互制圖？
  │   └─ 計算 5 個控制點：
  │       1. 純壓 Pmax = 0.80φ[0.85f'c(Ag-Ast)+Astfy]
  │       2. 平衡點（εt = εy, εc = 0.003）
  │       3. 拉力控制點（εt = 0.005）
  │       4. 純彎（Pu = 0）
  │       5. 純拉 Tmax = -φAstfy
  │
  └─ 關鍵判斷：壓力控制 or 拉力控制？
      └─ 平衡點 cb = 6120/(6120+fy) × d
          ├─ c > cb → 壓力控制，φ = 0.65
          └─ c < cb → 拉力控制，φ = 0.9（或內插）
```

## 高頻陷阱

- 壓力筋 As' 是否降伏需驗算：εs' = (c-d')/c × 0.003 ≥ fy/Es
- 中間排鋼筋：應變按比例計算，不可直接假設降伏
- Pmax 有上限：0.80φPo（箍筋柱）或 0.85φPo（螺旋柱）
