# 剪力/扭力題型診斷

**適用：** RC-U2-1 剪力設計、RC-U2-2 扭力設計

---

## 剪力診斷

```
拿到剪力題
  ├─ 臨界斷面在哪？
  │   ├─ 一般梁：距支承面 d 處
  │   └─ 深梁（ln/d < 4）或集中載重在支承面附近：另有規定
  │
  ├─ 計算 Vc（簡化式）：
  │   Vc = (1/6)√f'c × bw × d
  │   （詳細式考慮 Mu, Nu, Vu）
  │
  └─ 設計 Vs = (Vu/φ - Vc)
      s = Avfy d / Vs
      最大間距：d/2 或 600mm（Vs ≤ (1/3)√f'c bw d 時）
              d/4 或 300mm（Vs 更大時）
```

## 扭力診斷

```
拿到扭力題
  ├─ 先判斷門檻：φTth = φ·(1/12)√f'c·Acp²/pcp
  │   Tu ≤ φTth → 扭力忽略，無需設計扭力鋼筋
  │
  ├─ 計算 Aoh = 箍筋中心線圍合面積（非 Acp！）
  │   poh = 箍筋中心線周長
  │   Ao ≈ 0.85 × Aoh
  │
  ├─ 扭力箍筋：At/s = Tu/(2φAo fyv)
  │
  ├─ 縱向扭力筋：Al = (At/s)·poh·cot²θ
  │
  └─ 最大間距：min(poh/8, 300mm)
```
