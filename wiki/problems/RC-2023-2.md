# RC-2023-2 — T形雙筋梁壓力鋼筋未降伏設計彎矩強度

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第2題
**考年：** 2023（民國112年）
**主分類：** [[RC-U1-1]] RC 梁彎矩強度分析與設計
**設計法：** USD強度設計法
**標籤：** `T形梁` `雙筋梁` `壓力鋼筋未降伏` `應變相容` `雙排配筋` `有效深度計算` `Whitney應力塊` `翼板內中性軸`
**驗證狀態：** ✅ verified

## 題幹摘要

T 形雙筋梁（$b_w = 30$ cm，$b_e = 90$ cm，$h_f = 15$ cm，$h = 70$ cm），8-D32 雙排拉力筋（$A_s = 65.12$ cm²，規範最小覆蓋），壓力筋 $A'_s = 18$ cm²，$f'_c = 280$ kgf/cm²，$f_y = 4200$ kgf/cm²。求 $\phi M_n$。本題同時涉及雙排拉力筋有效深度計算、T 形翼板中性軸判斷，及壓力筋應變相容。

## 核心考點

- 雙排拉力筋有效深度 $d$：逐層計算（保護層 + 箍筋 + D32 半徑 + 層間距/2 + 第二層 D32 半徑）
- 壓力鋼筋降伏假設驗證：假設降伏求 $c$，若 $\varepsilon'_s < \varepsilon_y$ 則改用應變相容
- T 形梁中性軸判斷：$a = \beta_1 c$ 與 $h_f$ 比較（不是 $c$ 與 $h_f$ 比較）
- 壓力筋淨壓力：$f'_{s,\text{net}} = f'_s - 0.85f'_c$（$d' < a$ 時）

## 解題關鍵步驟

1. 計算雙排拉力筋有效深度：$d = h - [\text{保護層}(4) + \text{箍筋}(1) + d_{b,32}/2 + (2.5 + d_{b,32})/2]$
2. 壓力筋位置：$d' = 4 + 1 + D_{s'}/2$（單排壓力筋）
3. 假設壓力筋降伏（$f'_s = f_y$），以力平衡求 $c$；驗算 $\varepsilon'_s = 0.003(c-d')/c \geq \varepsilon_y$（本題未達）
4. 設 $f'_s = E_s \times 0.003(c-d')/c$，代入力平衡，展開為 $c$ 的二次方程
5. 驗算 $a = \beta_1 c$ 與 $h_f$ 的關係（確認中性軸是否在翼板內）
6. 計算 $\varepsilon_t$，確認 $\phi$，求 $\phi M_n$

## 用到的公式

$$d = h - c_c - d_{b,\text{stirrup}} - \frac{n_1 d_{b,1} + s_{clear}/2 + d_{b,2}/2}{1} \quad (\text{雙排加權})$$

$$\varepsilon'_s = \frac{0.003(c - d')}{c}, \quad f'_s = E_s \varepsilon'_s$$

$$f'_{s,\text{net}} = f'_s - 0.85 f'_c \quad (d' < a = \beta_1 c)$$

$$0.85 f'_c b_e a + A'_s f'_{s,\text{net}} = A_s f_y \quad (\text{力平衡，中性軸在翼板內})$$

$$\phi M_n = \phi \!\left[0.85 f'_c b_e a \!\left(d - \frac{a}{2}\right) + A'_s f'_{s,\text{net}} (d - d')\right]$$

## 涉及陷阱

- 雙排拉力筋 $d$ 的計算：需逐層加計規範最小覆蓋、箍筋直徑、鋼筋半徑、層間距，以加權合力位置求 $d$。
- $a$ 與 $h_f$ 的比較：壓縮中性軸深度 $a = \beta_1 c$（不是 $c$）才和 $h_f$ 比。
- 壓力筋淨壓力：$d' < a$ 時需扣除 $0.85f'_c$，避免重複計入混凝土壓力。
- 直接假設壓力筋降伏後若矛盾：先假設，若 $c$ 算出後驗算應變不符，才改用應變相容重解。
