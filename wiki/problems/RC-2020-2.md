# RC-2020-2 — 倒T型梁 d 偏移規則與剪力設計

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第2題
**考年：** 2020（民國109年）
**主分類：** [[RC-U2-1]] RC 剪力強度分析與設計
**設計法：** USD強度設計法
**標籤：** `倒T型梁` `臨界斷面` `d偏移規則` `最大肋筋間距` `最大設計剪力` `Vc簡化公式`
**驗證狀態：** ✅ verified

## 題幹摘要

簡支獨立倒 T 型梁（腹板在上、翼板在下），均佈設計載重 $w_u = 15$ tf/m，跨度 800 cm，$f'_c = 280$ kgf/cm²，$f_y = 4200$ kgf/cm²，腹板寬 $b_w = 35$ cm，全深 $h = 90$ cm，$d = 84$ cm，D13 閉合箍筋。三小題：（一）是否可採 $d$ 偏移；（二）支承位置最大肋筋間距；（三）斷面最大設計剪力。

## 核心考點

- $d$ 偏移規則適用條件：支承提供壓力且載重從梁頂（壓力側）作用時才可 $d$ 偏移
- $V_c$ 簡化公式計算：$V_c = 0.53\sqrt{f'_c} b_w d$，$\phi = 0.75$
- 箍筋間距上限雙門檻：$V_s \leq 2V_c$ 時 $s_\text{max} = d/2$；$V_s > 2V_c$ 時 $s_\text{max} = d/4$
- 斷面最大可承受設計剪力由 $V_{s,\max} = 4V_c$ 控制

## 解題關鍵步驟

1. （一）判斷 $d$ 偏移：支承反力在端部產生壓力，均佈載重施加於頂部（壓力側），故**可採** $d$ 偏移，臨界斷面距支承面 $d = 84$ cm 處
2. 計算 $V_c = 0.53\sqrt{280} \times 35 \times 84 = 26.07$ tf
3. 臨界斷面剪力：$V_u = 60 - 15 \times 0.84 = 47.4$ tf
4. 計算 $V_s = V_u/\phi - V_c = 47.4/0.75 - 26.07 = 37.13$ tf
5. （二）$V_s = 37.13 < 2V_c = 52.15$ tf，故 $s_\text{max,code} = d/2 = 42$ cm；強度需求 $s = A_v f_{yt} d / V_s = 24.1$ cm 控制
6. （三）$V_{u,\max} = \phi(V_c + 4V_c) = 0.75 \times 5 \times 26.07 = 97.8$ tf

## 用到的公式

$$V_c = 0.53\sqrt{f'_c}\, b_w d$$

$$V_s = \frac{V_u}{\phi} - V_c$$

$$s = \frac{A_v f_{yt} d}{V_s}$$

$$s_\text{max} = \begin{cases} d/2 & V_s \leq 2V_c \\ d/4 & V_s > 2V_c \end{cases}$$

$$V_{u,\max} = \phi(V_c + 4V_c) = 5\phi V_c \quad (V_{s,\max} = 4V_c)$$

## 涉及陷阱

- $\phi = 0.75$：台灣 2019 規範（土木 401）剪力用 $\phi = 0.75$，舊版為 0.85，勿混淆。
- $A_v = 2 \times 1.267 = 2.534$ cm²：閉合箍筋一組兩腳，非單腳面積 1.267 cm²。
- 倒 T 型梁 $d$ 偏移例外：若載重從下翼板（拉力側）懸吊入腹板，則不可採 $d$ 偏移，臨界斷面在支承面。
- 間距門檻判斷順序：須先計算 $V_s$，再與 $2V_c$ 比較，確定用 $d/2$ 還是 $d/4$。
