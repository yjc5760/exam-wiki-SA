# RC-2021-4 — 後拉法預力T型組合梁最大容許活載重

## 題幹摘要

後拉法預鑄預力矩形梁與場鑄版組合成 T 型梁，$f'_c = 350$ kgf/cm²，初始預力 $F_i = 300$ tf，有效預力 $F_e = 210$ tf，預鑄梁 $b_w = 30$ cm，$h = 50$ cm，場鑄版厚 12 cm，梁間距 250 cm，跨度 $L = 12$ m，靜載重 1.6 tf/m。求在服務載重容許應力條件下，組合 T 型梁可承載的最大均佈活載重 $w_L$。

## 核心考點

- 有效翼板寬三條件計算（組合 T 型梁）
- 組合斷面形心與慣性矩計算（預鑄梁 + 場鑄版面積矩法）
- 鋼腱偏心量 $e$ 以組合截面形心量起，非預鑄梁形心
- 底纖維拉應力為控制條件：$\sigma_{bot} \leq f_{ts} = 1.6\sqrt{f'_c}$

## 解題關鍵步驟

1. 有效翼板寬：$b_f = \min(L/4,\ b_w + 2 \times 8h_f,\ b_w + s_\text{clear})$
2. 組合斷面形心距底：加權面積矩法求 $\bar{y}_b$（組合截面）
3. 鋼腱偏心：$e = \bar{y}_b - d_p$（$d_p$ 為腱至梁底距離，以組合截面形心量起）
4. 底纖維拉應力控制：$\sigma_{bot} = -F_e/A + F_e e/S_b - M_T/S_b \leq f_{ts}$，解 $M_T$ 上限
5. 頂纖維壓應力控制：$\sigma_{top} = -F_e/A - F_e e/S_t + M_T/S_t \leq f_{cs}$，解 $M_T$ 上限
6. 取較嚴格的 $M_T$，換算 $w_L = (M_T - M_D) \times 8/L^2$

## 用到的公式

$$b_f = \min\!\left(\frac{L}{4},\ b_w + 16h_f,\ b_w + s_\text{clear}\right)$$

$$\sigma_{bot} = -\frac{F_e}{A} + \frac{F_e e}{S_b} - \frac{M_T}{S_b}$$

$$\sigma_{top} = -\frac{F_e}{A} - \frac{F_e e}{S_t} + \frac{M_T}{S_t}$$

$$f_{ts} = 1.6\sqrt{f'_c} \quad (\text{容許拉應力})$$

$$f_{cs} = 0.6 f'_c \quad (\text{容許壓應力})$$

## 涉及陷阱

- 偏心量 $e$ 從組合截面形心量起：不是從預鑄梁形心，差距可達數十公分。
- 底纖維通常控制：大活載重時底纖維承受拉力，是最嚴格條件。
- 有效翼板寬計算：$L/4$ 是總翼板寬（非每側），場鑄版厚度用於 $8h_f$ 計算。
