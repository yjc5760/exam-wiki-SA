# RC-2006-1 — 懸臂雙筋梁撓曲強度控制剪力設計 #3 箍筋最大間距

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第1題
**考年：** 2006（民國95年）
**主分類：** [[RC-U2-1]] RC 剪力強度分析與設計

## 題幹摘要

懸臂梁（L=4.5 m），b=30 cm，h=45 cm，d=37 cm，d'=8 cm，As=40 cm²（頂部張力筋），As'=24 cm²（底部壓力筋），f'c=280 kgf/cm²，fy=4200 kgf/cm²，#3 雙肢箍筋（Av=1.42 cm²）。由撓曲強度控制設計剪力，求最大箍筋間距。壓力鋼筋未降伏（εs'=0.000832 < εy），二次方程解 c=16.30 cm，φMn=44.8 tf·m；wu=44.25 kgf/cm，Vu=19913 kgf，Vs=16704 kgf，最大間距 s=13.2 cm。

## 核心考點

- 雙筋梁先假設壓力筋降伏再驗核：本題 c0=11.07 cm，εs'=0.000832 < εy → 未降伏
- 壓力筋未降伏時須二次方程求 c：力平衡方程整理為 6069c²−26832c−1175040=0
- 懸臂梁設計剪力取固定端面（不做 d 偏移，因無斜壓桿效應）
- 剪力設計：Vs=Vu/φ−Vc，由 s=AvfyD/Vs 求最大間距；需與規範 smax=d/2 比較

## 解題關鍵步驟

1. 假設壓力鋼筋降伏：c0=9.41/0.85=11.07 cm；驗核 εs'=0.003×3.07/11.07=0.000832 < εy → 未降伏
2. 二次方程：6069c²−26832c−1175040=0 → c=16.30 cm；a=13.86 cm，fs'=3116 kgf/cm²
3. φMn=0.9×(98936×30.07+69072×29)=0.9×4978305=44.8 tf·m
4. 懸臂梁：wu=2φMn/L²=2×4480475/202500=44.25 kgf/cm；Vu=wu×L=44.25×450=19913 kgf
5. Vc=0.53√280×30×37=9847 kgf；Vs=19913/0.75−9847=16704 kgf
6. s=AvfyD/Vs=1.42×4200×37/16704=13.2 cm；smax=min(13.2, d/2=18.5)=13.2 cm

## 用到的公式

- 壓力筋應變：$\varepsilon'_s = 0.003(c-d')/c$
- 二次方程力平衡：$A_s f_y = 0.85f'_c\beta_1 c \cdot b + A'_s(f'_s - 0.85f'_c)$
- 標稱彎矩：$M_n = C_c(d-a/2) + C_s(d-d')$
- 混凝土剪力強度：$V_c = 0.53\sqrt{f'_c}\cdot b_w \cdot d$
- 箍筋間距：$s = A_v f_y d / V_s$，$V_s = V_u/\phi - V_c$
- 最大間距（Vs ≤ 閾值時）：$s_{max} = d/2$

## 涉及陷阱

- 壓力鋼筋降伏與否需驗核：假設降伏算出 c0 後代入驗算 εs'，本題未降伏需改用二次方程
- 懸臂梁不能做 d 偏移：固定端剪力已是最大值，無法用 d 偏移減小設計剪力
- Vs 閾值判斷：需確認 Vs 是否超過 2.12√f'c×bw×d 以決定 smax 是否減半
