# RC-2004-5 — 先拉有黏結預力梁求開裂彎矩 Mcr 與彎矩設計強度 φMn

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第5題
**考年：** 2004（民國93年）
**主分類：** [[RC-U4-1]] 預力梁斷面應力分析
**副分類：** [[RC-U4-3]] 預力損失
**設計法：** 混合（WSD+USD）
**標籤：** `先拉法` `有黏結腱` `開裂彎矩Mcr` `fps公式` `Whitney應力塊` `β₁折減` `矩形斷面` `有效預力` `ωp延性驗核`
**驗證狀態：** ✅ verified

## 題幹摘要

先拉預力梁，b=40 cm，h=90 cm，dp=80 cm，Aps=6.16 cm²，fpu=16500 kgf/cm²，fe=0.6fpu=9900 kgf/cm²，f'c=350 kgf/cm²，γp=0.4。求（1）開裂彎矩 Mcr（2）彎矩設計強度 φMn。f'c=350 需折減 β₁=0.80，e=dp-h/2=35 cm，底纖維預壓應力 fce=56.5 kgf/cm²，Mcr=50.7 tf·m；fps=15751 kgf/cm²，ωp=0.087 ≤ 0.288，φMn=66.3 tf·m。

## 核心考點

- 開裂彎矩計算（WSD）：底纖維預壓應力加斷裂模數 fr 即為抵抗開裂的總能力，Mcr=Sb(fce+fr)
- 有效預力底纖維壓應力包含兩項：$f_{ce} = P_e/A + P_e \cdot e/S_b$（軸壓 + 偏心壓縮同方向）
- ACI fps 近似公式：$f_{ps} = f_{pu}(1 - \gamma_p/\beta_1 \cdot \rho_p f_{pu}/f'_c)$，β₁=0.80 不可用 0.85
- ωp 延性驗核：$\omega_p = A_{ps}f_{ps}/(bd_p f'_c) \le 0.36\beta_1$

## 解題關鍵步驟

1. 計算 β₁=0.80（f'c=350），fr=2√350=37.4 kgf/cm²，e=80-45=35 cm
2. 計算有效預力：Pe=6.16×9900=61024 kgf，底纖維壓應力 fce=16.95+39.55=56.5 kgf/cm²
3. 開裂彎矩：Mcr=54000×(56.5+37.4)=5070600 kgf·cm=50.7 tf·m
4. 計算 ρp=6.16/(40×80)=0.001925，fps=16500×(1-0.5×0.001925×47.14)=15751 kgf/cm²
5. 延性驗核：ωp=0.0867 ≤ 0.36×0.80=0.288 ✓
6. a=8.15 cm，φMn=0.9×97026×75.93=66.3 tf·m；驗核 φMn=66.3 ≥ 1.2Mcr=60.8 tf·m ✓

## 用到的公式

- 有效預力底纖維壓應力：$f_{ce} = P_e/A + P_e e/S_b$
- 開裂彎矩：$M_{cr} = S_b(f_{ce} + f_r)$，$f_r = 2.0\sqrt{f'_c}$
- ACI fps 公式：$f_{ps} = f_{pu}\!\left(1 - \dfrac{\gamma_p}{\beta_1}\rho_p\dfrac{f_{pu}}{f'_c}\right)$
- 延性驗核：$\omega_p = A_{ps}f_{ps}/(bd_p f'_c) \le 0.36\beta_1$
- 標稱彎矩：$M_n = A_{ps}f_{ps}(d_p - a/2)$，$a = A_{ps}f_{ps}/(0.85f'_c b)$

## 涉及陷阱

- β₁ 未折減（f'c=350 需 β₁=0.80）：誤用 0.85 會使 fps 略偏高，a 偏差
- 偏心距 e 應為 dp-h/2（鋼腱至形心距離），不是 dp 本身
- 底纖維壓應力兩項均為壓縮（同向相加），不是相減
