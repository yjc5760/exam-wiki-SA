# RC-2022-4 — RC 柱曲率韌性轉角韌性位移韌性評估

## 題幹摘要

RC 柱（80×80 cm，淨高 6.4 m，$f'_c = 280$，$f_y = 4200$，主筋 D32），柱底承受 $P = 270$ tf、$M = 232.5$ tf·m。依線性內插（$P = 180$ 與 360 tf 兩組塑鉸參數）求彎矩塑鉸性質，依 109 年公路橋梁耐震規範公式，評估曲率韌性需求 $\mu_\phi$、轉角韌性需求 $\mu_\theta$、位移韌性需求 $\mu$。

## 核心考點

- 線性內插：$P = 270$ tf 位於 180 與 360 tf 之間，各塑鉸參數內插求值
- 塑鉸長度 $L_p$ 公式的混合單位與下限控制
- 三個韌性需求的層次：曲率（斷面）→ 轉角（構件）→ 位移（整體）
- 懸臂長度 $L = l_u/2$（反曲點在淨高中點，單曲率柱）

## 解題關鍵步驟

1. 確認懸臂長度：$L = l_u/2 = 6.4/2 = 3.2$ m（反曲點在柱淨高中點）
2. 線性內插比例 $= (270-180)/(360-180) = 0.5$，求各塑鉸參數（$\phi_y$、$\phi_u$、$M_y$、$M_u$）
3. 計算 $L_p = 0.08L + 0.0022 d_b f_y \geq 0.0044 d_b f_y$（注意混合單位：$L$ 用 m，$d_b$ mm，$f_y$ MPa）
4. 計算 $\delta_y = \phi_y L^2/3$，$\theta_y = \delta_y/L$
5. 計算 $\delta_u = (M_u/M_y)\delta_y + (\phi_u - \phi_y) L_p (L - 0.5L_p)$
6. 三個韌性比：$\mu_\phi = \phi_u/\phi_y$，$\mu_\theta = \theta_u/\theta_y$，$\mu = \delta_u/\delta_y$

## 用到的公式

$$\text{線性內插：} X = X_1 + (X_2 - X_1)\frac{P - P_1}{P_2 - P_1}$$

$$L_p = 0.08L + 0.0022\,d_b f_y \geq 0.0044\,d_b f_y$$

$$\delta_y = \frac{\phi_y L^2}{3}$$

$$\delta_u = \frac{M_u}{M_y}\delta_y + (\phi_u - \phi_y) L_p \!\left(L - \frac{L_p}{2}\right)$$

$$\mu_\phi = \frac{\phi_u}{\phi_y}, \quad \mu_\theta = \frac{\theta_u}{\theta_y}, \quad \mu = \frac{\delta_u}{\delta_y}$$

## 涉及陷阱

- 懸臂長度 $L = l_u/2$：反曲點在柱淨高中點，$L$ 不是全柱高 $l_u$。
- $L_p$ 公式單位混合：$L$ 用 m，$d_b$ 用 mm，$f_y$ 用 MPa（轉換），確認單位一致。
- 下限控制：$L_p \geq 0.0044 d_b f_y$，確保塑鉸長度不低於最小值。
- 三種韌性層次不同：曲率在斷面層次，轉角在構件層次，位移在整體層次，數值不同。
