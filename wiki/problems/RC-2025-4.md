## 題幹摘要

簡支預力梁（$l = 12$ m，$b = 40$ cm，$h = 120$ cm），$P_i = 200$ tf，有效預力 $P_e = 0.85P_i = 170$ tf，傳遞時 $f'_{ci} = 320$ kgf/cm²，終期 $f'_c = 420$ kgf/cm²，均布活載重 $w_L = 1.5$ tf/m，自重 $w_D = 1.152$ tf/m。求梁中央點鋼腱偏心距的容許範圍（$e_\text{min}$ 到 $e_\text{max}$）。

## 核心考點

- 四個應力控制條件（兩個時間點 × 兩個纖維）決定偏心距範圍
- 傳遞時（$P_i$，僅靜載重 $M_D$）：頂纖維拉力 $\leq f_{ti}$（上界），底纖維壓力 $\leq f_{ci}$（上界）
- 使用時（$P_e$，$M_D + M_L$）：底纖維拉力 $\leq f_{ts}$（下界），頂纖維壓力 $\leq f_{cs}$（下界）
- 四條件解出 $e$ 的不等式，取交集確定可行 $e$ 範圍

## 解題關鍵步驟

1. 斷面幾何：$A = b \times h = 4800$ cm²；$I = bh^3/12$；$S_t = S_b = I/(h/2) = bh^2/6$（對稱矩形）
2. 容許應力計算：$f_{ti} = 0.25\sqrt{f'_{ci}}$，$f_{ci} = 0.6f'_{ci}$，$f_{ts} = 0.5\sqrt{f'_c}$，$f_{cs} = 0.45f'_c$
3. 各階段彎矩：$M_D = w_D l^2/8$，$M_T = (w_D + w_L)l^2/8$
4. 由四條件各解 $e$ 的不等式：傳遞時兩條件給 $e$ 上界，使用時兩條件給 $e$ 下界
5. 取四個條件的交集：$e_\text{min} \leq e \leq e_\text{max}$

## 用到的公式

$$\sigma_{top} = -\frac{P}{A} - \frac{Pe}{S_t} + \frac{M}{S_t}$$

$$\sigma_{bot} = -\frac{P}{A} + \frac{Pe}{S_b} - \frac{M}{S_b}$$

$$f_{ti} = 0.25\sqrt{f'_{ci}}, \quad f_{ci} = 0.6 f'_{ci}$$

$$f_{ts} = 0.5\sqrt{f'_c}, \quad f_{cs} = 0.45 f'_c$$

$$S_t = S_b = \frac{bh^2}{6} \quad (\text{對稱矩形斷面})$$

## 涉及陷阱

- 傳遞時用 $P_i$，使用時用 $P_e$：兩個時間點預力不同，容易混淆。
- 傳遞時混凝土強度 $f'_{ci}$：非終期 $f'_c$，傳遞時混凝土尚未充分硬化，容許值不同。
- 對稱矩形斷面：$S_t = S_b = bh^2/6$，非 $I/(h)$（勿漏除以 2）。
- 四個條件各對應不同時間點和纖維：分類為 $e$ 上界或下界後取交集，不可直接取任一條件。
