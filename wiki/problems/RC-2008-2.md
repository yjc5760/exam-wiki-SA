# RC-2008-2 — 56×56 cm 柱拉力鋼筋應變為零特殊條件求 Mn=68.2 tf·m

## 題幹摘要

56×56 cm 方形柱，8-#8，f'c=350 kgf/cm²，fy=4200 kgf/cm²，三排配筋（d1=8, d2=28, d3=48 cm）。當拉力側鋼筋（d3=48 cm）應變剛好為零時，求 Pn 與 Mn。由 ε3=0 直接推得 c=d3=48 cm；β₁=0.80（f'c=350），a=38.4 cm；ε1=0.00250（降伏），ε2=0.00125（未降伏，fs2=2550）；Pn=722 tf；對形心取矩：Mn=68.2 tf·m。

## 核心考點

- 拉力鋼筋應變為零直接給定 c=d3=48 cm（不需試算），是 P-M 互制圖上特殊邊界點
- β₁ 折減：f'c=350 > 280，β₁=0.85-0.05×(350-280)/70=0.80（非 0.85）
- 中排鋼筋（d2=28 cm < a=38.4 cm）在壓力區：需計算 Cs2=(fs2-0.85f'c)×As2
- 壓力鋼筋扣 0.85f'c：Whitney 塊已含該區混凝土，鋼筋淨力需扣除避免重複

## 解題關鍵步驟

1. ε3=0 → c=48 cm；β₁=0.80；a=0.80×48=38.4 cm；0.85f'c=0.85×350=297.5 kgf/cm²
2. ε1=0.003×(48-8)/48=0.00250 > εy → 降伏，fs1=4200；ε2=0.003×(48-28)/48=0.00125 → 未降伏，fs2=0.00125×2040000=2550 kgf/cm²
3. Cc=0.85×350×38.4×56=639744 kgf；Cs1=(4200-297.5)×15.21=59357 kgf；Cs2=(2550-297.5)×10.14=22840 kgf；T3=0
4. Pn=639744+59357+22840+0=721941 kgf≈722 tf
5. 對形心（h/2=28 cm）取矩：Mn=639744×8.8+59357×20+0+0=6816887 kgf·cm=68.2 tf·m

## 用到的公式

- 由特殊條件直接確定 c：$\varepsilon_3 = 0 \Rightarrow c = d_3$
- β₁ 折減：$\beta_1 = 0.85 - 0.05(f'_c-280)/70$（f'c > 280）
- 壓力區鋼筋淨力：$C_{si} = (f_{si}-0.85f'_c) \times A_{si}$（$d_i < a$）
- 應變相容：$\varepsilon_i = \varepsilon_{cu}(c-d_i)/c$
- 彎矩（對形心）：$M_n = C_c(h/2-a/2) + \Sigma C_{si}(h/2-d_i)$

## 涉及陷阱

- β₁ 必須折減（f'c=350）：誤用 0.85 使 a=40.8 cm，各排應力均偏差
- 中排鋼筋在 Whitney 塊內（d2=28 < a=38.4）：需計算 Cs2（非 T3），且對形心矩臂為 0
- 拉力鋼筋 T3=0（應變為零，對 Pn 和 Mn 均無貢獻）
