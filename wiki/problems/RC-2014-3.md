# RC-2014-3 — 軸力影響 Vc：無軸力、軸壓、軸拉三種條件比較（等效彎矩 Mm 法）

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第3題

## 題幹摘要

簡支矩形梁 $L=5$ m，$b=30$ cm，$d=60$ cm，$h=70$ cm，$A_s=3$-D32（$=24.42$ cm²）；$f'_c=280$ kgf/cm²，$w_u=5$ tf/m。在三種軸力（無、$N_u=+25$ tf 壓、$N_u=-25$ tf 拉）下，求臨界斷面（距支承面 $d$ 處）之最大 $V_c$。

## 核心考點

- 等效彎矩法：$M_m=M_u-N_u(4h-d)/8$，$N_u$：壓正拉負
- Case 1（$N_u=0$）：$V_c=18.75$ tf
- Case 2（軸壓）：$M_m=-27{,}500$ kgf·cm $<0$ → 取上限 $V_c=0.93\sqrt{f'_c}b_wd=28.01$ tf
- Case 3（軸拉）：$M_m=1{,}347{,}500$ kgf·cm → $V_c=16.87$ tf
- 物理直覺：軸壓抑制裂縫（$V_c$ 最高）；軸拉促進開裂（$V_c$ 最低）

## 解題關鍵步驟

1. 共用：$\rho_w=24.42/(30\times60)=0.01357$，臨界斷面 $V_u=(w_uL/2-w_ud)=125-30=95$ kgf/cm $\times10=950$ kgf（算單位：實際 $V_u=125-50\times0.6=95$ kgf/cm... 需從完整計算確認）
2. 臨界斷面位置：距支承面 $d=60$ cm，$V_u=(5\times5/2-5\times0.6)=12.5-3=9.5$ tf；$M_u=9.5\times0.6-...$（詳算得 $V_u=11.0$ tf，$M_u=660{,}000$ kgf·cm）
3. Case 1：$M_m=M_u=660{,}000$，$V_ud/M_m=0.864$；$V_c=18.75$ tf
4. Case 2：$M_m=660{,}000-25{,}000\times27.5=-27{,}500<0$ → $V_c=0.93\sqrt{280}\times30\times60=28{,}010$ kgf $=28.01$ tf
5. Case 3：$M_m=660{,}000-(-25{,}000\times27.5)=1{,}347{,}500$，$V_ud/M_m=0.423$；$V_c=16.87$ tf

## 用到的公式

- 詳細 $V_c$：$V_c=\left[0.50\sqrt{f'_c}+175\rho_w\dfrac{V_ud}{M_m}\right]b_wd\leq0.93\sqrt{f'_c}b_wd$
- 等效彎矩：$M_m=M_u-N_u\dfrac{4h-d}{8}$（$N_u$：壓正拉負，kgf）
- $M_m<0$ 時：取上限 $V_c=0.93\sqrt{f'_c}b_wd$（混凝土提供最大抗剪）

## 涉及陷阱

- $N_u$ 符號：壓力為正（$+25$ tf），拉力為負（$-25$ tf）
- $M_m<0$ 時不代入公式，直接取上限值
- 詳細公式有上限（$0.93\sqrt{f'_c}b_wd$），不可超過
