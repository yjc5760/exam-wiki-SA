# RC-2022-2 — 四種材料組合梁設計選最經濟方案

## 題幹摘要

矩形梁（$b = 30$ cm，$h = 50$ cm，$d = 43.5$ cm），設計彎矩 $M_u = 19.5$ tf·m，使用 D25 拉力筋（$A_b = 5.067$ cm²），從四種材料組合（$f'_c = 210$ 或 280 kgf/cm²，$f_y = 4200$ 或 5600 kgf/cm²）中選出最經濟（D25 支數最少）的組合，並評論材料強度對鋼筋用量的影響。

## 核心考點

- 矩形梁 $A_s$ 設計方程式（四種材料組合分別求解）
- $f_y$ 影響大（$A_s \propto 1/f_y$）；$f'_c$ 影響相對小（僅影響 $a$ 的大小）
- 求 $A_s$ 後無條件進位取 D25 支數，再驗算 $\varepsilon_t$ 及 $\rho_\text{min}$
- 最經濟含義：最少支數下選較低 $f'_c$（降低材料成本）

## 解題關鍵步驟

1. 對四種組合各解 $\phi A_s f_y(d - A_s f_y/(1.7 f'_c b)) = M_u$，假設 $\phi = 0.9$
2. 除以 $A_b = 5.067$ cm²，無條件進位取整數支數
3. 驗算 $\varepsilon_t$ 確認 $\phi = 0.90$，驗算 $\rho \geq \rho_\text{min}$
4. 比較：$f_y = 5600$ 時最少支數，且 $f'_c = 210$ 已足夠
5. 評論：$f_y$ 提升對減少鋼筋用量的效果顯著；$f'_c$ 提升效果有限

## 用到的公式

$$\phi M_n = \phi A_s f_y \!\left(d - \frac{A_s f_y}{1.7 f'_c b}\right) \geq M_u$$

$$A_{s,\text{req}} = \frac{M_u}{\phi f_y (d - a/2)} \quad (\text{迭代或展開二次方程})$$

$$\rho_\text{min} = \max\!\left(\frac{14}{f_y},\, \frac{0.8\sqrt{f'_c}}{f_y}\right)$$

$$\varepsilon_t = \frac{0.003(d - c)}{c} \geq 0.005 \quad (\phi = 0.9)$$

## 涉及陷阱

- 假設 $\phi = 0.90$ 需事後驗算：確認 $\varepsilon_t \geq 0.005$；否則進入過渡區需修正。
- 支數必須無條件進位：$A_{s,\text{req}} / A_b$ 非整數時向上取整，不可取下。
- 最經濟含義：最少支數，支數相同時選較低 $f'_c$（不是最小 $A_s$ 數值）。
- 梁寬限制：$b = 30$ cm 時 D25 每層最多 3 支，超過需排雙層（影響 $d$）。
