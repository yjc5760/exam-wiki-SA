# RC-2006-3 — 靜載重後施加集中活載重求跨中即時撓度增量 3.27 cm

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第3題
**考年：** 2006（民國95年）
**主分類：** [[RC-U3-1]] 梁工作性要求（含撓度、裂縫）
**副分類：** 無
**設計法：** USD強度設計法
**標籤：** `簡支梁` `即時撓度增量` `有效慣性矩Ie` `開裂彎矩Mcr` `轉換斷面法` `集中活載重` `忽略壓力筋`
**驗證狀態：** ✅ verified

---

## 題幹摘要

簡支梁 L=8 m，b=30 cm，h=34 cm，d=26 cm，5-#9 張力筋（As=32.35 cm²），壓力筋忽略，f'c=280 kgf/cm²，n=8，wD=0.8 tf/m（均布靜載），PL=5 tf（跨中集中活載）。求施加 PL 後跨中即時撓度增量。MD=64 tf·cm，MD+L=164 tf·m；Icr=64668 cm⁴；IeD=65609 cm⁴（Ma≈Mcr 附近），IeD+L=64713 cm⁴；ΔD=2.551 cm，ΔD+L=5.819 cm，ΔPL=3.27 cm。

## 核心考點

- 增量撓度=全部載重下撓度−僅 DL 下撓度；兩種載重水準各自對應不同的 Ie
- 開裂慣性矩 Icr：二次方程求 kd（轉換裂縫斷面），Icr=b(kd)³/3+nAs(d-kd)²
- 有效慣性矩 Ie：Ie=(Mcr/Ma)³Ig+[1-(Mcr/Ma)³]Icr；當 Ma >> Mcr 時 Ie≈Icr
- 混合撓度公式：ΔD+L=5wDL⁴/(384EcIe)+PLL³/(48EcIe)，兩項共用同一 Ie

## 解題關鍵步驟

1. Ec=Es/n=2040000/8=255000 kgf/cm²；Ig=30×34³/12=98260 cm⁴
2. Mcr=2√280×98260/17=193500 kgf·cm；MD=640000 kgf·cm；MD+L=640000+1000000=1640000 kgf·cm
3. kd 二次方程：15kd²−32.35×8(26−kd)=0 → kd=14.24 cm；Icr=64668 cm⁴
4. IeD=(193500/640000)³×98260+[1−(0.302)³]×64668=65609 cm⁴
5. IeD+L=(193500/1640000)³×98260+[1−...]×64668=64713 cm⁴
6. ΔD=5×8×800⁴/(384×255000×65609)=2.551 cm；ΔD+L=2.551×65609/64713+5000×800³/(48×255000×64713)=5.819 cm；ΔPL=3.27 cm

## 用到的公式

- 開裂中性軸：$b(kd)^2/2 = nA_s(d-kd)$
- 開裂慣性矩：$I_{cr} = b(kd)^3/3 + nA_s(d-kd)^2$
- 有效慣性矩：$I_e = (M_{cr}/M_a)^3 I_g + [1-(M_{cr}/M_a)^3]I_{cr}$
- 均布載重撓度：$\Delta = 5wL^4/(384E_cI_e)$
- 集中載重撓度：$\Delta = PL^3/(48E_cI_e)$
- 開裂彎矩：$M_{cr} = f_r I_g / y_t$，$f_r = 2\sqrt{f'_c}$（kgf/cm² 制）

## 涉及陷阱

- 兩個 Ie 不同：ΔD 用 IeD，ΔD+L 用 IeD+L；不能用同一個 Ie 計算兩個狀態
- 混合撓度計算：均布與集中載重各用對應公式（5wL⁴/384 與 PL³/48）疊加
- 壓力筋題目明確說可忽略：若誤納入 nAs'，kd 與 Icr 均偏差
