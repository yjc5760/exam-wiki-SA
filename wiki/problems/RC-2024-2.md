# RC-2024-2 — 雙筋懸臂梁壓力鋼筋未降伏最大均佈載重

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第2題
**考年：** 2024（民國113年）
**主分類：** [[RC-U1-1]] RC 梁彎矩強度分析與設計
**設計法：** USD強度設計法
**標籤：** `雙筋梁` `壓力鋼筋未降伏` `應變相容` `懸臂梁` `因數化均佈載重` `標稱彎矩強度` `二次方程求中性軸`
**驗證狀態：** ✅ verified

## 題幹摘要

懸臂梁（$b = 30$ cm，$d = 44$ cm，$d' = 8$ cm，$L = 3$ m），底部壓力區 $A'_s = 2\text{-D32} = 16.28$ cm²，頂部拉力區 $A_s = 3\text{-D32} = 24.42$ cm²，$f'_c = 210$ kgf/cm²，$f_y = 4200$ kgf/cm²。（一）求 $\phi M_n$ 及壓力筋應力 $f'_s$；（二）求最大因數化均佈載重 $\omega_u$。

## 核心考點

- 懸臂梁固定端底部受壓：壓力區在底部，$d$ 和 $d'$ 均從壓力面（底面）量起
- 壓力鋼筋降伏假設驗證：假設降伏求 $c$，若 $c < d'$ 則矛盾（壓力筋在拉力側），改用應變相容
- 應變相容法：設 $f'_s = E_s \varepsilon'_s$ 代入力平衡，展開為 $c$ 的二次方程
- 懸臂梁力矩公式：$M_u = \omega_u L^2/2$（均佈載重）

## 解題關鍵步驟

1. 假設壓力筋降伏（$f'_s = f_y$）：力平衡求 $c$；若 $c < d' = 8$ cm 矛盾（壓力筋在拉力側），需重解
2. 設 $f'_s = E_s \times 0.003(c - 8)/c$，代入力平衡，展開為二次方程求正根 $c$
3. 計算 $a = \beta_1 c$，求 $f'_s$；驗算拉力筋應變 $\varepsilon_t > \varepsilon_y$，判斷 $\phi$
4. 計算 $M_n$（對拉力筋取矩），求 $\phi M_n$
5. 由 $\phi M_n \geq M_u = \omega_u L^2/2$，求最大 $\omega_u = 2\phi M_n/L^2$

## 用到的公式

$$\varepsilon'_s = \frac{0.003(c - d')}{c}$$

$$f'_s = E_s \varepsilon'_s \quad (\text{壓力筋未降伏時})$$

$$f'_{s,\text{net}} = f'_s - 0.85 f'_c \quad (d' < a = \beta_1 c)$$

$$C_c + A'_s f'_{s,\text{net}} = A_s f_y \quad (\text{力平衡})$$

$$M_u = \frac{\omega_u L^2}{2} \quad (\text{懸臂梁均佈載重})$$

$$\omega_u = \frac{2 \phi M_n}{L^2}$$

## 涉及陷阱

- 懸臂梁方向感：固定端底部受壓（負彎矩），$d$ 和 $d'$ 從底面（壓力面）量起，與正彎矩梁相反。
- 假設降伏後需驗算 $c$ 與 $d'$ 的大小：$c < d'$ 表示壓力筋在中性軸拉力側，假設矛盾。
- 懸臂梁力矩公式：均佈載重 $M = \omega_u L^2/2$，非 $\omega_u L^2/8$（簡支梁公式）。
- 壓力筋淨壓力：$d' < a$ 時需扣除 $0.85f'_c$，避免重複計算。
