# RC-2025-3 — 矩形梁空間桁架類比扭矩強度設計

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第3題
**考年：** 2025（民國114年）
**主分類：** [[RC-U2-2]] RC 扭力強度設計
**設計法：** USD強度設計法
**標籤：** `扭矩強度` `空間桁架類比` `Tn` `閉合箍筋` `縱向鋼筋驗核` `Aoh` `矩形梁扭力`
**驗證狀態：** ✅ verified

---

## 題幹摘要

矩形梁（$b = 40$ cm，$h = 70$ cm），下層 4-D32（$A_{sl,bot} = 32.56$ cm²），上層 2-D29（$A_{sl,top} = 12.94$ cm²），D13 閉合箍筋間距 15 cm（$A_t = 1.27$ cm²/腳），淨保護層 4 cm，$f'_c = 280$ kgf/cm²，$f_{yt} = 2800$ kgf/cm²。假設箍筋僅用於抵抗扭矩，求標稱扭矩強度 $T_n$，並驗核縱向鋼筋是否足夠。

## 核心考點

- 空間桁架類比：扭矩由閉合箍筋（橫向）與縱向鋼筋共同承擔
- $A_{oh}$（箍筋圍合面積）從淨保護層推至箍筋中心線（加箍筋半徑）
- $A_o = 0.85 A_{oh}$（平均剪力流路徑折減係數）
- 箍筋貢獻公式：$T_n = 2A_o A_t f_{yt}/s$（$\theta = 45°$，$\cot\theta = 1$）
- 縱向鋼筋需求量驗核（需求 $A_{sl,req}$ vs. 提供量）

## 解題關鍵步驟

1. 箍筋中心線距梁面：$c_c + d_{b,13}/2 = 4 + 0.635 = 4.635$ cm（淨保護層加箍筋半徑）
2. 箍筋圍合尺寸：$x_1 = 40 - 2 \times 4.635 = 30.73$ cm；$y_1 = 70 - 2 \times 4.635 = 60.73$ cm
3. $A_{oh} = x_1 \times y_1 = 1866$ cm²；$p_h = 2(x_1 + y_1) = 183$ cm
4. $A_o = 0.85 \times 1866 = 1586$ cm²
5. 箍筋提供 $T_n = 2 \times A_o \times (A_t/s) \times f_{yt} = 2 \times 1586 \times (1.27/15) \times 2800$（kgf·cm）
6. 驗核縱筋需求：$A_{sl,req} = (A_t/s) p_h (f_{yt}/f_y)\cot^2\theta$；比較提供量 $A_{sl} = A_{sl,bot} + A_{sl,top}$

## 用到的公式

$$A_{oh} = x_1 \times y_1, \quad p_h = 2(x_1 + y_1)$$

$$A_o = 0.85 A_{oh}$$

$$T_n = 2 A_o \frac{A_t}{s} f_{yt} \cot\theta \quad (\theta = 45°,\ \cot\theta = 1)$$

$$A_{sl,req} = \frac{A_t}{s} p_h \frac{f_{yt}}{f_y} \cot^2\theta$$

$$A_{sl,\text{provide}} = A_{sl,bot} + A_{sl,top} \geq A_{sl,req}$$

## 涉及陷阱

- $A_{oh}$ 算到箍筋中心線：非外緣（$b \times h$）也非內緣；中心線距梁面 $= c_c + d_b/2$（保護層加箍筋半徑）。
- $A_t$ 為單腳面積：公式 $T_n = 2A_o(A_t/s)f_{yt}$ 中的 $A_t$ 為一腳，係數 2 已涵蓋兩腳效果。
- $\theta = 45°$：非預力梁取 $\theta = 45°$，$\cot\theta = 1.0$，直接代入。
- 縱筋需求需驗核：計算 $A_{sl,req}$ 並確認提供量 $\geq$ 需求量，不足則須補充縱向鋼筋。
