# RC-2024-3 — 方形柱拉力鋼筋應變為零 P-M 強度計算

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第3題
**考年：** 2024（民國113年）
**主分類：** [[RC-U1-2]] RC 柱強度分析與設計
**設計法：** USD強度設計法
**標籤：** `方形柱` `P-M互制` `拉力鋼筋應變為零` `應變相容` `壓力鋼筋未降伏` `軸力彎矩組合` `壓力控制區`
**驗證狀態：** ✅ verified

## 題幹摘要

40×40 cm 方柱，8-D25（3+2+3 排列），$f'_c = 280$ kgf/cm²，$f_y = 4200$ kgf/cm²，$\varepsilon_u = 0.003$。各排距頂壓縮面：頂排 $y_1 = 8$ cm，中排 $y_2 = 20$ cm（形心），底排 $y_3 = 32$ cm。破壞時底排（拉力鋼筋）應變恰為零。求標稱軸力強度 $P_n$ 及標稱彎矩強度 $M_n$。

## 核心考點

- 「底排應變 = 0」等於中性軸剛好落在底排鋼筋處：直接確認 $c = d_t = 32$ cm
- 應力塊覆蓋頂兩排，底排應變為零（不貢獻軸力也不貢獻彎矩）
- 頂排和中排均在應力塊內（$y_i < a = 27.2$ cm），需用壓力筋淨壓力
- 壓力控制區：$c = 32 > c_b$，$\varepsilon_t = 0 < \varepsilon_y$

## 解題關鍵步驟

1. 由 $\varepsilon_t = 0$ 直接確認 $c = d_t = 32$ cm；計算 $a = \beta_1 c = 0.85 \times 32 = 27.2$ cm
2. 各排應變：頂排（$y_1 = 8$）壓縮；中排（$y_2 = 20$）微壓縮；底排（$y_3 = 32$）= 0
3. 驗算各排是否在應力塊內（$y_i \leq a = 27.2$ cm）：頂排和中排均在，需用淨壓力
4. 計算各排鋼筋應變與應力，判斷降伏
5. 軸力：$P_n = C_c + \Sigma C_{si} - \Sigma T_i$（底排貢獻為零）
6. 彎矩（對截面形心取矩，$y_2 = 20$ cm 中排力臂為零）：$M_n = C_c(20 - a/2) + \Sigma C_{si}(20 - y_i)$

## 用到的公式

$$c = d_t = 32\text{ cm} \quad (\varepsilon_t = 0 \text{ 邊界條件})$$

$$a = \beta_1 c = 0.85 \times 32 = 27.2\text{ cm}$$

$$\varepsilon_{si} = \frac{0.003(c - y_i)}{c} \quad (\text{各排應變})$$

$$f_{si,\text{net}} = f_{si} - 0.85 f'_c \quad (y_i < a,\text{ 壓縮區鋼筋淨壓力})$$

$$P_n = C_c + \Sigma C_{si,\text{net}} - \Sigma T_i$$

$$M_n = C_c \!\left(\frac{h}{2} - \frac{a}{2}\right) + \Sigma C_{si,\text{net}} \!\left(\frac{h}{2} - y_i\right)$$

## 涉及陷阱

- 底排應變 = 0 直接給出 $c = d_t$：無需迭代，一步確認中性軸位置。
- 壓縮區鋼筋必須用淨壓力：頂排和中排均在應力塊（$a = 27.2 > 20$），均需扣除 $0.85f'_c$。
- 底排對力矩無貢獻：$f_t = 0$，底排鋼筋既不提供軸力也不提供彎矩。
- 中排力矩貢獻為零：中排在截面形心（$y_2 = 20 = h/2$），力臂為零，但有軸力貢獻。
