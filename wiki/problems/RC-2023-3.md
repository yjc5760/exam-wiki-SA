# RC-2023-3 — 懸臂梁臨界剪力與扭力忽略門檻偏心量

## 題幹摘要

矩形懸臂梁（$b = 35$ cm，$h = 60$ cm，$d = 53$ cm，跨度 3.5 m），承受偏心垂直均佈載重 $W_u = 2.5$ tf/m（偏心量 $e$），$f'_c = 280$ kgf/cm²，$\phi = 0.75$。（一）求臨界斷面最大設計剪力 $V_u$；（二）求可忽略扭力的最大偏心量 $e_\text{max}$。

## 核心考點

- 懸臂梁臨界斷面位置：距固定端支承面 $d$ 處，$V_u = W_u(L - d)$
- CNS 1480 扭力忽略門檻：$T_u \leq \phi T_{th}$（裂縫扭矩的門檻值）
- $A_{cp}$ 和 $p_{cp}$ 使用全斷面外輪廓（非箍筋圍合區域）
- 偏心載重同時產生剪力與扭矩：$T_u = V_u \times e$

## 解題關鍵步驟

1. 懸臂梁臨界斷面距固定端 $d = 53$ cm：$V_u = 2.5 \times (3.5 - 0.53) = 7.425$ tf
2. 斷面幾何：$A_{cp} = 35 \times 60 = 2100$ cm²，$p_{cp} = 2(35 + 60) = 190$ cm
3. 計算扭力忽略門檻：$T_{th} = 0.265\sqrt{f'_c}(A_{cp}^2/p_{cp})$
4. $\phi T_{th} = 0.75 \times T_{th}$
5. 由 $T_u = V_u \times e \leq \phi T_{th}$，解 $e_\text{max} = \phi T_{th}/V_u$

## 用到的公式

$$V_u = W_u(L - d) \quad (\text{懸臂梁臨界斷面})$$

$$T_{th} = 0.265\sqrt{f'_c} \cdot \frac{A_{cp}^2}{p_{cp}}$$

$$A_{cp} = b \times h \quad (\text{全斷面外輪廓面積})$$

$$p_{cp} = 2(b + h) \quad (\text{全斷面外輪廓周長})$$

$$T_u = V_u \times e \leq \phi T_{th} \implies e_\text{max} = \frac{\phi T_{th}}{V_u}$$

## 涉及陷阱

- 懸臂梁臨界斷面位置：距固定端（支承面）$d$ 處，非距自由端，$V_u = W_u(L - d)$。
- $A_{cp}$ 是全斷面外輪廓面積：$A_{cp} = b \times h$，非箍筋圍合面積 $A_{oh}$（後者更小）。
- $\phi = 0.75$：扭力折減係數同剪力，均為 0.75。
- 扭矩與偏心的乘積關係：$T_u = V_u \times e$，加大偏心即加大扭矩。
