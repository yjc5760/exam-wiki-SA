# RC-2003-2 — 耐震方形柱 16-#9 詳細計算法求 Class B 拉力搭接長度

## 題幹摘要

耐震方形柱，60×60 cm，淨高 300 cm，16-#9 主筋，#4@10cm 箍筋，淨保護層 4 cm，f'c=280 kgf/cm²，fy=4200 kgf/cm²。用詳細計算法求柱主筋搭接長度，並標示於立面圖。關鍵在於每面 5 根主筋，c 由半中心距（5.825 cm）控制，(c+Ktr)/db=9.45 > 2.5，取上限 2.5，得 ld=80.7 cm，ls=1.3ld=105 cm，搭接位置限於淨高中段 1/2（75~225 cm）。

## 核心考點

- c 值取小：保護層至中心 c_cover=6.705 cm，半中心距 c_spacing=5.825 cm，取小值 5.825 cm（間距控制）
- Ktr 橫向鋼筋指標：$K_{tr}=A_{tr}f_{yt}/(10sn)$，n 為沿潛在劈裂面被搭接的鋼筋根數（每面 5 根）
- (c+Ktr)/db 觸及上限 2.5：箍筋充足時取最優圍束上限，可得最短 ld
- 耐震柱 Class B 搭接：地震逆作主筋受拉，ls=1.3ld；位置限於柱淨高中間半段

## 解題關鍵步驟

1. 確認每面 5 根鋼筋，計算保護層至中心 c_cover=4+1.27+1.435=6.705 cm
2. 計算主筋中心距 scc=(60-2×6.705)/4=11.65 cm，半中心距 c_spacing=5.825 cm，c=min=5.825 cm
3. 計算 Ktr=2.54×4200/(10×10×5)=21.3，(c+Ktr)/db=(5.825+21.3)/2.87=9.45 > 2.5，取 2.5
4. 代入詳細公式：ld=0.28×2.87×4200/√280/2.5=80.7 cm
5. Class B 搭接：ls=1.3×80.7=105 cm
6. 搭接位置：柱淨高中段（75~225 cm），75+105=180 cm < 225 cm ✓

## 用到的公式

- 詳細計算法錨定長度：$l_d = \dfrac{0.28\,d_b f_y}{\sqrt{f'_c}} \cdot \dfrac{\alpha\beta\lambda}{(c+K_{tr})/d_b}$，上限 $(c+K_{tr})/d_b \le 2.5$
- 橫向鋼筋指標：$K_{tr} = \dfrac{A_{tr} f_{yt}}{10 s n}$
- Class B 拉力搭接：$l_s = 1.3\,l_d$

## 涉及陷阱

- n 的定義：沿潛在劈裂面方向同時被搭接的鋼筋根數（每面 5 根），不是全斷面根數
- c 取兩種情況的較小值：保護層到中心 vs 半中心距，本題由間距控制
- 耐震柱搭接不得用壓力搭接長度：地震時主筋受拉，必須用拉力 Class B
