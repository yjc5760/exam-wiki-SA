# RC-2024-4 — 超靜定預力梁等效載重法次彎矩 C 線設計

## 題幹摘要

後張法 propped cantilever（一端固定一端滾支），跨度 12 m，$b = 40$ cm，$h = 60$ cm，有效預力 $P = 120$ tf。拋物線腱偏心：固定端 A $e = -25$ cm（腱在形心上方），跨中 $e = +15$ cm（腱在形心下方），滾支端 B $e = 0$。求（a）等效載重法支承反力；（b）各斷面次彎矩；（c）C 線偏心距；（d）一致腱概念說明。

## 核心考點

- 拋物線腱等效載重：均佈上升力 $w_{eq} = 8Pf/L^2$ 及端部錨定力矩 $M_{eq} = Pe_A$
- 超靜定梁（propped cantilever）受等效載重後的超靜定計算（不對稱邊界）
- 次彎矩定義：$M_s(x) = P[e_C(x) - e(x)]$
- C 線（壓力線）：$e_C(x) = e(x) + M_s(x)/P$
- 一致腱：腱形與 C 線重合，次彎矩 = 0

## 解題關鍵步驟

1. 三點偏心配合拋物線，擬合 $e(x) = ax^2 + bx + c$，確認矢高 $f$
2. 等效載重：均佈上升力 $w_{eq} = 8Pf/L^2$，端部力矩 $M_{eq} = Pe_A$（固定端腱端偏心引起）
3. 將等效載重施於 propped cantilever，求超靜定支承反力（超靜定分析）
4. 由支承反力計算各斷面彎矩 $M(x)$（即次彎矩 $M_s(x)$）
5. C 線偏心距：$e_C(x) = e(x) + M_s(x)/P$
6. 說明一致腱：腱形 = C 線時，次彎矩 = 0，超靜定束制不產生額外應力

## 用到的公式

$$w_{eq} = \frac{8P f}{L^2} \quad (f = \text{拋物線矢高})$$

$$M_{eq} = P \times e_A \quad (\text{端部錨定力矩})$$

$$M_s(x) = P[e_C(x) - e(x)] \quad (\text{次彎矩})$$

$$e_C(x) = e(x) + \frac{M_s(x)}{P} \quad (\text{C 線偏心距})$$

## 涉及陷阱

- propped cantilever 為超靜定：一端固定有三個反力，不是靜定梁，需用超靜定分析（力法或位移法）。
- 端部錨定力矩：腱端偏心不為零（固定端 $e_A = -25$ cm），需考慮端部彎矩作為等效載重的一部分。
- 次彎矩的定義：$M_s = P(e_C - e)$，不是外載引起的彎矩，而是超靜定束制引起的附加彎矩。
- 一致腱的意義：腱形與 C 線一致時，超靜定束制不產生額外應力（次彎矩 = 0）。
