# RC-2011-1 — P-M互制圖完整求算步驟（概念題）

## 題幹摘要

請簡要說明鋼筋混凝土柱斷面之 $P_u$-$M_u$ 關係曲線圖（P-M 互制圖）之求算步驟，其中 $P_u$ 及 $M_u$ 分別為柱斷面之設計軸力強度（$\phi P_n$）及設計彎矩強度（$\phi M_n$）。本題為純概念說明題，無附圖，無具體數值。（10 分）

## 核心考點

- P-M 互制圖為柱斷面在各種偏心距下最大承載能力的包絡線，任何 $(P_u, M_u)$ 落在曲線內部表示安全
- 三個關鍵特徵點：純壓點（$e=0$）、平衡點（$\varepsilon_t=\varepsilon_y$）、純彎點（$P_n=0$）
- $\phi$ 值隨 $\varepsilon_t$ 在三個區段過渡：壓力控制（$\phi=0.65$）→ 過渡區（線性插值）→ 拉力控制（$\phi=0.90$）
- 應變相容（strain compatibility）與力平衡（force equilibrium）兩原則的聯合應用

## 解題關鍵步驟

1. 確立座標系（縱軸 $\phi P_n$，橫軸 $\phi M_n$），確認材料常數（$\beta_1$、$\varepsilon_y$、$\varepsilon_{cu}=0.003$）
2. 求純軸壓點 Point A：$P_{n0} = 0.85f'_c(A_g-A_{st})+f_yA_{st}$，設計上限 $\phi P_{n,\max}=0.80\phi P_{n0}$（矩形箍）
3. 求平衡點 Point B：$c_b = \frac{6120}{6120+f_y}\cdot d$，$a_b=\beta_1 c_b$，依應變相容計算各鋼筋力，求 $P_{nb}$、$M_{nb}$
4. 求純彎點 Point C：令 $P_n=0$，以 $c$ 為未知數解平衡方程，得 $M_{nc}$（此處 $\phi=0.90$）
5. 在特徵點之間取若干假設 $c$ 值，對每個 $c$ 計算 $\varepsilon_t$、$\varepsilon'_s$、各力、$P_n$、$M_n$
6. 依 $\varepsilon_t$ 套用對應 $\phi$ 值，計算設計強度 $(\phi M_n,\;\phi P_n)$ 後描點連線

## 用到的公式

- 純軸壓標稱強度：$P_{n0} = 0.85f'_c(A_g - A_{st}) + f_y A_{st}$
- 平衡中性軸：$c_b = \dfrac{6120}{6120+f_y}\cdot d$（kgf/cm² 系統）
- 任意點拉力鋼筋應變：$\varepsilon_t = \varepsilon_{cu}\cdot\dfrac{d-c}{c}$
- 壓力鋼筋應力修正：$C_s = A'_s(f'_s - 0.85f'_c)$
- $\phi$ 過渡區插值：$\phi = 0.65 + 0.25\cdot\dfrac{\varepsilon_t - \varepsilon_y}{0.005 - \varepsilon_y}$（$\varepsilon_y < \varepsilon_t < 0.005$）

## 涉及陷阱

- 純軸壓設計上限係數：矩形箍用 **0.80**，螺旋柱用 **0.85**，不可混用
- $c_b$ 公式單位：kgf/cm² 制為 $6120/(6120+f_y)$，SI 制為 $600/(600+f_y)$，確認單位再代入
- 壓力鋼筋 $C_s$ 需扣除混凝土占位（$-0.85f'_c$），否則重複計算
- $\phi$ 值需以 $\varepsilon_t$ 線性插值，不可在過渡區直接取 0.65 或 0.90
