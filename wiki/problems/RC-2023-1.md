# RC-2023-1 — 矩形梁曲率延展比降伏曲率與極限曲率分析

## 題幹摘要

矩形梁（$b = 40$ cm，$h = 60$ cm，$d = 53$ cm），配 4-D25（$A_s = 20.28$ cm²），$f_y = 4200$ kgf/cm²。分別計算 $f'_c = 210$ 及 280 kgf/cm² 時的曲率延展比 $\mu_\phi = \phi_u/\phi_y$，並求兩者比值。本題考驗降伏曲率（彈性轉換斷面法）與極限曲率（Whitney 應力塊法）兩種不同分析模型。

## 核心考點

- 降伏曲率 $\phi_y$：用彈性轉換斷面法（$k_y d$）求降伏時中性軸位置
- 極限曲率 $\phi_u$：用 Whitney 應力塊求極限狀態中性軸 $c_u$，$\phi_u = \varepsilon_{cu}/c_u$
- 兩種模型物理意義完全不同，不可混用
- $f'_c$ 提升 → $c_u$ 減小 → $\phi_u$ 增大 → $\mu_\phi$ 增大（高強混凝土延性更好）

## 解題關鍵步驟

1. 計算材料參數：$E_c = 15000\sqrt{f'_c}$（kgf/cm²），$n = E_s/E_c$，$\rho = A_s/(bd)$
2. 彈性轉換斷面法求 $k_y$：解 $k_y^2 + 2n\rho k_y - 2n\rho = 0$，$c_y = k_y d$
3. 降伏曲率：$\phi_y = \varepsilon_y/(d - c_y)$（鋼筋剛降伏時的曲率）
4. 極限中性軸（Whitney 應力塊）：$c_u = A_s f_y/(0.85 f'_c b)$
5. 極限曲率：$\phi_u = 0.003/c_u$（$\varepsilon_{cu} = 0.003$）
6. 延展比：$\mu_\phi = \phi_u/\phi_y$，分別算兩個 $f'_c$，求比值

## 用到的公式

$$k_y^2 + 2n\rho\, k_y - 2n\rho = 0, \quad c_y = k_y d$$

$$\phi_y = \frac{\varepsilon_y}{d - c_y} = \frac{f_y/E_s}{(1 - k_y)d}$$

$$c_u = \frac{A_s f_y}{0.85 f'_c b}$$

$$\phi_u = \frac{\varepsilon_{cu}}{c_u} = \frac{0.003}{c_u}$$

$$\mu_\phi = \frac{\phi_u}{\phi_y}$$

## 涉及陷阱

- 降伏狀態用轉換斷面法，極限狀態用 Whitney：兩種模型分析不同狀態，不可混用。
- $k_y d$ 與 $c_u$ 物理意義不同：前者是彈性降伏中性軸（壓力側 $kd$），後者是極限破壞中性軸。
- $E_c$ 公式：$E_c = 15000\sqrt{f'_c}$（kgf/cm²），兩個 $f'_c$ 分別計算 $n$。
- 高強混凝土延性更好：$f'_c$ 提升使 $c_u$ 減小，$\phi_u$ 增大，結論反直覺但正確。
