# RC-2019-1 — 單筋矩形梁過渡區 φ 值與設計彎矩強度

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第1題

**設計法：** USD強度設計法
**標籤：** `單筋梁` `過渡區φ值` `Whitney應力塊` `矩形斷面` `二次方程求As` `應變相容` `φ=0.65~0.9`
**驗證狀態：** ✅ verified

## 題幹摘要

單筋矩形梁斷面，已知斷面尺寸（$b$、$h$）、混凝土抗壓強度 $f'_c$ 及鋼筋降伏強度 $f_y$，承受給定設計彎矩 $M_u$。題目明示強度折減係數 $\phi$ 並非固定為 0.9，要求考生在過渡區（$0.002 < \varepsilon_t < 0.005$）正確建立 $\phi$–$A_s$–$\varepsilon_t$ 聯立方程，化為二次方程求解所需拉力鋼筋量 $A_s$，最後驗算延性條件與最小鋼筋量。

## 核心考點

- Whitney 等效矩形應力塊參數 $\beta_1$ 的正確取值（依 $f'_c$ 大小）
- 過渡區強度折減係數 $\phi$ 線性內插公式（箍筋柱型，$\phi = 0.65 \to 0.90$）
- 應變相容驗算：確認假設之 $\phi$ 值與解得的 $\varepsilon_t$ 相符
- 含 $\phi$ 的二次方程式求解 $A_s$（不可直接假設 $\phi = 0.9$）
- 最大及最小鋼筋量限制驗核

## 解題關鍵步驟

1. 確認 $\beta_1$：$f'_c \leq 280$ 時 $\beta_1 = 0.85$；每增加 70 kgf/cm² 減 0.05，不低於 0.65
2. 將 $a$、$c$、$\varepsilon_t$ 全部以 $A_s$ 表示，再代入過渡區 $\phi$ 公式
3. 代入 $\phi M_n = M_u$，展開後化簡為關於 $A_s$ 的二次方程，取合理正根
4. 驗算 $\varepsilon_t$ 是否確實落在過渡區（$\varepsilon_y < \varepsilon_t < 0.005$）
5. 驗核 $A_{s,\min} \leq A_s \leq A_{s,\max}$（$\varepsilon_t \geq 0.004$ 為最大鋼筋量限制）

## 用到的公式

$$\beta_1 = 0.85 - 0.05 \cdot \frac{f'_c - 280}{70} \quad (f'_c > 280\,\text{kgf/cm}^2,\ \beta_1 \geq 0.65)$$

$$a = \frac{A_s f_y}{0.85 f'_c b}, \quad c = \frac{a}{\beta_1}$$

$$\varepsilon_t = \frac{d - c}{c} \times 0.003$$

$$\phi = 0.65 + (\varepsilon_t - 0.002) \times \frac{0.25}{0.003} \quad (0.002 \leq \varepsilon_t \leq 0.005,\text{ 箍筋柱型})$$

$$\phi M_n = \phi A_s f_y \left(d - \frac{a}{2}\right) \geq M_u$$

$$A_{s,\min} = \max\!\left(\frac{0.8\sqrt{f'_c}}{f_y},\, \frac{14}{f_y}\right) b_w d$$

## 涉及陷阱

- 直接假設 $\phi = 0.9$：若鋼筋量較大使 $\varepsilon_t < 0.005$，斷面落在過渡區，設計偏不安全。
- $\phi$ 線性內插公式混淆：螺旋柱（$\phi = 0.70 \to 0.90$）與箍筋柱（$\phi = 0.65 \to 0.90$）公式不同，梁設計用箍筋柱型。
- 誤以為 $A_s = M_u / (\phi f_y (d - a/2))$ 可直接解：因 $a$ 也含 $A_s$，且 $\phi$ 也含 $A_s$，必須展開為二次方程。
