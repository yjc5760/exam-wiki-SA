# RC-2025-2 — 特殊矩形框架梁可能彎矩強度與密箍區設計

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第2題
**考年：** 2025（民國114年）

**設計法：** USD強度設計法
**標籤：** `耐震梁` `特殊矩形框架梁` `可能彎矩強度Mpr` `Vc=0條件` `密箍區` `剪力筋設計` `閉合箍筋`

## 題幹摘要

耐震梁（特殊矩形框架梁），淨跨距 $l_n = 9$ m，$b_w = 60$ cm，$h = 76$ cm，$d = 67$ cm，頂層 8-D32（$A_{s,top} = 65.12$ cm²），底層 6-D32（$A_{s,bot} = 48.84$ cm²），因數化重力載重 $w_u = 2$ tf/m，D13 剪力筋（$f_{yt} = 2800$ kgf/cm²），$f'_c = 280$ kgf/cm²，$f_y = 4200$ kgf/cm²。設計靠近柱端的剪力筋，包含密箍區間距與長度。

## 核心考點

- 可能彎矩強度 $M_{pr}$：用 $1.25f_y$（超強係數）代替 $f_y$，$\phi = 1.0$
- 兩個搖擺方向的柱端剪力：各取地震剪力 + 重力剪力，取最大值為設計剪力 $V_u$
- $V_c = 0$ 的條件：地震剪力 $> V_u/2$ 且軸力不大（保守不計混凝土貢獻）
- 密箍區：自柱面算起 $2h$，最大間距為 $\min(d/4, 6d_b, 150\text{ mm})$

## 解題關鍵步驟

1. 計算 $M_{pr}^-$（頂筋受拉，$1.25f_y = 5250$）和 $M_{pr}^+$（底筋受拉），$\phi = 1.0$
2. 兩個搖擺方向各計算柱端剪力：$V_u = (M_{pr,l} + M_{pr,r})/l_n \pm w_u l_n/2$，取較大者
3. 判斷 $V_c = 0$ 條件：地震剪力 $V_E = (M_{pr,l} + M_{pr,r})/l_n > 0.5V_u$ → $V_c = 0$
4. $V_s = V_u/\phi$（$V_c = 0$），計算 D13 箍筋間距 $s = A_v f_{yt} d/V_s$
5. 耐震密箍區間距上限：$s_o = \min(d/4, 6d_b, 150\text{ mm})$，取最嚴格值
6. 密箍區長度 $= 2h$ 從柱面算起；第一支距柱面 $\leq s_o/2$

## 用到的公式

$$M_{pr} = 1.0 \times A_s \times 1.25 f_y \times \!\left(d - \frac{a'}{2}\right), \quad a' = \frac{A_s \times 1.25 f_y}{0.85 f'_c b_w}$$

$$V_u = \frac{M_{pr,l}^- + M_{pr,r}^+}{l_n} + \frac{w_u l_n}{2} \quad (\text{搖擺方向一})$$

$$V_u = \frac{M_{pr,l}^+ + M_{pr,r}^-}{l_n} - \frac{w_u l_n}{2} \quad (\text{搖擺方向二，取絕對值})$$

$$s_o = \min\!\left(\frac{d}{4},\, 6d_b,\, 150\text{ mm}\right) \quad (\text{密箍區最大間距})$$

$$s = \frac{A_v f_{yt} d}{V_s} \leq s_o$$

## 涉及陷阱

- $M_{pr}$ 用 $1.25f_y$：不是 $f_y$，超強係數確保梁端塑鉸能充分發展，$M_{pr}$ 遠大於 $M_n$。
- $V_c = 0$ 的判斷：地震剪力超過總剪力一半時，不計混凝土貢獻（保守安全設計）。
- 密箍區第一支位置：距柱面 $\leq s_o/2$（非 $s_o$）。
- 兩個搖擺方向都算：取較大的 $V_u$ 作為設計剪力。
