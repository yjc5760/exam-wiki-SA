# RC-2020-3 — 無梁版內柱穿孔剪力三公式設計

## 題幹摘要

地下室無梁版柱系統，均佈設計載重 $w_u = 4.6$ tf/m²，版全深 40 cm，有效深度 $d = 35$ cm，方形內柱 $100 \times 100$ cm，柱間距 800 cm。（一）驗核混凝土穿孔剪力強度是否足夠；（二）若不足，計算雙肢剪力鋼筋所需間距。本題考驗三公式取最小、加剪力筋後 $V_c$ 折減等關鍵觀念。

## 核心考點

- 穿孔剪力臨界周長 $b_o = 4(c+d)$（距柱面 $d/2$ 處）的計算
- 三個 $V_c$ 公式（依 $\beta_c$、$\alpha_s$、上限）全算取最小值
- 加入剪力筋後混凝土貢獻降低：$V_c$ 係數從 $1.06$ 降至 $0.53$
- 剪力筋間距同時受強度需求與規範上限 $d/2$ 控制

## 解題關鍵步驟

1. 計算臨界周長：$b_o = 4(100 + 35) = 540$ cm
2. 計算穿孔剪力需求：$V_u = w_u \times [800^2 - 135^2] \times 10^{-4} = 286$ tf（扣除臨界周長內面積）
3. 三公式算最小 $V_c$，公式（3）$= 1.06\sqrt{f'_c} b_o d$ 通常控制；$\phi V_c < 286$ tf，不足
4. 加入剪力筋：$V_c^{(w)} = 0.53\sqrt{f'_c} b_o d$（混凝土貢獻取保守值）
5. 計算 $V_s = V_u/\phi - V_c^{(w)}$
6. 強度需求 $s = A_v f_{yt} b_o / V_s$；規範上限 $s \leq d/2 = 17.5$ cm，取較嚴格者

## 用到的公式

$$b_o = 4(c + d) \quad (\text{內柱})$$

$$V_u = w_u \left[L^2 - (c+d)^2\right]$$

$$V_{c,1} = 0.265\!\left(1 + \frac{4}{\beta_c}\right)\!\sqrt{f'_c}\, b_o d$$

$$V_{c,2} = \left(0.083\alpha_s \frac{d}{b_o} + 0.265\right)\!\sqrt{f'_c}\, b_o d$$

$$V_{c,3} = 1.06\sqrt{f'_c}\, b_o d \quad (\text{上限})$$

$$V_{c}^{(w)} = 0.53\sqrt{f'_c}\, b_o d \quad (\text{加剪力筋後混凝土貢獻})$$

$$s = \frac{A_v f_{yt} b_o}{V_s}, \quad s \leq \frac{d}{2}$$

## 涉及陷阱

- $V_u$ 面積需扣除臨界周長內面積：$V_u = w_u[L^2 - (c+d)^2]$，若用全面積 $L^2$ 則偏大。
- 三個公式全算取最小：$\beta_c = 1$（方形柱），通常公式（3）為控制值。
- 加剪力筋後 $V_c$ 反而降低：從 $1.06$ 係數降至 $0.53$，但總 $V_n = V_c^{(w)} + V_s$ 仍提升。
- 規範上限 $d/2$ 控制：本題強度需求 $\approx 26.9$ cm 大於規範上限 17.5 cm，以規範控制。
