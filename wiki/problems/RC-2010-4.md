# RC-2010-4 — 曲線型鋼腱優點及四控制條件求偏心距範圍 27.68 cm≤ep≤30.18 cm

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第4題
**考年：** 2010（民國99年）
**主分類：** [[RC-U4-2]] 預力量與偏心量設計
**副分類：** [[RC-U4-1]] 預力梁斷面應力分析
**設計法：** WSD工作應力法
**標籤：** `曲線型鋼腱` `等效均布載重` `偏心距限制` `四控制條件` `傳遞階段` `服務階段` `完全預力` `容許應力` `矩形斷面` `對稱斷面` `窄偏心範圍`
**驗證狀態：** ✅ verified

## 題幹摘要

(一) 說明曲線型鋼腱相比直線型的優點（等效均布載重 We=8Pep/L²，平衡部分外載重）。(二) 矩形斷面 50×90 cm，L=15 m，Pi=200 tf，Pe=170 tf（損失 15%），WG=1080 kgf/m，WL=1500 kgf/m；完全預力（不允許拉應力），傳遞壓≤168 kgf/cm²，服務壓≤157.5 kgf/cm²。四控制條件中控制上限的為③傳遞頂拉≥0（ep≤30.18 cm），控制下限的為⑧服務底拉≥0（ep≥27.68 cm），偏心距允許範圍：27.68 cm ≤ ep ≤ 30.18 cm。

## 核心考點

- 曲線腱優點：等效均布向上力 We=8Pep/L²，平衡自重，減小梁彎矩，主動降低撓度與裂縫
- 矩形對稱斷面 Zt=Zb=Z=bh²/6：A=4500 cm²，Z=67500 cm³（h=90 cm）
- 四個控制條件（8 個不等式）：傳遞頂/底、服務頂/底，各兩個壓力與拉力限制
- 損失後 Pe=0.85Pi（Pi=200 tf → Pe=170 tf）
- 完全預力：允許應力中拉力限制=0（最嚴格）

## 解題關鍵步驟

1. A=50×90=4500 cm²，Z=50×90²/6=67500 cm³；Pe=0.85×200000=170000 kgf
2. MG=1080/100×1500²/8=3037500 kgf·cm；ML=1500/100×1500²/8=4218750 kgf·cm
3. 傳遞頂：σt,i=44.44-2.963ep+45.00=89.44-2.963ep；傳遞底：σb,i=-0.556+2.963ep
4. 服務頂：σt,s=37.78-2.519ep+107.5=145.28-2.519ep；服務底：σb,s=-69.72+2.519ep
5. 控制條件：③傳遞頂拉≥0：89.44-2.963ep≥0 → ep≤30.18 cm（上限）
6. ⑧服務底拉≥0：-69.72+2.519ep≥0 → ep≥27.68 cm（下限）
7. 允許範圍：27.68 cm ≤ ep ≤ 30.18 cm（約 2.5 cm 窄範圍）

## 用到的公式

- 曲線腱等效載重：$W_e = 8Pe_p/L^2$
- 斷面性質（矩形）：$A = bh$，$Z = bh^2/6$，$Z_t = Z_b$
- 纖維應力（傳遞）：$\sigma_{t,i} = P_i/A - P_i e_p/Z + M_G/Z$
- 纖維應力（服務）：$\sigma_{t,s} = P_e/A - P_e e_p/Z + (M_G+M_L)/Z$
- 控制條件：$\sigma_{t,i} \ge 0$（上限）；$\sigma_{b,s} \ge 0$（下限）

## 涉及陷阱

- 矩形對稱斷面 Zt=Zb（不必分別計算），計算錯誤易來自誤設不等高斷面
- 傳遞階段用 Pi，服務階段用 Pe（二者不同，損失後縮小）
- 完全預力拉力限制=0（不允許拉應力），使允許範圍最窄；若採用寬鬆標準（允許小量拉應力）範圍會擴大
