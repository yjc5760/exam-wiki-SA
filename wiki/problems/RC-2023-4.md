# RC-2023-4 — 後拉法無握裹鋼絞線預力版撓曲強度

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第4題
**考年：** 2023（民國112年）
**主分類：** [[RC-U4-1]] 預力梁斷面應力分析
**設計法：** USD強度設計法
**標籤：** `後拉法` `無握裹鋼絞線` `fps公式` `L/dp比值` `單向版` `有效預力` `撓曲強度` `β₁折減`
**驗證狀態：** ✅ verified

## 題幹摘要

簡支後拉預力混凝土單向版（$L = 10$ m，$h = 25$ cm，$d_p = 22$ cm），無握裹鋼絞線每根 $A_p = 1.47$ cm²（間距 20 cm），$f_{pu} = 19000$ kgf/cm²，$f_{se} = 11000$ kgf/cm²，$f'_c = 420$ kgf/cm²。求每根鋼絞線對應條帶的計算撓曲強度 $M_n$（不乘 $\phi$）。

## 核心考點

- 無握裹腱 vs. 有握裹腱：無握裹腱全長自由滑動，不能用截面應變相容，只能用 ACI 經驗公式
- $L/d_p$ 比值決定 $f_{ps}$ 公式係數：簡支梁 $L/d_p \geq 35$ 選分母 $300\rho_p$ 公式
- 兩個上限：$f_{ps} \leq f_{py} = 0.85f_{pu}$ 且 $f_{ps} \leq f_{se} + 4200$ kgf/cm²
- 高強混凝土 $f'_c = 420 > 280$ kgf/cm² 需折減 $\beta_1$

## 解題關鍵步驟

1. 計算 $L/d_p = 1000/22 = 45.5 \geq 35$，選用簡支梁公式（分母 $300\rho_p$）
2. 計算配筋率 $\rho_p = A_p/(b \times d_p) = 1.47/(20 \times 22)$（以每根條帶分析）
3. 代入 $f_{ps} = f_{se} + 700 + f'_c/(300\rho_p)$，並驗算兩個上限
4. 計算 $\beta_1$（$f'_c = 420$ kgf/cm² > 280）：$\beta_1 = 0.85 - 0.05(420-280)/70 = 0.75$
5. 計算應力塊高度 $a = A_p f_{ps}/(0.85 f'_c b)$
6. 求 $M_n = A_p f_{ps}(d_p - a/2)$

## 用到的公式

$$f_{ps} = f_{se} + 700 + \frac{f'_c}{300\rho_p} \quad \left(\frac{L}{d_p} \geq 35,\text{ 簡支梁}\right)$$

$$f_{ps} \leq \min\!\left(f_{py},\, f_{se} + 4200\right) \quad \text{(上限雙驗算)}$$

$$\beta_1 = 0.85 - 0.05 \cdot \frac{f'_c - 280}{70} \quad (f'_c > 280\text{ kgf/cm}^2)$$

$$a = \frac{A_p f_{ps}}{0.85 f'_c b}$$

$$M_n = A_p f_{ps}\!\left(d_p - \frac{a}{2}\right)$$

## 涉及陷阱

- $L/d_p$ 判斷公式選擇：簡支梁 $L/d_p \geq 35$ 用分母 $300\rho_p$；若為懸臂（$L/d_p \leq 35$）分母改 $300$。
- 兩個上限都需驗算：$f_{ps} \leq f_{py}$ 和 $f_{ps} \leq f_{se} + 4200$，不能只查一個。
- 高強混凝土 $\beta_1$ 折減：$f'_c = 420$ 時 $\beta_1 = 0.75$，不是 0.85。
- 每根絞線分析單元：$A_p$ 和 $b$ 都以單根/20 cm 條帶計算，維持一致性。
