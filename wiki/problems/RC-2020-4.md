# RC-2020-4 — 預力連續梁一致腱均勻軸壓偏心量設計

## 題幹摘要

兩跨等跨預力連續梁（A-B-C），各跨 $L = 15$ m，均佈靜載重 $w = 1.6$ tf/m，拋物線鋼腱在支承 B 偏心 35 cm（形心以上）。（一）求跨中偏心 $e_D = e_E$ 及有效預力 $P_e$，使全梁無二次彎矩且斷面受均勻軸壓；（二）求在初始預力 $P_i$ 及靜載重下，斷面 B 上下緣應力。

## 核心考點

- 一致腱（concordant tendon）設計原理：鋼腱形狀使預力等效荷載與外力彎矩等值反向
- 均勻軸壓條件：$P_e \cdot e(x) = M_\text{applied}(x)$ 在每個斷面成立（無殘餘彎矩）
- 兩跨等跨連續梁彎矩公式：$M_B = -wL^2/8$，$M_D = M_E = +wL^2/16$
- 小題（二）：$P_i > P_e$，等效上拱力超過外載，B 點出現殘餘正彎矩

## 解題關鍵步驟

1. 計算斷面幾何：$A$、$I$、$S_t = S_b$（矩形斷面）
2. 兩跨等跨彎矩：$M_B = -wL^2/8 = -45$ tf·m，$M_D = +wL^2/16 = +22.5$ tf·m
3. 由均勻軸壓條件（B 點）：$P_e = |M_B|/e_B = 45/0.35 = 128.6$ tf
4. 由均勻軸壓條件（D 點）：$e_D = M_D/P_e = 22.5/128.6 = 17.5$ cm（形心以下）
5. 小題（二）：$P_i = P_e/0.8 = 160.7$ tf；B 點等效上拱力 $q_i > w$，殘餘彎矩 $= +11.25$ tf·m
6. 斷面 B 應力：$\sigma = -P_i/A \pm M_\text{total}/S$（上緣壓力更大，下緣壓力減少）

## 用到的公式

$$P_e \cdot e(x) = M_\text{applied}(x) \quad (\text{均勻軸壓條件})$$

$$M_D = \frac{wL^2}{16}, \quad M_B = -\frac{wL^2}{8} \quad (\text{兩跨等跨連續梁})$$

$$\sigma_{top} = -\frac{P_i}{A} - \frac{P_i e_B}{S_t} + \frac{M_\text{total}}{S_t}$$

$$\sigma_{bot} = -\frac{P_i}{A} + \frac{P_i e_B}{S_b} - \frac{M_\text{total}}{S_b}$$

$$q_\text{eq,i} = \frac{8 P_i f}{L^2} \quad (\text{等效均佈上拱力})$$

## 涉及陷阱

- 兩跨彎矩公式：$M_D = wL^2/16$（非 $wL^2/8$），跨中彎矩是中支點彎矩的一半。
- 小題（二）等效力矩方向：$P_i > P_e$，等效上拱力 $> w$，B 點殘餘彎矩為正（上拱過大）。
- 偏心符號：B 點腱在形心上方（$e_B$ 負號），D 點在形心以下（$e_D$ 正號），方向不同。
- 一致腱幾何意義：$e_B : e_D = -2 : 1$，與外力彎矩圖 $M_B : M_D = -2 : 1$ 成比例。
