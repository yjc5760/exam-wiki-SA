# RC-2020-1 — T形梁有效翼板寬與最少拉力鋼筋量設計

**來源：** 結構工程技師高考 · 鋼筋混凝土設計與預力 · 第1題
**考年：** 2020（民國109年）
**主分類：** [[RC-U1-1]] RC 梁彎矩強度分析與設計
**設計法：** USD強度設計法
**標籤：** `T形梁` `有效翼板寬` `中性軸在翼板` `最少鋼筋量` `Whitney應力塊` `正彎矩設計`
**驗證狀態：** ✅ verified

## 題幹摘要

梁版系統中的 T 形梁，承受設計正彎矩 $M_u = 130$ tf·m（含自重），$f'_c = 210$ kgf/cm²，$f_y = 4200$ kgf/cm²，梁跨度 $L = 500$ cm，版厚 $h_f = 10$ cm，腹板寬 $b_w = 50$ cm，全深 $h = 80$ cm，梁間淨距 400 cm（每側）。要求計算有效翼板寬 $b_e$ 後，設計所需最少拉力鋼筋量 $A_s$，並驗算延性及最小鋼筋量控制。

## 核心考點

- T 形梁有效翼板寬三個控制條件（取最小值）
- 假設中性軸在翼板內（等效矩形梁），驗算 $a \leq h_f$
- USD 設計彎矩強度二次方程求解 $A_s$
- 最小鋼筋量公式中以腹板寬 $b_w$ 計算，非翼板寬 $b_e$

## 解題關鍵步驟

1. 計算有效翼板寬 $b_e$：三條件（$L/4$、$b_w + 16h_f$、$b_w + s_\text{clear}$）取最小值，得 $b_e = 125$ cm
2. 假設中性軸在翼板內，取 $b_e = 125$ cm，假設 $\phi = 0.9$（拉力控制）
3. 列 $\phi M_n = M_u$ 二次方程求解 $A_s$，取小根（延性解）
4. 驗算 $a = A_s f_y / (0.85 f'_c b_e) < h_f = 10$ cm（確認中性軸在翼板）
5. 驗算延性：$\varepsilon_t = 0.003(d-c)/c \geq 0.005$，確認 $\phi = 0.9$
6. 驗核最小鋼筋量：以 $b_w$ 計算 $A_{s,\min}$，取強度需求與最小值兩者較大

## 用到的公式

$$b_e = \min\!\left(\frac{L}{4},\ b_w + 16h_f,\ b_w + s_\text{clear}\right)$$

$$a = \frac{A_s f_y}{0.85 f'_c b_e}$$

$$\phi M_n = \phi A_s f_y \!\left(d - \frac{a}{2}\right) \geq M_u$$

$$\varepsilon_t = \frac{0.003(d - c)}{c}, \quad c = \frac{a}{\beta_1}$$

$$A_{s,\min} = \max\!\left(\frac{0.8\sqrt{f'_c}}{f_y},\, \frac{14}{f_y}\right) b_w d$$

## 涉及陷阱

- 單位換算：$M_u = 130$ tf·m $= 130 \times 10^5$ kgf·cm（乘 $10^5$，非 $10^4$）。
- 三條件全算取最小：不可只算 $L/4$ 一條，需三條件均計算後比較。
- 最小鋼筋量用 $b_w$：T 形梁 $A_{s,\min}$ 公式中用腹板寬 $b_w$，不用有效翼板寬 $b_e$。
- 中性軸驗算不可省略：$a \approx 9.34$ cm 僅稍小於 $h_f = 10$ cm，差距極小，必須驗算。
