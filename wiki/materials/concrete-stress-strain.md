# 混凝土應力-應變行為

## 關鍵參數

| 參數 | 典型值 | 說明 |
|------|--------|------|
| 峰值應力 | f'c | 28天標準圓柱試體抗壓強度 |
| 峰值應變 | ε₀ ≈ 0.002 | 對應 f'c 的應變 |
| 極限壓應變 | εcu = 0.003 | ACI 假設值，用於強度計算 |
| 彈性模數 | Ec = 4700√f'c（MPa） | = 15000√f'c（kgf/cm²） |

## Whitney 等值應力塊假設

實際拋物線形應力分布 → 等效矩形：
- 應力強度：0.85f'c
- 深度：a = β1·c（β1 ≤ 0.85，隨 f'c 增加而折減）

## 圍束混凝土

有圍束箍筋時，f'cc > f'c，εcu 可達 0.005~0.02（Mander 模型）。
