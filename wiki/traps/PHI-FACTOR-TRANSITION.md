# 過渡區 φ 值計算陷阱

**陷阱 ID：** PHI-FACTOR-TRANSITION
**知識分類：** RC-U1-1、RC-U1-2
**危險係數：** ★★★（高頻、高誤率）

---

## 陷阱描述

ACI 318 的 **φ 值取決於最外層拉力鋼筋應變 $\varepsilon_t$**，並非固定值。
考生最常犯的錯誤是：對過渡區（$0.002 < \varepsilon_t < 0.005$）直接套用 $\phi = 0.65$ 或 $\phi = 0.90$。

---

## φ 值區分規則（USD）

| 斷面狀態 | $\varepsilon_t$ 範圍 | $\phi$ 值（矩形箍柱）| $\phi$ 值（螺旋柱）|
|---------|---------------------|-----------------|----------------|
| 壓力控制 | $\varepsilon_t \leq 0.002$ | **0.65** | **0.75** |
| 過渡區 | $0.002 < \varepsilon_t < 0.005$ | **線性插值** | **線性插值** |
| 拉力控制 | $\varepsilon_t \geq 0.005$ | **0.90** | **0.90** |

### 矩形箍柱過渡區插值公式

$$\phi = 0.65 + (\varepsilon_t - 0.002) \times \frac{0.25}{0.003}$$

### 螺旋柱過渡區插值公式

$$\phi = 0.75 + (\varepsilon_t - 0.002) \times \frac{0.15}{0.003}$$

---

## 判斷流程

```
Step 1：用 Whitney 應力塊求中性軸 c
     ↓
Step 2：ε_t = 0.003 × (d_t - c) / c  （d_t = 最外層拉力筋深度）
     ↓
Step 3：比較 ε_t 與 0.002、0.005
     ↓
Step 4：套用對應 φ 值（或插值）
```

---

## 常見陷阱場景

| 陷阱 | 說明 |
|------|------|
| 多排鋼筋的 $d_t$ | $d_t$ 應取**最外層**拉力筋的深度，不是重心深度 $d$ |
| 雙筋梁的 $\varepsilon_t$ | 壓力鋼筋的存在不影響 $\varepsilon_t$ 的計算，$\varepsilon_t$ 只看最外層拉力筋 |
| 過筋梁（$\rho > \rho_b$）| $\varepsilon_t < 0.002$ → $\phi = 0.65$（壓力控制），且鋼筋不降伏，需應變相容 |
| 以 $\varepsilon_t = 0.004$ 為最大配筋控制時 | 此時 $\phi$ 在過渡區，需插值，**不是** 0.90 |
