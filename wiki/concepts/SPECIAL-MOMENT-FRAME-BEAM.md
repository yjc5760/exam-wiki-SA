# 特殊矩形框架梁（耐震梁）

**概念 ID：** SPECIAL-MOMENT-FRAME-BEAM
**知識分類：** RC-U3-3
**規範來源：** ACI 318 §18.6；耐震設計規範

## 定義

特殊矩形框架梁（SMF Beam）是高地震風險區 RC 框架梁依 ACI 318 §18.6 的特殊配筋與構造規定，確保梁端塑鉸具備充分的旋轉能力（延性）。核心要求包含：（1）剪力設計採能力設計法（以 Mpr 計算 Ve）；（2）塑鉸區（距梁端 2h 範圍）設置密箍；（3）當地震剪力為主控時 Vc = 0；（4）縱向鋼筋的最小比例和配置要求。

## 前置概念

- [[SEISMIC-DESIGN]] 能力設計法決定耐震梁的設計剪力
- [[STRONG-COLUMN-WEAK-BEAM]] 耐震梁設計以確保梁端先形成塑鉸為目標

## 關鍵公式

能力設計剪力（正負地震方向取包絡值）：

$$V_e = \frac{M_{pr,l} + M_{pr,r}}{l_n} \pm \frac{w_u l_n}{2}$$

可能彎矩強度（φ = 1.0，fy 換成 1.25fy）：

$$M_{pr} = A_s \cdot 1.25 f_y \cdot \left(d - \frac{a_{pr}}{2}\right)$$

密箍區最大間距（距梁端 2h 範圍）：

$$s \leq \min\!\left(\frac{d}{4},\; 8d_{b,長筋},\; 24d_{b,箍筋},\; 300\text{ mm}\right)$$

Vc = 0 條件（地震剪力主控）：

$$V_e \text{ (地震) } > \frac{V_u}{2} \quad \text{且} \quad N_u < \frac{A_g f'_c}{20}$$

## 常見陷阱

- Mpr 計算中鋼筋強度取 1.25fy，且 φ = 1.0（不含強度折減），此為關鍵
- 密箍區長度從梁端算起不小於 2h（h 為梁全高），不是 2d
- 正負彎矩方向的 Mpr 通常不同（上部 As 和下部 As 不等），兩個方向都需計算
- 非密箍區的箍筋間距不得超過 d/2（一般剪力設計規定仍適用）
