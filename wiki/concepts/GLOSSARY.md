# 鋼筋混凝土設計術語速查表（GLOSSARY）

> 考試常見符號與術語的快速定義，附公式與規範來源。
> 詳細概念說明見各 `wiki/concepts/[CONCEPT-ID].md`。

---

## 材料強度符號

| 符號 | 名稱 | 說明 |
|------|------|------|
| $f'_c$ | 混凝土圓柱抗壓強度 | 標準 28 天養護值，常用 280 kgf/cm²（SD 280）或 350 kgf/cm² |
| $f_y$ | 鋼筋降伏強度 | 常用 2800（SD 280）、4200（SD 420）、5600（SD 560）kgf/cm² |
| $f_{yt}$ | 橫向鋼筋（箍筋）降伏強度 | 通常與 $f_y$ 相同，設計剪力公式中用 $f_{yt}$ |
| $E_s$ | 鋼筋彈性模數 | $E_s = 2{,}040{,}000$ kgf/cm²（= 200 GPa） |
| $E_c$ | 混凝土彈性模數 | $E_c = 15{,}000\sqrt{f'_c}$（kgf/cm² 單位）或 $4700\sqrt{f'_c}$（MPa 單位） |
| $n$ | 彈性模數比 | $n = E_s/E_c$，轉換斷面法使用 |
| $f_r$ | 混凝土破裂模數 | $f_r = 2\sqrt{f'_c}$（kgf/cm²）或 $0.7\sqrt{f'_c}$（MPa），計算開裂彎矩用 |

## 斷面幾何符號

| 符號 | 名稱 | 說明 |
|------|------|------|
| $b$, $b_w$ | 梁寬 / 腹板寬 | T 形梁取 $b_w$ 為腹板寬 |
| $h$ | 梁總深度 | 含保護層 |
| $d$ | 有效深度 | 壓力面至拉力鋼筋合力作用點的距離 |
| $d'$ | 壓力鋼筋中心距壓力面 | 雙筋梁或柱設計使用 |
| $A_g$ | 梁/柱全截面積 | $= b \times h$ |
| $A_s$ | 拉力鋼筋面積 | |
| $A'_s$ | 壓力鋼筋面積 | |
| $A_{st}$ | 柱縱向鋼筋總面積 | $0.01 \leq \rho_g = A_{st}/A_g \leq 0.08$ |
| $I_g$ | 全斷面慣性矩 | 未開裂，含全截面 |
| $I_{cr}$ | 開裂斷面慣性矩 | 僅受壓區 + 轉換鋼筋，用轉換斷面法計算 |

## 梁彎曲設計符號（USD）

| 符號 | 名稱 | 說明 |
|------|------|------|
| $\beta_1$ | 應力塊折減係數 | $f'_c \leq 280$ kgf/cm² → $\beta_1=0.85$；每增 70 減 0.05，最小 0.65 |
| $a$ | 等值矩形應力塊深度 | $a = A_s f_y / (0.85 f'_c b)$；$a = \beta_1 c$ |
| $c$ | 中性軸深度 | 距壓力面距離 |
| $M_n$ | 標稱彎矩強度 | $M_n = A_s f_y (d - a/2)$（單筋梁） |
| $M_u$ | 因數化彎矩需求 | $M_u \leq \phi M_n$ |
| $\phi$ | 強度折減係數 | $\varepsilon_t \geq 0.005$：$\phi=0.90$；$\varepsilon_t = 0.004$：$\phi=0.65/0.75$；過渡區線性插值 |
| $\varepsilon_t$ | 最外層拉力鋼筋應變 | $\varepsilon_t = 0.003(d-c)/c$ |
| $\rho$ | 拉力鋼筋比 | $\rho = A_s/(b \cdot d)$ |
| $\rho_b$ | 平衡鋼筋比 | $\rho_b = 0.85\beta_1 f'_c/f_y \cdot 6120/(6120+f_y)$ |
| $\rho_{min}$ | 最小鋼筋比 | $\max(0.8\sqrt{f'_c}/f_y,\; 14/f_y)$（kgf/cm² 單位） |

## 剪力與扭力符號

| 符號 | 名稱 | 說明 |
|------|------|------|
| $V_c$ | 混凝土剪力強度 | 簡化式：$V_c = (1/6)\sqrt{f'_c}\,b_w\,d$（N,mm）或 $\approx 0.53\sqrt{f'_c}\,b_w\,d$（kgf/cm²） |
| $V_s$ | 箍筋剪力強度 | $V_s = A_v f_{yt} d / s$ |
| $V_n$ | 標稱剪力強度 | $V_n = V_c + V_s$ |
| $A_v$ | 箍筋面積（單排） | |
| $T_n$ | 標稱扭力強度 | 空間桁架法：$T_n = 2 A_o (A_t/s) f_{yt} \cot\theta$ |
| $A_{oh}$ | 閉合箍筋圍合面積 | 閉合箍筋中心線圍成的面積 |
| $A_o$ | 有效受力面積 | $A_o \approx 0.85 A_{oh}$ |
| $p_{oh}$ | 閉合箍筋周長 | 箍筋中心線周長 |

## 柱設計符號

| 符號 | 名稱 | 說明 |
|------|------|------|
| $P_n$ | 標稱軸力強度 | P-M 互制圖上的軸力 |
| $M_n$ | 標稱彎矩強度 | P-M 互制圖上的彎矩 |
| $e$ | 偏心距 | $e = M_u/P_u$ |
| $\phi$ | 強度折減係數（柱） | 純壓控制：0.65（矩形箍）/ 0.75（螺旋）；過渡區線性插值至 0.90 |
| $c_b$ | 平衡點中性軸深度 | $c_b = 6120/(6120+f_y) \cdot d$ |
| $k l_u$ | 有效柱長 | $k$ = 有效長度係數，$l_u$ = 柱淨高 |

## 預力混凝土符號

| 符號 | 名稱 | 說明 |
|------|------|------|
| $f_{pi}$ | 初始預力應力 | 施拉後、扣除即時損失前 |
| $P_e$ | 有效預力 | 扣除所有損失後 |
| $e$ | 預力腱偏心距 | 距斷面形心的距離 |
| $f_{ps}$ | 預力鋼絞線應力（極限） | 依 ACI 318 公式或應變相容法求得 |
| $\Delta f_{pES}$ | 彈性縮短損失 | 先拉：$n \cdot f_{cpa}$；後拉：$(n-1)/(2n) \cdot n \cdot f_{cpa}$ |
| $\Delta f_{pF}$ | 摩擦損失（後拉） | $\Delta f_{pF} = f_{pi}[1-e^{-(\mu\alpha + Kx)}]$ |
