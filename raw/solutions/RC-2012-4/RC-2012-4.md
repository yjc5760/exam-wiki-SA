## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心精神：**
強柱弱梁機制中，梁端塑鉸形成時會對梁柱接頭施加極大水平剪力。外接頭（角隅邊柱）因缺乏對向梁的圍束，接頭剪力強度最低（係數 3.2），故是否充裕須嚴格驗算。

**出題者意圖：**
1. 測試「外接頭水平剪力 $V_{jh}$」的正確推導（以梁筋力 $T$ 減去柱剪力 $V_{col}$）
2. 測試接頭有效面積 $A_j$ 的計算（$b_j \times h_{col}$）
3. 測試圍束條件分類（角隅邊柱 → 外接頭 → 「其他」→ $3.2\sqrt{f'_c}\,A_j$）
4. 測試改善措施的知識（若不足時）

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
```
Step 1：計算接頭水平剪力需求 Vjh（梁頂筋拉力 - 柱剪力）
Step 2：判斷圍束條件 → 選用正確係數（3.2）
Step 3：計算接頭有效面積 Aj = bj × hcol
Step 4：計算接頭剪力設計強度
Step 5：比較 Vjh vs 設計強度 → 結論
```

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | **Vjh 忘記減 Vcol**：接頭剪力 ≠ T，應扣除柱剪力 | $V_{jh} = T_1 - V_{col} = 159.7 - 50.3 = 109.4\ \text{tf}$ |
| 2 | **用 fy 計算 T**：接頭剪力需求來自梁端塑鉸，應用 $f_{ps} = 1.25 f_y$ | $T_1 = A_{s,\text{top}} \times f_{ps} = 30.42 \times 5250 = 159.7\ \text{tf}$ |
| 3 | **接頭面積用梁寬**：$b_j$ 取柱寬與（梁寬 + 柱高）之小值，此題柱寬 60 控制 | $A_j = 60 \times 60 = 3600\ \text{cm}^2$ |
| 4 | **圍束係數用 3.9 或 5.3**：角隅邊柱為外接頭，梁僅從單側框入，屬「其他」→ 3.2 | 係數 = **3.2** |
| 5 | **以正彎矩 Mpr⁺ 計算 Vjh**：應取最大剪力 → EQ→右，頂筋受拉（Mpr⁻）控制 | $T_1$ 用頂筋力 159.7 tf |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`計算 Vjh（水平接頭剪力需求），與設計強度 3.2√f'c·Aj 比較，判斷外接頭是否足夠`

### 本題關鍵公式（依計算順序）

$$\text{Step 1:}\quad T_1 = A_{s,\text{top}} \times f_{ps}$$

$$\text{Step 2:}\quad V_{jh} = \boxed{T_1} - V_{col}$$

$$\text{Step 3:}\quad b_j = \min(b_{\text{col}},\ b_w + h_{\text{col}})$$

$$\text{Step 4:}\quad A_j = \boxed{b_j} \times h_{\text{col}}$$

$$\text{Step 5:}\quad \phi V_n = 3.2\sqrt{f'_c} \times \boxed{A_j}$$

### L1：題目直接給定

| 符號 | 數值 | 說明 |
|------|------|------|
| $A_{s,\text{top}}$ | 30.42 cm² | 梁頂筋 6-#8 |
| $f_{ps}$ | 5250 kgf/cm² | $1.25 f_y$ |
| $V_{col}$ | 50.33 tf | 由 RC-2012-3 得 |
| $b_w$ | 50 cm | 梁寬 |
| $b_{\text{col}} = h_{\text{col}}$ | 60 cm | 方形柱邊長 |
| $f'_c$ | 280 kgf/cm² | |

### L2：需知識點推導

| 符號 | 公式／來源 | 卡關? |
|------|----------|:-----:|
| $T_1$ | $30.42 \times 5250 = 159{,}705\ \text{kgf} = 159.7\ \text{tf}$ | |
| $V_{jh}$ | $159.7 - 50.33 = 109.4\ \text{tf}$ | |
| $b_j$ | $\min(60,\ 50+60) = 60\ \text{cm}$（柱寬控制） | |
| $A_j$ | $60 \times 60 = 3600\ \text{cm}^2$ | |
| $\phi V_n$ | $3.2 \times \sqrt{280} \times 3600 = 3.2 \times 16.73 \times 3600 = 192{,}800\ \text{kgf} = 192.8\ \text{tf}$ | |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|:-----:|
| **外接頭 Vjh 推導** | 梁頂筋拉力 $T_1$ 水平拉入接頭，柱剪力 $V_{col}$ 抵消一部分，因此 $V_{jh} = T_1 - V_{col}$。對向無梁（外接頭），無反側壓力分擔 | |
| **T 用 fps 而非 fy** | 接頭剪力來自梁端塑鉸機構，對應 $M_{pr}$；$M_{pr}$ 用 $f_{ps} = 1.25 f_y$，故 $T_1$ 也用 $f_{ps}$ | |
| **圍束條件分類** | 四面圍束（γ=5.3）→ 四側均有梁框入；三面或對面（γ=3.9）→ 三側有梁或兩對面有梁；其他（γ=3.2）→ 僅一側或兩相鄰側有梁（本題角隅外接頭屬此類） | |
| **bj 計算** | $b_j = \min(b_{\text{col}},\ b_w + h_{\text{col}})$；此題 $\min(60,\ 110) = 60\ \text{cm}$（柱寬控制）。ACI 318 限制 bj 不超過柱寬 | |
| **改善外接頭不足的方法** | ①增大柱斷面（增 $A_j$）；②增加梁柱接頭上下方的垂直（N-S 向）梁，升級圍束條件係數（3.2→3.9→5.3）；③減少梁頂筋量（降低 $T_1$）；④增大 $f'_c$（提升混凝土強度） | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：接頭水平剪力需求 $V_{jh}$

最大剪力需求發生於 EQ→右時，梁端發展 $M_{pr}^-$（頂筋受拉）。

梁頂筋拉力：

$$T_1 = A_{s,\text{top}} \times f_{ps} = 30.42 \times 5{,}250 = \boxed{159{,}705\ \text{kgf} = 159.7\ \text{tf}}$$

外接頭水平剪力（梁力 − 柱剪力）：

$$V_{jh} = T_1 - V_{col} = 159.7 - 50.33 = \boxed{109.4\ \text{tf}}$$

---

### Step 2：判斷圍束條件

本題柱為**結構角隅邊柱**（corner edge column），梁柱外接頭特徵：
- E-W 向僅有**一根梁**從單側框入
- 另三面無梁（或僅有柱，不算圍束）

→ 屬「**其他**」類（exterior joint, one-sided beam）→ 使用係數 **3.2**

---

### Step 3：接頭有效斷面積 $A_j$

有效接頭寬度：

$$b_j = \min(b_{\text{col}},\ b_w + h_{\text{col}}) = \min(60,\ 50 + 60) = \min(60,\ 110) = \boxed{60\ \text{cm}}$$

接頭有效面積（柱寬 × 柱深，方形柱）：

$$A_j = b_j \times h_{\text{col}} = 60 \times 60 = \boxed{3{,}600\ \text{cm}^2}$$

---

### Step 4：接頭剪力設計強度

$$\phi V_n = 3.2\sqrt{f'_c} \times A_j = 3.2 \times \sqrt{280} \times 3{,}600$$

$$= 3.2 \times 16.73 \times 3{,}600 = \boxed{192{,}800\ \text{kgf} = 192.8\ \text{tf}}$$

---

### Step 5：強度檢核

$$V_{jh} = 109.4\ \text{tf} \leq \phi V_n = 192.8\ \text{tf} \quad \boxed{\checkmark\ \text{足夠}}$$

$$\text{強度比} = \frac{109.4}{192.8} = 0.568 \quad \text{（尚有 43\% 餘裕）}$$

---

### 結論

$$\boxed{\text{梁柱外接頭剪力強度足夠，無須改善。}\ V_{jh} = 109.4\ \text{tf} \leq \phi V_n = 192.8\ \text{tf}}$$

---

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 若接頭不足時的改善方案（進階備考）**

雖然本題通過，但若未來遇到接頭不足的情形，標準改善措施：

| 方法 | 機制 | 效果 |
|------|------|------|
| **增大柱斷面** | 增加 $A_j = b_j \times h_{\text{col}}$ | 直接提升接頭強度 |
| **增加垂直向（N-S）梁** | 圍束條件由「其他（3.2）」升為「三面受圍束（3.9）」，強度提升 22% | 改善效果顯著 |
| **降低梁頂筋量** | 減少 $T_1$ → 降低 $V_{jh}$ | 但可能影響梁彎矩強度 |
| **提高混凝土強度** | $\phi V_n \propto \sqrt{f'_c}$ | 效果有限（取 √f'c） |
| **確保接頭箍筋密置** | 維持圍束條件分類適用性 | 如本題「接頭區箍筋配筋同柱箍筋」已達此目標 |

**② 為何外接頭係數最低（3.2 vs 3.9 vs 5.3）**

接頭剪力由接頭核心混凝土的對角壓力桿（diagonal strut）機制承擔。梁從更多方向框入，能對接頭核心提供更好的圍束效果，使混凝土能承受更大的斜向壓力。角隅外接頭（僅單側有梁）圍束最差，故係數最低（3.2）。

**③ 接頭剪力與 $V_{col}$ 的方向關係**

$V_{col}$ 從柱傳入接頭，方向與 $T_1$ 相反，因此接頭的有效水平剪力 $= T_1 - V_{col}$（非 $T_1 + V_{col}$）。直覺記法：柱剪力「幫忙抵消」一部分梁筋拉力，所以接頭剪力 < 梁筋全部拉力。
