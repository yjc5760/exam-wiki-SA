**設計法：** USD 強度設計法  
**標籤：** `矩形梁` `開裂扭矩` `扭矩剪力組合` `斷面適當性檢核` `空間桁架類比` `Aoh計算` `閉合箍筋` `Tu=Tcr條件`

## 1. 原始題目重述 (Problem Restatement)

矩形 RC 梁斷面：$b = 35$ cm，$d = 53$ cm，$h = 60$ cm  
材料：$f'_c = 280$ kgf/cm²，$f_y = f_{yt} = 2800$ kgf/cm²  
配置：#4@8 cm 閉合箍筋，淨保護層 4 cm  
#4 鋼筋：$d_b = 1.27$ cm，$A_b = 1.267$ cm²（一根）

**已知條件：** 梁承受扭矩 $T_u = T_{cr}$（等於此斷面之開裂扭矩）

**求：** 在此扭矩條件下，梁斷面尚可承受之**最大設計剪力** $V_u$。（20 分）

> 本題為純文字題，無附圖，無截圖需求。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 矩形實心斷面在扭矩與剪力**同時作用**下，受兩個限制：
1. **箍筋能力**：箍筋必須同時抵抗扭矩與剪力，兩者共享
2. **斷面適當性**：混凝土受壓斜撐不得壓碎（ACI §11.6.3.1），此為**上限**

**出題者意圖：**
1. 測驗是否能正確計算開裂扭矩 $T_{cr} = (\sqrt{f'_c}/3) \times A_{cp}^2/p_{cp}$
2. 測驗 $A_{oh}$、$p_h$ 的幾何計算（保護層 + 箍筋半徑）
3. 測驗聯合扭矩-剪力斷面適當性公式（ACI §11.6.3.1）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 求 $A_{cp}$、$p_{cp}$ → 算 $T_{cr}$
2. 求 $A_{oh}$、$p_h$（需考慮保護層 + 箍筋半徑至中心線）
3. 計算 $V_c$（混凝土剪力貢獻）
4. 代入斷面適當性公式，解出 $V_u$
5. 驗算箍筋是否足夠（確認控制條件）

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ❶ $A_{oh}$ 以箍筋中心線圍成 | $A_{oh} \ne b\times h$，需扣掉保護層 + 箍筋半徑 | 各邊扣除 $cc + d_b/2$ |
| ❷ $T_{cr}$ 用 $A_{cp}$（全斷面），$A_{oh}$ 用箍筋中心線圍成面積 | 兩個面積定義不同 | $A_{cp}$：全斷面；$A_{oh}$：箍筋中心線 |
| ❸ 兩個控制條件 | 斷面適當性（壓碎限制）與箍筋能力各自限制 $V_u$，取小值 | 兩個都算，看哪個控制 |
| ❹ $\phi = 0.75$（剪力與扭力） | 不是 0.9，也不是 0.65 | ACI §9.3.2.3：剪力扭力取 $\phi = 0.75$ |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

在 $T_u = T_{cr}$ 已固定之條件下，求此矩形梁斷面能額外承受的最大設計剪力 $V_u$。

### 本題關鍵公式（依計算順序）

$$\text{Step 1：} T_{cr} = \frac{\sqrt{f'_c}}{3}\cdot\frac{A_{cp}^2}{p_{cp}}$$

$$\text{Step 2：} A_{oh} = x_{oh}\cdot y_{oh},\quad x_{oh} = b - 2\!\left(cc + \frac{d_b}{2}\right),\quad p_h = 2(x_{oh}+y_{oh})$$

$$\text{Step 3：} V_c = 0.53\sqrt{f'_c}\cdot b_w\cdot d$$

$$\text{Step 4（斷面適當性）：} \sqrt{\left(\frac{V_u}{b_w d}\right)^2 + \left(\frac{\boxed{T_{cr}}\cdot\boxed{p_h}}{1.7\,\boxed{A_{oh}}^2}\right)^2} \le \phi\!\left(\frac{V_c}{b_w d} + 2.12\sqrt{f'_c}\right)$$

$$\text{Step 5（箍筋驗算）：} \frac{A_t}{s}\bigg|_{req,T} = \frac{T_u}{\phi\cdot2\cdot A_o\cdot f_{yt}},\quad \frac{A_v}{s}\bigg|_{req,V} = \frac{V_u - \phi V_c}{\phi\cdot f_{yt}\cdot d}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：全斷面幾何

$$A_{cp} = b\times h = 35\times60 = 2100 \text{ cm}^2$$

$$p_{cp} = 2(b+h) = 2(35+60) = 190 \text{ cm}$$

---

### Step 2：開裂扭矩 $T_{cr}$

$$T_{cr} = \frac{\sqrt{f'_c}}{3}\cdot\frac{A_{cp}^2}{p_{cp}} = \frac{\sqrt{280}}{3}\times\frac{2100^2}{190}$$

$$= \frac{16.733}{3}\times\frac{4{,}410{,}000}{190} = 5.578\times23{,}210.5 = \boxed{129{,}444 \text{ kgf·cm}}$$

---

### Step 3：箍筋中心線幾何（$A_{oh}$、$p_h$）

各邊面到箍筋中心線距離：
$$cc + \frac{d_b}{2} = 4 + \frac{1.27}{2} = 4 + 0.635 = 4.635 \text{ cm}$$

$$x_{oh} = b - 2\times4.635 = 35 - 9.27 = 25.73 \text{ cm}$$

$$y_{oh} = h - 2\times4.635 = 60 - 9.27 = 50.73 \text{ cm}$$

$$\boxed{A_{oh} = 25.73\times50.73 = 1305.3 \text{ cm}^2}$$

$$p_h = 2(x_{oh}+y_{oh}) = 2(25.73+50.73) = 2\times76.46 = 152.92 \text{ cm}$$

$$A_o = 0.85\times A_{oh} = 0.85\times1305.3 = 1109.5 \text{ cm}^2$$

---

### Step 4：混凝土剪力強度 $V_c$

$$V_c = 0.53\sqrt{f'_c}\cdot b_w\cdot d = 0.53\times\sqrt{280}\times35\times53$$

$$= 0.53\times16.733\times1855 = 8.869\times1855 = 16{,}452 \text{ kgf}$$

---

### Step 5：扭矩對應的剪應力 $\tau_t$

$$\tau_t = \frac{T_u\cdot p_h}{1.7\cdot A_{oh}^2} = \frac{129{,}444\times152.92}{1.7\times1305.3^2}$$

$$= \frac{19{,}793{,}377}{1.7\times1{,}703{,}808} = \frac{19{,}793{,}377}{2{,}896{,}474} = \boxed{6.83 \text{ kgf/cm}^2}$$

---

### Step 6：斷面適當性限制（ACI §11.6.3.1）

實心斷面檢核：

$$\sqrt{\left(\frac{V_u}{b_w d}\right)^2 + \left(\frac{T_u\cdot p_h}{1.7\cdot A_{oh}^2}\right)^2} \le \phi\left(\frac{V_c}{b_w d} + 2.12\sqrt{f'_c}\right) \quad [\text{kgf/cm}^2]$$

計算右側上限（$\phi = 0.75$）：

$$2.12\sqrt{f'_c} = 2.12\times\sqrt{280} = 2.12\times16.733 = 35.47 \text{ kgf/cm}^2$$

$$\frac{V_c}{b_w d} = \frac{16{,}452}{35\times53} = \frac{16{,}452}{1855} = 8.87 \text{ kgf/cm}^2$$

$$\text{上限} = \phi\!\left(\frac{V_c}{b_w d} + 2.12\sqrt{f'_c}\right) = 0.75\times(8.87+35.47) = 0.75\times44.34 = \boxed{33.26 \text{ kgf/cm}^2}$$

**代入已知 $\tau_t = 6.83$ kgf/cm²，解 $V_u$：**

$$\sqrt{\left(\frac{V_u}{1855}\right)^2 + 6.83^2} = 33.26$$

$$\left(\frac{V_u}{1855}\right)^2 = 33.26^2 - 6.83^2 = 1{,}106.2 - 46.6 = 1{,}059.6$$

$$\frac{V_u}{1855} = \sqrt{1{,}059.6} = 32.55 \text{ kgf/cm}^2$$

$$\boxed{V_u = 32.55\times1855 = 60{,}380 \text{ kgf} \approx 60.4 \text{ tf}}$$

---

### Step 7：驗算箍筋是否足夠

**扭矩所需 $A_t/s$：**

$$\frac{A_t}{s}\bigg|_{req,T} = \frac{T_u}{\phi\cdot2\cdot A_o\cdot f_{yt}} = \frac{129{,}444}{0.75\times2\times1109.5\times2800} = \frac{129{,}444}{4{,}659{,}900} = 0.02778 \text{ cm}^2/\text{cm}$$

**剪力所需 $A_v/s$：**

$$\phi V_c = 0.75\times16{,}452 = 12{,}339 \text{ kgf}$$

$$\frac{A_v}{s}\bigg|_{req,V} = \frac{V_u - \phi V_c}{\phi\cdot f_{yt}\cdot d} = \frac{60{,}380 - 12{,}339}{0.75\times2800\times53} = \frac{48{,}041}{111{,}300} = 0.4317 \text{ cm}^2/\text{cm}$$

**合計需求 vs. 提供量：**

$$\left(\frac{A_v}{s} + 2\frac{A_t}{s}\right)_{req} = 0.4317 + 2\times0.02778 = 0.4317 + 0.05556 = 0.487 \text{ cm}^2/\text{cm}$$

$$\left(\frac{A_v+2A_t}{s}\right)_{prov} = \frac{4\times A_b}{s} = \frac{4\times1.267}{8} = 0.634 \text{ cm}^2/\text{cm}$$

$$0.487 < 0.634 \quad \checkmark \text{（箍筋充足）}$$

**結論：斷面適當性（Step 6）控制，$V_u$ 由混凝土壓碎上限決定。**

$$\boxed{V_u = 60.4 \text{ tf}}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 為何斷面適當性才是控制條件？**

本題箍筋 #4@8cm 非常密，箍筋能力 $(A_v+2A_t)/s = 0.634$ cm²/cm 遠大於需求 0.487 cm²/cm。箍筋不是限制，而是混凝土斜壓撐的上限（section adequacy）才是。

若改用較稀疏的箍筋（如 #3@20cm），箍筋能力將成為控制因素。

**② $2.12\sqrt{f'_c}$（kgf/cm² 系統）的來源**

| 單位系統 | 公式 | f'c=280 kgf/cm² 時 |
|----------|------|---------------------|
| SI（MPa） | $0.66\sqrt{f'_c\text{[MPa]}}$ | $0.66\times\sqrt{27.46} = 3.46$ MPa $= 35.3$ kgf/cm² |
| 美制（psi） | $8\sqrt{f'_c\text{[psi]}}$ | $8\times\sqrt{3984} = 505$ psi $= 35.5$ kgf/cm² |
| 台灣（kgf/cm²） | $2.12\sqrt{f'_c\text{[kgf/cm²]}}$ | $2.12\times16.73 = 35.5$ kgf/cm² |

三者等效，只是單位換算。

**③ 開裂扭矩 $T_{cr}$ 的物理意義**

$T_{cr} = (\sqrt{f'_c}/3) \times A_{cp}^2/p_{cp}$ 來自薄壁管理論（thin-walled tube analogy）：
- 等效薄壁厚度 $t \approx A_{cp}/(4\times p_{cp})$... 但實際上 ACI 使用 $A_{cp}^2/p_{cp}$ 的形式
- 裂縫時主拉應力達 $\approx (1/3)\sqrt{f'_c}$（混凝土拉裂強度）

**④ 考場速解提示**

本題的計算順序：$T_{cr}$ → $A_{oh}$ → $V_c$ → 斷面適當性公式反解 $V_u$

不需要解聯立方程式，但需要留意：
- $\tau_t = T_{cr} \times p_h / (1.7 \times A_{oh}^2)$ 是一個已知的「扭矩剪應力」
- 斷面限制是一個圓方程：$\tau_v^2 + \tau_t^2 \le R^2$，解 $\tau_v$ 後乘以 $b_w d$ 即得 $V_u$
