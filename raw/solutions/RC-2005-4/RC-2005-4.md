# 考題編號：RC-2005-4

**主分類：** `RC-U3-1` 梁工作性要求（含撓度、裂縫）
**副分類：** 無
**設計法：** 概念題
**標籤：** `潛變` `潛變係數Cc` `長期變形` `彈性縮短` `軸向變形` `混凝土時間相依行為` `純壓柱變形` `Ec計算`

## 1. 原始題目重述 (Problem Restatement)

**（甲）定義題（10 分）**
何謂混凝土之潛變與潛變係數？潛變對鋼筋混凝土構件之行為有何影響？

**（乙）計算題（10 分）**
一混凝土矩形柱長 1.5 m，斷面為 25 cm × 35 cm，採用混凝土強度 $f'_c = 280\ \text{kgf/cm}^2$；若此柱承受軸向壓力 $P = 15{,}000\ \text{kgf}$，試求三年後柱之**總軸向變形**。（已知三年後潛變係數 $C_c = 3.0$，不計收縮變形）

**已知條件：**

| 參數 | 數值 |
|------|------|
| 柱長 $L$ | $150\ \text{cm}$ |
| 斷面 | $25 \times 35\ \text{cm}$ |
| $f'_c$ | $280\ \text{kgf/cm}^2$ |
| 軸向壓力 $P$ | $15{,}000\ \text{kgf}$ |
| 潛變係數 $C_c$（3 年後）| $3.0$ |

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 混凝土的時間相依行為——區分**彈性應變**（立即）和**潛變應變**（長期），以潛變係數串聯兩者。

**測驗重點：**
1. 潛變的定義及其實質影響（定性分析，影響梁撓度、柱載重重分配、預力損失）
2. 計算彈性變形 $\delta_e = PL/(E_c A)$，其中 $E_c = 15{,}000\sqrt{f'_c}$（kgf/cm² 制）
3. 總變形 = 彈性變形 × $(1 + C_c)$

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 回答定義：潛變 = 持續應力下隨時間增加的應變；$C_c$ = 潛變應變 ÷ 彈性應變
2. 條列潛變的影響（梁撓度、柱載重重分配、預力損失、超靜定力重分佈）
3. 計算 $E_c$、$\delta_e$、$\delta_{cr}$、$\delta_{total}$

**關鍵陷阱（2 個）：**
- ⚠ **總變形不是只加潛變**：$\delta_{total} = \delta_e + \delta_{cr} = \delta_e(1+C_c)$，必須保留彈性部分
- ⚠ **$E_c$ 公式**：台灣 kgf/cm² 制：$E_c = 15{,}000\sqrt{f'_c}$，勿與 SI 制 $4700\sqrt{f'_c}$（MPa）混淆

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標
三年後柱的總軸向壓縮變形 $\delta_{total}$（cm）。

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } E_c = 15{,}000\sqrt{f'_c}\ (\text{kgf/cm}^2)$$

$$\text{Step 2: } \delta_e = \frac{P \cdot L}{E_c \cdot A_c}$$

$$\text{Step 3: } \delta_{cr} = C_c \cdot \boxed{\delta_e}$$

$$\text{Step 4: } \delta_{total} = \boxed{\delta_e} + \boxed{\delta_{cr}} = \boxed{\delta_e}\cdot(1 + C_c)$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 甲：潛變定義與影響（概念）

**（一）潛變（Creep）的定義**

混凝土在**持續不變的應力**下，隨時間推移產生超過即時彈性應變的附加應變，稱為**潛變**。其特性如下：
- 潛變在施載初期發展最快，爾後速率遞減，趨近最終值
- 溫度升高、濕度降低、水灰比增大、齡期短、應力高時，潛變增大

**（二）潛變係數（Creep Coefficient，$C_c$）的定義**

$$C_c = \frac{\varepsilon_{cr}}{\varepsilon_e}$$

其中：
- $\varepsilon_{cr}$ = 潛變應變（隨時間發展之附加應變）
- $\varepsilon_e$ = 彈性應變（施載瞬間的即時應變）

$C_c$ 代表「潛變應變是彈性應變的幾倍」，一般在長期作用下 $C_c = 1.5 \sim 3.0$。

**（三）潛變對 RC 構件行為的影響**

| 構件類型 | 影響 |
|----------|------|
| **RC 梁、版** | 長期撓度增加。ACI 規定：長期附加撓度 = $\lambda \cdot \Delta_{i(DL)}$，其中 $\lambda = \xi/(1+50\rho')$，反映潛變（及收縮）累積 |
| **RC 柱（鋼筋混凝土）** | 混凝土潛變使軸力重分配——混凝土收縮後，鋼筋承擔更多壓力，可能使鋼筋達到降伏應力 |
| **預力混凝土** | 潛變使混凝土軸向縮短，導致鋼腱鬆弛，即**預力損失**（潛變損失 $\Delta f_{pCR}$） |
| **超靜定結構** | 潛變引起的變形會在超靜定結構中重分配內力（次彎矩），長期下需特別注意 |

### 乙：柱總軸向變形計算

#### ① 混凝土彈性模數 $E_c$

$$E_c = 15{,}000\sqrt{f'_c} = 15{,}000 \times \sqrt{280} = 15{,}000 \times 16.733 = 250{,}995 \approx 251{,}000\ \text{kgf/cm}^2$$

#### ② 斷面積 $A_c$

$$A_c = 25 \times 35 = 875\ \text{cm}^2$$

#### ③ 柱平均應力（檢核）

$$f_c = \frac{P}{A_c} = \frac{15{,}000}{875} = 17.1\ \text{kgf/cm}^2$$

$$\frac{f_c}{f'_c} = \frac{17.1}{280} = 0.061 \ll 0.45 \quad \checkmark \text{（應力遠低於允許壓應力，彈性假設有效）}$$

#### ④ 彈性軸向變形 $\delta_e$

$$\delta_e = \frac{P \cdot L}{E_c \cdot A_c} = \frac{15{,}000 \times 150}{251{,}000 \times 875} = \frac{2{,}250{,}000}{219{,}625{,}000} = 0.01025\ \text{cm}$$

#### ⑤ 潛變軸向變形 $\delta_{cr}$

$$\delta_{cr} = C_c \times \delta_e = 3.0 \times 0.01025 = 0.03075\ \text{cm}$$

#### ⑥ 三年後總軸向變形 $\delta_{total}$

$$\delta_{total} = \delta_e + \delta_{cr} = \delta_e (1 + C_c) = 0.01025 \times (1 + 3.0) = 0.01025 \times 4.0$$

$$\boxed{\delta_{total} = 0.041\ \text{cm} = 0.41\ \text{mm}}$$

**變形組成彙整：**

| 變形類型 | 數值 (cm) | 比例 |
|----------|----------|------|
| 彈性變形 $\delta_e$ | $0.0103$ | 25% |
| 潛變變形 $\delta_{cr}$ | $0.0308$ | 75% |
| **總變形 $\delta_{total}$** | **$0.0411$** | **100%** |

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 潛變係數的實際範圍**

ACI 209 報告指出，長期（50 年）潛變係數 $C_c$ 的終極值約為 1.5～3.0，視下列因素而定：
- 養護條件（水中 < 空氣中）
- 施載齡期（越早施載，潛變越大）
- 水灰比（越大，潛變越大）
- 構件尺寸（越小，潛變越大）

本題給定 $C_c = 3.0$（3 年後），屬於上限值，說明環境條件不利或施載較早。

**② RC 柱（有鋼筋）的長期變形**

若柱有鋼筋（鋼筋面積 $A_s$），由於混凝土潛變縮短，鋼筋被迫額外壓縮，造成**荷載重分配**：混凝土應力降低，鋼筋應力上升。長期應力重分配量為：

$$\Delta f_s = n \cdot f_{c,0} \cdot \frac{C_c \cdot n \rho}{1 + n \rho (1 + C_c)}$$

（$n = E_s/E_c$，$\rho = A_s/A_c$，$f_{c,0}$ = 初始混凝土應力）

本題為純混凝土柱，不需此修正。

**③ 為何不計收縮變形**

混凝土收縮（乾縮）是無應力作用下因水分散失引起的體積收縮，與載重無關。題目排除收縮，讓計算聚焦於潛變機制，是合理的簡化。實務上，收縮量級與潛變相當，需一併計算才能得到真實長期變形。
