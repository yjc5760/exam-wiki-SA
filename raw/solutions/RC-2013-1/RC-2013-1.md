# 考題編號：RC-2013-1

**主分類：** `RC-U3-2` 樓版與基腳設計
**副分類：** （無）
**設計法：** USD 強度設計法
**標籤：** `單向版` `多跨連續版` `最小版厚` `ACI彎矩係數法` `溫度收縮鋼筋` `外支承負彎矩` `撓度控制` `版厚設計`

---

## 1. 原始題目重述 (Problem Restatement)

已知一**多跨連續的鋼筋混凝土單向版**：

- 梁支承中心跨度 $L = 4.8\ \text{m}$，梁寬 $b_w = 30\ \text{cm}$
- 活載重 $w_L = 0.6\ \text{tf/m}^2$，靜載重（不含自重）$w_{SDL} = 0.1\ \text{tf/m}^2$
- $f'_c = 280\ \text{kgf/cm}^2$，$f_y = 4{,}200\ \text{kgf/cm}^2$，$E_s = 2.04\times10^6\ \text{kgf/cm}^2$

試依規範規定：（25 分）
1. **設計版厚**（最小厚度）
2. **設計外支承處之撓曲鋼筋**（integral beam support 情況）

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心精神：**
連續單向版設計採 **ACI 彎矩係數法**（Moment Coefficient Method）：先由撓度控制決定最小版厚，再計算各關鍵斷面之設計彎矩，確認撓曲鋼筋需求量——此題考點在於「最小鋼量（溫度收縮筋）控制」的識別。

**外支承彎矩係數：**
版與端梁一體澆築（integral with spandrel beam）→ 外支承處係數 $= 1/24$（負彎矩），若外支承為簡支（unrestrained），係數 = 0，但仍需配溫度鋼筋。

**出題者意圖：**
1. 測試 $l_n/24$ 版厚公式（一端連續）
2. 測試 ACI 彎矩係數法及外支承係數選用
3. 測試溫度收縮筋（$\rho_{min} = 0.0018$）是否控制（通常外支承彎矩小，最小鋼量控制）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

```
Step 1：淨跨 l_n = L_cc - b_w = 4.8 - 0.3 = 4.5 m = 450 cm
Step 2：版厚 h_min = l_n/24（一端連續，外跨控制）→ 取整
Step 3：計算自重、wu（載重因數組合）
Step 4：外支承負彎矩 Mu = wu × l_n²/24（係數法）
Step 5：求 As_flex，比較 As_min = 0.0018 × b × h → 取大者
Step 6：選擇鋼筋間距
```

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | **版厚用 c-c 跨而非淨跨**：有些題用 L_cc，有些用 l_n | CNS/ACI Table 9.5(a)：用**淨跨 l_n**；h_min = l_n/24 |
| 2 | **外支承彎矩係數搞錯**：係數 1/16（柱）vs 1/24（端梁） | 此題支承為端梁（邊梁）→ 係數 **1/24**（負彎矩） |
| 3 | **忘記溫度收縮筋最小量**：外支承彎矩小，直接套公式得到很小的 As | fy=4200 時 ρ_min = **0.0018**；As_min = 0.0018×b×h → 通常控制 |
| 4 | **保護層混淆**：樓版 clear cover = 2 cm（室內），外露版 = 3 cm | 本題未特別指明 → 用 **2 cm**（室內版標準） |
| 5 | **最大鋼筋間距忽略**：s ≤ min(3h, 45 cm) | 任何版鋼筋間距都必須驗算此限制 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`決定版厚 h（撓度控制）；設計外支承處頂部撓曲鋼筋（負彎矩，最小鋼量控制）`

### 本題關鍵公式（依計算順序）

$$\text{Step 1:}\quad l_n = L_{cc} - b_w,\quad h_{\min} = l_n/24 \text{（外跨，一端連續）}$$

$$\text{Step 2:}\quad w_u = 1.2\,(w_{SW} + w_{SDL}) + 1.6\,w_L \quad \text{（} w_{SW} = 2400\,h \text{ kgf/m}^2\text{）}$$

$$\text{Step 3:}\quad M_u = \frac{w_u\,l_n^2}{24} \quad \text{（外支承負彎矩係數 1/24）}$$

$$\text{Step 4:}\quad A_{s,\text{flex}} = \rho\,b\,d,\quad A_{s,\min} = 0.0018\,b\,h \quad \Rightarrow \quad A_s = \max(A_{s,\text{flex}},\,A_{s,\min})$$

$$\text{Step 5:}\quad s = \frac{A_b}{A_s}\times b \leq \min(3h,\ 45\ \text{cm})$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Part 1：設計版厚

**淨跨：**
$$l_n = L_{cc} - b_w = 480 - 30 = \boxed{450\ \text{cm}}$$

**最小版厚（CNS/ACI Table 9.5，一端連續，適用 $f_y = 4200\ \text{kgf/cm}^2$）：**

$$h_{\min} = \frac{l_n}{24} = \frac{450}{24} = 18.75\ \text{cm}$$

圓整至整數，且需 ≥ 12 cm（結構板最小厚度）：

$$\boxed{h = 19\ \text{cm}}$$

（外跨適用「一端連續」；內跨為「兩端連續」= $l_n/28 = 16.1$ cm < 19 cm，故外跨控制）

---

### Part 2：外支承處撓曲鋼筋

**載重計算：**

$$w_{SW} = 2400\ \text{kgf/m}^3 \times 0.19\ \text{m} = 456\ \text{kgf/m}^2 = 0.456\ \text{tf/m}^2$$

$$w_D = w_{SDL} + w_{SW} = 0.1 + 0.456 = 0.556\ \text{tf/m}^2$$

$$w_u = 1.2\,w_D + 1.6\,w_L = 1.2\times0.556 + 1.6\times0.600 = 0.667 + 0.960 = \boxed{1.627\ \text{tf/m}^2}$$

取 1 m 寬設計帶：$w_u = 1.627\ \text{tf/m} = 16.27\ \text{kgf/cm}$

**驗算 ACI 彎矩係數法適用條件：**
- 跨數 ≥ 2 ✓；各跨相等 ✓；均布載重 ✓
- $w_L/w_D = 0.6/0.556 = 1.08 \leq 3$ ✓

**外支承負彎矩（版與端梁一體澆築，係數 = 1/24）：**

$$M_u = \frac{w_u\,l_n^2}{24} = \frac{16.27 \times 450^2}{24} = \frac{16.27 \times 202{,}500}{24} = \frac{3{,}294{,}675}{24} = \boxed{137{,}278\ \text{kgf·cm/m}}$$

（= 1.373 tf·m/m）

**有效深度（D10 bars，clear cover = 2 cm）：**

$$d = h - c_{cover} - \frac{d_b}{2} = 19 - 2 - \frac{0.96}{2} = 19 - 2 - 0.48 = \boxed{16.52\ \text{cm}}$$

**撓曲設計（$b = 100$ cm per 1 m strip，$\phi = 0.9$）：**

$$R_n = \frac{M_u}{\phi\,b\,d^2} = \frac{137{,}278}{0.9\times100\times(16.52)^2} = \frac{137{,}278}{0.9\times100\times272.91} = \frac{137{,}278}{24{,}562} = 5.59\ \text{kgf/cm}^2$$

$$\rho_{\text{flex}} = \frac{0.85\,f'_c}{f_y}\!\left(1 - \sqrt{1 - \frac{2\,R_n}{0.85\,f'_c}}\right) = \frac{0.85\times280}{4200}\!\left(1 - \sqrt{1 - \frac{2\times5.59}{238}}\right)$$

$$= 0.05667 \times \!\left(1 - \sqrt{0.9530}\right) = 0.05667 \times (1 - 0.9762) = 0.05667 \times 0.02380 = 0.001349$$

$$A_{s,\text{flex}} = \rho_{\text{flex}}\,b\,d = 0.001349 \times 100 \times 16.52 = \boxed{2.23\ \text{cm}^2/\text{m}}$$

**最小鋼筋量（溫度收縮，$f_y = 4200$，SD420）：**

$$A_{s,\min} = \rho_t \times b \times h = 0.0018 \times 100 \times 19 = \boxed{3.42\ \text{cm}^2/\text{m}}$$

$$A_{s,\min} = 3.42 > A_{s,\text{flex}} = 2.23 \quad \Rightarrow \quad \textbf{溫度收縮筋最小量控制}$$

**使用鋼筋量：** $A_s = 3.42\ \text{cm}^2/\text{m}$

**選用鋼筋（D10，$A_b = 0.71\ \text{cm}^2$）：**

$$s = \frac{A_b \times b}{A_s} = \frac{0.71 \times 100}{3.42} = 20.76\ \text{cm} \quad \Rightarrow \quad \text{取} s = 20\ \text{cm}$$

$$A_{s,\text{provided}} = \frac{0.71 \times 100}{20} = 3.55\ \text{cm}^2/\text{m} \geq 3.42\ ✓$$

**最大間距驗算：**

$$s_{\max} = \min(3h,\ 45\ \text{cm}) = \min(57,\ 45) = 45\ \text{cm}$$

$$s = 20\ \text{cm} \leq 45\ \text{cm}\ ✓$$

$$\boxed{\text{外支承處頂部撓曲鋼筋：D10@20 cm（}A_s = 3.55\ \text{cm}^2/\text{m}\text{）}}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 外支承負彎矩係數：1/24 或 0？**

版規範明確區分：
- 外支承與端梁一體（integral with spandrel beam）→ 係數 **1/24**（需配頂筋）
- 外支承與柱一體 → 係數 **1/16**（較大負彎矩）
- 外支承為簡支（unrestrained）→ 係數 **0**（不需負彎矩筋，但仍需 $A_{s,\min}$ 作為溫度筋）

考場策略：題目說「多跨連續版」且有外支承設計問題 → 通常假設端梁一體，用 **1/24**。若題目明確說「簡支」則係數 = 0。

**② 最小版厚 19 cm vs 20 cm**

若使用中心跨距 $L_{cc} = 480\ \text{cm}$ 代入（不扣梁寬）：$480/24 = 20\ \text{cm}$。兩種作法在考場上都可接受，差異僅 1 cm。採淨跨 $l_n = 450\ \text{cm}$ 更符合 CNS 規範條文定義，但採中心跨距是台灣考場常見的快速計算方式。

**③ 溫度收縮筋控制的工程意義**

外支承負彎矩 $M_u = 1.37\ \text{tf·m/m}$ 遠小於第一內支承 $3.29\ \text{tf·m/m}$。外支承的設計鋼筋量（2.23 cm²/m）小於溫度收縮最小量（3.42 cm²/m），這說明外支承是本版「最不危險」的斷面，溫度裂縫控制是設計驅動力。這在薄版設計中很常見。
