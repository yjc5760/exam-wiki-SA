# 考題編號：RC-2016-3

**主分類：** `RC-U1-3` 細長柱
**副分類：** `RC-U1-2` RC 柱強度分析與設計
**設計法：** USD強度設計法
**標籤：** `細長柱` `無側移構架` `放大彎矩法` `Cm係數` `雙曲度` `臨界載重Pc` `δns計算`

---

## 1. 原始題目重述（Problem Restatement）

無側移構架中之矩形柱，資料如下：

- **斷面：** 40 cm × 40 cm 正方形
- **淨高：** $l_u = 8$ m = 800 cm
- **有效長度係數：** $K = 0.9$
- **EI 值：** $EI = 0.25\,E_c I_g$
- **材料：** $f'_c = 280 \text{ kgf/cm}^2$，$f_y = 4200 \text{ kgf/cm}^2$
- **極限軸力：** $P_u = 130$ tf
- **端部彎矩：** 一端 $M_1 = 35$ tf-m（較小端），另一端 $M_2 = 40$ tf-m（較大端）
- **受力型式：** 雙曲度（S 形）

**試求設計彎矩與軸力。（20 分）**

考卷提供參考公式（已由考卷截取）：

$$E_c = 15000\sqrt{f'_c} \text{ kgf/cm}^2$$

$$\frac{kl_u}{r} \leq 34 - 12\frac{M_1}{M_2}$$

$$\delta_{ns} = \frac{C_m}{1 - P_u/(0.75P_c)} \geq 1.0$$

$$C_m = 0.6 + 0.4\frac{M_1}{M_2} \geq 0.4$$

$$P_c = \frac{\pi^2 EI}{(kl_u)^2}$$

> **無附圖：** 純文字敘述題，無需截圖。

---

## 2. 考題核心精神與出題者意圖（Core Concepts & Examiner's Intent）

**核心觀念：**
1. **細長柱判斷**：以 $kl_u/r$ 與臨界值 $34-12(M_1/M_2)$ 比較，決定是否需要放大彎矩
2. **雙曲度符號慣例**：雙曲度時 $M_1/M_2$ 取**負值**（ACI/CNS 慣例），使 $C_m$ 降低、放大係數相對小
3. **放大係數 $\delta_{ns}$**：ACI 無側移構架放大彎矩法，最終設計彎矩 $M_c = \delta_{ns} M_2$
4. **EI 計算**：題目直接給定 $EI = 0.25E_cI_g$，考驗 $E_c$ 計算

**出題者意圖：**
- 測驗細長柱判斷流程是否完整（含臨界值計算，不可直接跳到放大係數）
- 確認雙曲度下 $M_1/M_2$ 符號的正確取法（負值）與 $C_m$ 下限的應用
- 驗證 $P_c$ 計算的精確性

## 3. 解題戰略地圖與陷阱分析（Strategic Roadmap & Trap Analysis）

### 作戰計畫

```
Step 1：計算迴轉半徑 r，求細長比 klu/r
Step 2：計算臨界細長比（雙曲度），判斷是否為細長柱
Step 3：計算 Ec 與 Ig，求 EI
Step 4：計算歐拉臨界載重 Pc
Step 5：計算 Cm（注意雙曲度符號）
Step 6：計算放大係數 δns
Step 7：計算設計彎矩 Mc = δns × M2
```

### 關鍵陷阱

| # | 陷阱 | 說明 | 應對策略 |
|---|------|------|---------|
| ① | 雙曲度 $M_1/M_2$ 取負值 | 雙曲度（S形）代表兩端彎矩方向相反，ACI 規定此時 $M_1/M_2 < 0$ | 記憶：單曲度（弓形）→ 正值（危險）；雙曲度（S形）→ 負值（有利） |
| ② | $C_m$ 下限 0.4 | 計算出 $C_m = 0.25 < 0.4$，必須取 $C_m = 0.4$ | 每次計算 $C_m$ 後都要與 0.4 比較取大值 |
| ③ | 細長比臨界值也含 $M_1/M_2$ | 臨界值 $= 34-12\times(-0.875)=44.5$，而非固定值 34 | 代入含負號的 $M_1/M_2$ 計算，不可省略 |
| ④ | $r$ 的計算：用 $0.3h$ 近似 | 方形斷面 $r = h/\sqrt{12} \approx 0.289h$，工程上取 $r = 0.3h$ | 考場可直接用 $r = 0.3h$，精算用 $h/\sqrt{12}$ |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`計算放大係數 δns → 求設計彎矩 Mc = δns × M2（設計軸力 Pu 不變）`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } r = 0.3h \quad \Rightarrow \quad \frac{kl_u}{r} = \frac{0.9 \times 800}{\boxed{r}}$$

$$\text{Step 2: } 34 - 12\!\left(\frac{M_1}{M_2}\right) \quad \left(\frac{M_1}{M_2} < 0\ \text{雙曲度}\right)$$

$$\text{Step 3: } E_c = 15000\sqrt{f'_c},\quad I_g = \frac{b h^3}{12},\quad EI = 0.25\,E_c I_g$$

$$\text{Step 4: } P_c = \frac{\pi^2 \boxed{EI}}{(kl_u)^2}$$

$$\text{Step 5: } C_m = 0.6 + 0.4\!\left(\frac{M_1}{M_2}\right) \geq 0.4$$

$$\text{Step 6: } \delta_{ns} = \frac{\boxed{C_m}}{1 - P_u/(0.75\boxed{P_c})} \geq 1.0$$

$$\text{Step 7: } M_c = \boxed{\delta_{ns}} \times M_2$$

| $M_2$ | 40 tf-m | 較大端彎矩（控制端） |
| $M_1$ | 35 tf-m | 較小端彎矩（注意：雙曲度需取負號） |
| $f'_c$ | 280 kgf/cm² | 混凝土強度 |
| $f_y$ | 4200 kgf/cm² | 鋼筋強度 |
| EI | $0.25\,E_c I_g$ | 有效彎曲剛度（題目給定） |

### L2：需知識點推導

**Step 1：細長比計算**

| 符號 | 公式/來源 | 卡關? |
|------|----------|:-----:|

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|:-----:|
| 雙曲度 vs 單曲度的物理意義 | 雙曲度：兩端彎矩相互對抗，降低中央撓曲（有利）；$C_m < 1$ 反映此有利效應 | |
| $C_m = 0.4$ 下限的意義 | 即使雙曲度很強，最小放大因子仍不得小於 0.4，確保最低安全係數 | |

## 4. 步驟化詳細計算過程（Step-by-Step Detailed Calculation）

### Step 1：細長比計算

**迴轉半徑（方形斷面，取近似值）：**

$$r = 0.3\,h = 0.3 \times 40 = 12 \text{ cm}$$

> 精確值：$r = h/\sqrt{12} = 40/3.464 = 11.547$ cm，工程實務常取 $0.3h = 12$ cm。

**細長比：**

$$\frac{kl_u}{r} = \frac{0.9 \times 800}{12} = \frac{720}{12} = \boxed{60}$$

---

### Step 2：細長柱臨界值判斷

**雙曲度下 $M_1/M_2$ 取負值（ACI/CNS 符號慣例）：**

$$\frac{M_1}{M_2} = -\frac{35}{40} = -0.875 \quad \text{（雙曲度，S 形，取負號）}$$

**臨界細長比：**

$$34 - 12\!\left(\frac{M_1}{M_2}\right) = 34 - 12 \times (-0.875) = 34 + 10.5 = \boxed{44.5}$$

**判斷：**

$$\frac{kl_u}{r} = 60 > 44.5 \quad \Rightarrow \text{為細長柱，需進行放大彎矩計算}$$

> 📌 **策略註解：** 另外確認 $kl_u/r = 60 \leq 100$（ACI 規定超過100不得用本法），本題符合。

---

### Step 3：計算 $E_c$、$I_g$、EI

**混凝土彈性模數（依考卷參考公式）：**

$$E_c = 15000\sqrt{f'_c} = 15000 \times \sqrt{280} = 15000 \times 16.733 = 250{,}995 \text{ kgf/cm}^2$$

**毛斷面慣性矩（方形斷面）：**

$$I_g = \frac{bh^3}{12} = \frac{40 \times 40^3}{12} = \frac{40 \times 64{,}000}{12} = \frac{2{,}560{,}000}{12} = 213{,}333 \text{ cm}^4$$

**有效彎曲剛度（依題目給定）：**

$$EI = 0.25\,E_c I_g = 0.25 \times 250{,}995 \times 213{,}333$$

$$= 0.25 \times 5.354 \times 10^{10} = \boxed{1.339 \times 10^{10} \text{ kgf·cm}^2}$$

> 📌 中間計算：$250{,}995 \times 213{,}333 = 5.354 \times 10^{10}$

---

### Step 4：計算歐拉臨界載重 $P_c$

**有效長度：**

$$kl_u = 0.9 \times 800 = 720 \text{ cm}$$

**臨界載重：**

$$P_c = \frac{\pi^2 EI}{(kl_u)^2} = \frac{9.8696 \times 1.339 \times 10^{10}}{720^2} = \frac{9.8696 \times 1.339 \times 10^{10}}{518{,}400}$$

分子：

$$\pi^2 \times EI = 9.8696 \times 1.339 \times 10^{10} = 1.321 \times 10^{11} \text{ kgf·cm}^2$$

$$P_c = \frac{1.321 \times 10^{11}}{518{,}400} = 254{,}900 \text{ kgf} = \boxed{254.9 \text{ tf}}$$

---

### Step 5：計算 $C_m$ 係數

$$C_m = 0.6 + 0.4 \times \frac{M_1}{M_2} = 0.6 + 0.4 \times (-0.875) = 0.6 - 0.35 = 0.25$$

**但 $C_m \geq 0.4$（規範下限）：**

$$C_m = \max(0.25,\ 0.4) = \boxed{0.4}$$

> 📌 **策略註解：** 雙曲度時 $C_m$ 計算值往往低於 0.4 下限，本題即為典型案例。$C_m = 0.4$ 代表即使變形最有利，仍需保留 40% 的放大效應作為安全餘裕。

---

### Step 6：計算放大係數 $\delta_{ns}$

$$0.75\,P_c = 0.75 \times 254.9 = 191.2 \text{ tf}$$

$$\frac{P_u}{0.75\,P_c} = \frac{130}{191.2} = 0.6800$$

$$\delta_{ns} = \frac{C_m}{1 - P_u/(0.75\,P_c)} = \frac{0.4}{1 - 0.6800} = \frac{0.4}{0.3200} = \boxed{1.250}$$

**檢核：**

$$\delta_{ns} = 1.250 \geq 1.0 \quad \checkmark$$

---

### Step 7：計算設計彎矩

$$M_c = \delta_{ns} \times M_2 = 1.250 \times 40 = \boxed{50.0 \text{ tf-m}}$$

---

### 最終答案

$$\boxed{
\begin{aligned}
&\text{設計軸力：} P_u = 130 \text{ tf（不變）}\\
&\text{設計彎矩：} M_c = \delta_{ns} \times M_2 = 1.250 \times 40 = 50.0 \text{ tf-m}
\end{aligned}
}$$

| 項目 | 計算值 |
|------|--------|
| 細長比 $kl_u/r$ | 60 |
| 臨界細長比 | 44.5 |
| 結論 | 細長柱（60 > 44.5） |
| $E_c$ | 250,995 kgf/cm² |
| $I_g$ | 213,333 cm⁴ |
| $EI$ | $1.339 \times 10^{10}$ kgf-cm² |
| $P_c$ | 254.9 tf |
| $C_m$ | 0.4（下限控制） |
| $\delta_{ns}$ | 1.250 |
| **設計彎矩 $M_c$** | **50.0 tf-m** |
| **設計軸力 $P_u$** | **130 tf（不變）** |

## 5. 關鍵爭議點與進階探討（Critical Issues & Advanced Discussion）

### 爭議點一：$r$ 的精確計算 vs 近似值

- **近似值**（考場常用）：$r = 0.3h = 12$ cm → $kl_u/r = 60$
- **精確值**：$r = h/\sqrt{12} = 40/3.464 = 11.547$ cm → $kl_u/r = 720/11.547 = 62.3$

以精確值計算：
$$C_m/(1 - P_u/0.75P_c)$$
$P_c$ 與 $r$ 無關（EI 直接給定），故 $P_c$ 及 $\delta_{ns}$ 結論相同，僅臨界值判斷略有差異（62.3 > 44.5，結論不變）。考場使用 $r = 0.3h$ 即可。

### 爭議點二：雙曲度的符號使用

本題考卷明確說明「雙曲度」，但部分考生可能習慣以絕對值處理 $M_1/M_2$：

| 做法 | $M_1/M_2$ | $C_m$ | 說明 |
|------|-----------|-------|------|
| **正確（ACI 慣例）** | $-0.875$ | $0.4$（下限） | 雙曲度取負 |
| 錯誤（忽略符號） | $+0.875$ | $0.95$ | 將高估放大係數 |

若錯誤取 $+0.875$：$C_m = 0.6 + 0.4 \times 0.875 = 0.95$
$\delta_{ns} = 0.95/0.3200 = 2.97$ → $M_c = 118.8$ tf-m（嚴重高估）

### 爭議點三：$E_c$ 公式版本

- 考卷提供：$E_c = 15000\sqrt{f'_c}$（kgf/cm²，舊版 ACI，適用常重混凝土）
- 現代 SI：$E_c = 4700\sqrt{f'_c}$（MPa）

兩者等價（$15000 \sqrt{\text{kgf/cm}^2} \approx 4730 \sqrt{\text{MPa}}$，差異來自單位換算係數）。考試直接用題目所給公式。

### 進階觀念：$P_u$ 接近 $0.75P_c$ 的危險

本題 $P_u/0.75P_c = 0.680$，表示柱承受了理論穩定載重的 68%。若繼續增大軸力：
- 當 $P_u = 0.75P_c = 191.2$ tf → $\delta_{ns} \to \infty$（柱失穩）
- 規範因此限制 $\delta_{ns}$ 分母不得趨近零，實際設計須確保 $P_u < 0.75P_c$

本題 $P_u = 130$ tf < $0.75P_c = 191.2$ tf，安全餘裕合理。
