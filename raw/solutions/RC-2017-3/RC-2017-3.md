## 1. 原始題目重述 (Problem Restatement)

**規範依據：** 土木 401-100（$f_y = 4{,}200\text{ kgf/cm}^2$ 沿用全卷）

一方形獨立基腳，尺寸 $4.0\text{ m} \times 4.0\text{ m}$，中央承接方形柱 $45\text{ cm} \times 45\text{ cm}$（柱混凝土 $f'_c = 350\text{ kgf/cm}^2$）。

**柱載重（服務載重）：**
- 靜載重：$P_D = 180\text{ tf}$
- 活載重：$P_L = 120\text{ tf}$

**基腳資料：**
- 厚度：$h = 80\text{ cm}$
- 混凝土：$f'_c = 210\text{ kgf/cm}^2$
- 鋼筋：$f_y = 4{,}200\text{ kgf/cm}^2$

**土壤條件：**
- 容許承載力：$q_a = 3\text{ kgf/cm}^2 = 30\text{ tf/m}^2$
- 覆土深度：$1.0\text{ m}$（基腳頂面以上）
- 土壤重度：$\gamma_s = 1{,}900\text{ kg/m}^3 = 1.9\text{ tf/m}^3$
- RC 單位重：$\gamma_{RC} = 2.4\text{ tf/m}^3$

**試求：** 底版所需鋼筋量，並檢核剪力強度是否足夠。（25分）

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 方形基腳的設計哲學是「倒置懸臂梁」——柱傳來的集中力透過基腳底板向四周懸挑，底部向上的土壓力為均佈載重，臨界設計截面在柱面（彎矩）與距柱面 $d/2$（穿孔剪力）、距柱面 $d$（單向剪力）處。

**出題者測試重點：**
1. 能否正確區分「服務載重驗地基」vs「因數載重驗混凝土」
2. 穿孔剪力臨界周長 $b_o$ 的計算與三公式取最小
3. 設計有效深度 $d$ 的合理假設

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 假設有效深度 $d$（根據 $h=80\text{cm}$、保護層、鋼筋尺寸）
2. 地基驗核（服務載重）
3. 因數載重 $P_u$、淨因數土壓力 $q_u$
4. 彎矩設計（臨界截面：柱面）→ $A_s$
5. 單向剪力驗核（臨界截面：距柱面 $d$）
6. 雙向穿孔剪力驗核（臨界截面：距柱面 $d/2$）

**關鍵陷阱：**

| # | 陷阱 | 正確做法 |
|---|------|---------| 
| 1 | **服務/因數載重混用**：用因數載重驗地基、用服務載重設計混凝土 | 地基用 $P_D+P_L=300\text{ tf}$；混凝土用 $P_u=408\text{ tf}$ |
| 2 | **穿孔剪力臨界面位置**：放在柱面 | 距柱面 $d/2$ 處 → $b_o = 4(c+d)$ |
| 3 | **彎矩臨界截面位置**：放在柱中心 | 在柱**面**（不是柱中心） |
| 4 | **$q_u$ 計算方式**：含基腳自重因數化 | 保守簡化法：$q_u = P_u/B^2$（扣除對應之因數化覆土後近似） |
| 5 | **$\phi$ 不分**：彎矩與剪力用同一個 $\phi$ | 彎矩 $\phi=0.9$（拉力控制），剪力 $\phi=0.75$ |

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } d = h - \text{cover}_{\text{footing}} - r_{\text{bar}} \approx 72\text{ cm}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 【前置】材料參數

$$f'_c = 210\text{ kgf/cm}^2,\quad f_y = 4{,}200\text{ kgf/cm}^2,\quad \beta_1 = 0.85 \quad (f'_c \leq 280)$$

$$\phi_{\text{flex}} = 0.9 \text{（拉力控制，待驗）},\quad \phi_{\text{shear}} = 0.75$$

---

### 【Step 1】有效深度假設

基腳與土壤接觸面之保護層（ACI §20.6.1.3）：$\text{cover}_{\min} = 7.5\text{ cm}$。

採用 D25 鋼筋（$d_b = 2.54\text{ cm}$）：

$$d = h - \text{cover} - \frac{d_b}{2} = 80 - 7.5 - 1.27 = \boxed{71.23\text{ cm} \approx 72\text{ cm}}$$

> 📝 **策略註解：** 考場若題目未明確給 $d$，明確說明假設並記錄即可；$d=72\text{ cm}$ 為合理估值。

---

### 【Step 2】地基承載力驗核（服務載重）

$$P_{\text{service}} = P_D + P_L = 180 + 120 = 300\text{ tf}$$

$$W_f = 4.0\times4.0\times0.80\times2.4 = 30.72\text{ tf}$$

$$W_s = 4.0\times4.0\times1.00\times1.9 = 30.40\text{ tf}$$

$$q_{\text{net}} = \frac{300 + 30.72 + 30.40}{4.0\times4.0} = \frac{361.12}{16} = \boxed{22.6\text{ tf/m}^2} < q_a = 30\text{ tf/m}^2 \quad \checkmark$$

---

### 【Step 3】因數載重與淨因數土壓力

$$P_u = 1.2P_D + 1.6P_L = 1.2\times180 + 1.6\times120 = 216 + 192 = \boxed{408\text{ tf}}$$

**淨因數化上壓力（簡化法）：**
$$q_u = \frac{P_u}{B^2} = \frac{408}{4.0\times4.0} = \boxed{25.5\text{ tf/m}^2 = 2.55\text{ kgf/cm}^2}$$

> 📝 **策略註解：** 基腳自重（$1.2\times30.72$）與對應面積上的因數化土壓力（$1.2\times30.72$）相互抵銷，採 $q_u = P_u/A$ 為工程常用保守簡化法。

---

### 【Step 4】彎矩設計（臨界截面：柱面）

**懸臂長度：**
$$L = \frac{B - c}{2} = \frac{400 - 45}{2} = 177.5\text{ cm} = 1.775\text{ m}$$

**因數化彎矩：**
$$M_u = q_u \times B \times \frac{L^2}{2} = 2.55 \times 400 \times \frac{177.5^2}{2} = 2.55 \times 400 \times 15{,}750.6 = \boxed{16{,}065{,}600\text{ kgf·cm} \approx 160.7\text{ tf·m}}$$

**迭代求 Whitney 應力塊深度 $a$：**

*第一次估算（$a_0 = 0.1d = 7.2\text{ cm}$）：*
$$A_{s,0} = \frac{M_u}{\phi f_y (d - a_0/2)} = \frac{16{,}065{,}600}{0.9\times4{,}200\times(72-3.6)} = \frac{16{,}065{,}600}{258{,}552} = 62.1\text{ cm}^2$$

*修正 $a$：*
$$a_1 = \frac{A_{s,0} f_y}{0.85 f'_c B} = \frac{62.1\times4{,}200}{0.85\times210\times400} = \frac{260{,}820}{71{,}400} = 3.66\text{ cm}$$

*第二次迭代：*
$$A_{s,1} = \frac{16{,}065{,}600}{0.9\times4{,}200\times(72-1.83)} = \frac{16{,}065{,}600}{265{,}258} = 60.6\text{ cm}^2$$

*驗算 $a$：*
$$a_2 = \frac{60.6\times4{,}200}{71{,}400} = 3.566\text{ cm} \approx a_1 \quad \text{（收斂）}$$

**所需鋼筋量：**
$$\boxed{A_s = 60.6\text{ cm}^2 \text{（各方向）}}$$

**延性確認（$\phi = 0.9$ 驗核）：**
$$c = \frac{a}{\beta_1} = \frac{3.566}{0.85} = 4.19\text{ cm}$$

$$\varepsilon_t = 0.003\times\frac{d - c}{c} = 0.003\times\frac{72 - 4.19}{4.19} = 0.003\times16.18 = 0.0485 \geq 0.005 \quad \Rightarrow \quad \phi = 0.9 \quad \checkmark$$

**最小鋼筋量確認：**
$$\rho_{\min} = \max\!\left(\frac{0.25\sqrt{210}}{4{,}200},\,\frac{1.4}{4{,}200}\right) = \max(0.000863,\,0.000333) = 0.000863$$

$$A_{s,\min} = 0.000863\times400\times72 = 24.9\text{ cm}^2 \ll 60.6\text{ cm}^2 \quad \checkmark$$

**鋼筋選配（各方向）：**

選用 D25（$A_b = 5.067\text{ cm}^2$）：
$$n = \left\lceil\frac{60.6}{5.067}\right\rceil = \lceil 11.96 \rceil = 12\text{ 支}$$

$$s = \frac{400 - 2\times7.5}{12 - 1} = \frac{385}{11} = 35\text{ cm}$$

$$A_{s,\text{actual}} = 12\times5.067 = \boxed{60.8\text{ cm}^2 > 60.6\text{ cm}^2 \quad \checkmark}$$

> **∴ 底版配筋：各方向配置 $\boxed{12\text{-D25} @ 35\text{ cm}}$，$A_s = 60.8\text{ cm}^2$**

---

### 【Step 5】單向剪力強度驗核（臨界截面：距柱面 $d$）

距基腳邊緣距離：
$$L_{\text{crit,1}} = \frac{B-c}{2} - d = 177.5 - 72 = 105.5\text{ cm}$$

**因數化單向剪力：**
$$V_{u,1} = q_u \times B \times L_{\text{crit,1}} = 2.55\times400\times105.5 = 107{,}610\text{ kgf} = \boxed{107.6\text{ tf}}$$

**混凝土剪力強度（考卷公式 $V_c = 0.53\sqrt{f'_c}\,b_w d$）：**
$$V_c = 0.53\times\sqrt{210}\times400\times72 = 0.53\times14.49\times28{,}800 = 221{,}355\text{ kgf} = 221.4\text{ tf}$$

$$\phi V_c = 0.75\times221.4 = \boxed{166.0\text{ tf}} > V_{u,1} = 107.6\text{ tf} \quad \checkmark$$

> ✅ **單向剪力通過（裕度 54%）**

---

### 【Step 6】穿孔剪力（雙向衝剪）強度驗核（臨界截面：距柱面 $d/2$）

**臨界周長（方柱 $45\text{ cm}\times45\text{ cm}$）：**
$$b_o = 4(c + d) = 4\times(45 + 72) = 4\times117 = \boxed{468\text{ cm}}$$

**臨界周線內面積：**
$$(c+d)^2 = 117^2 = 13{,}689\text{ cm}^2 = 1.369\text{ m}^2$$

**因數化穿孔剪力：**
$$V_{u,2} = q_u\times\bigl[B^2 - (c+d)^2\bigr] = 2.55\times(400^2 - 117^2) = 2.55\times(160{,}000-13{,}689) = 2.55\times146{,}311 = \boxed{373{,}093\text{ kgf} \approx 373.1\text{ tf}}$$

**穿孔剪力強度（三公式取最小，$\beta_c = 1$，$\alpha_s = 40$ 內柱）：**

換算至 SI：$f'_c = 210/10.197 = 20.59\text{ MPa}$，$\sqrt{f'_c} = 4.539\text{ MPa}^{0.5}$

| 公式 | 計算（kgf，cm 換算結果） | $V_c$ (tf) |
|------|------------------------|:----------:|
| ① $0.53(1+2/\beta_c)\sqrt{f'_c}\cdot b_o\cdot d$ | $0.53\times3\times14.49\times468\times72$ | **776.3** |
| ② $0.053(\alpha_s d/b_o+2)\sqrt{f'_c}\cdot b_o\cdot d$，$\alpha_s=40$ | $0.053\times(40\times72/468+2)\times14.49\times33696$ | **647.2** |
| ③ $\mathbf{1.06\sqrt{f'_c}\cdot b_o\cdot d}$ | $1.06\times14.49\times468\times72$ | **518.0 ← 控制** |

$$\phi V_c = 0.75\times518.0 = \boxed{388.5\text{ tf}} > V_{u,2} = 373.1\text{ tf} \quad \checkmark$$

$$\text{裕度比 } = \frac{388.5}{373.1} = 1.041 \text{（4.1\% 裕度，通過但屬邊界值）}$$

> 📝 **策略註解：** 穿孔剪力僅 4% 裕度，若需保守設計，可：①增加 $d$（最有效）；②縮小基腳尺寸使 $q_u$ 提高、$b_o$ 縮小（適得其反）；③加設剪力柱（Stud Rail）或剪力帽（Shear Cap）。

> ✅ **穿孔剪力通過（裕度 4%，邊界合格）**

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**爭議1：穿孔剪力公式選擇（3個公式的適用條件）**
- **公式①**：控制非方形柱（$\beta_c > 2$）時
- **公式②**：控制臨界周長非常大時（$b_o \gg \alpha_s d$）
- **公式③**：方柱（$\beta_c = 1$）時最常控制，也是最保守的下限
- 本題 $\beta_c = 1$，公式③控制 ✓

**爭議2：有效深度 $d$ 的假設**
- 本題假設 $d = 72\text{ cm}$（保護層 7.5cm + D25 半徑 1.27cm ≈ 8.77cm → 取 8cm）
- 若題目允許較大保護層（如 8cm 清保護層），$d = 80 - 8 - 1.27 = 70.7\text{ cm}$，穿孔剪力裕度更小
- 考場須明確說明假設

**爭議3：柱-基腳承壓（Bearing）是否需要驗核？**
- 題目未要求，但可快速確認：
  - 基腳頂面承壓強度 $\phi V_n = 0.65\times0.85\times210\times(45\times45)\times2 = 469\text{ tf} > P_u = 408\text{ tf}$ ✓
  - 柱底採高強混凝土（$f'_c = 350\text{ kgf/cm}^2$），遠大於基腳，不控制

**進階思考：若穿孔剪力不足的設計選擇**
- 方案①：增加 $d$（最直接，每增 5cm，$\phi V_c$ 提升約 7%）
- 方案②：擴大基腳尺寸（降低 $q_u$，同時增大 $b_o$，兩效果綜合）
- 方案③：增設剪力帽（Shear Capital）或鋼板剪力頭（Shearhead）
