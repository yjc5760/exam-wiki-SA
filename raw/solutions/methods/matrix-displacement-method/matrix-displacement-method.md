# 方法論：矩陣位移法／直接勁度法（Matrix Displacement Method / Direct Stiffness Method）

**method_id：** `matrix-displacement-method`
**適用單元：** SA-U2-4 靜不定結構之矩陣分析法
**適用題型：** 剛架、桁架、梁－柱系統之系統化分析（含半剛性接頭、彈簧支承、格柵/樓板結構）
**核心公式：**
- 元素局部勁度矩陣（梁元素，忽略軸向）：$k_{11}=12EI/L^3$、$k_{22}=4EI/L$、$k_{12}=6EI/L^2$
- 座標轉換：$[k]_{global} = [T]^T[k]_{local}[T]$
- 整體組裝：$[K] = \Sigma [T]^T[k][T]$（依自由度編號疊加各元素貢獻）
- 求解：$\{\Delta\} = [K]^{-1}\{P\}$；元素內力：$\{f\} = [k]\{d\} - \{f_F\}$（含固端力修正）

## 步驟摘要

1. 建立自由度編號（節點位移：平移＋轉角），標示束制自由度（支承處）
2. 對每個元素建立局部勁度矩陣，若座標不與整體座標軸重合需做旋轉轉換
3. 依自由度編號將各元素勁度矩陣疊加組裝成整體勁度矩陣 $[K]$
4. 若元素有非節點載重（跨間均布載重等），需計算固端力向量 $\{f_F\}$ 並轉為等效節點力
5. 求解 $\{Δ\} = [K]^{-1}\{P\}$ 得節點位移，回代各元素求桿端內力（含固端力修正項）
6. 特殊自由度處理：靜態凝縮（消去無載重自由度）、剛性桿件（軸向無限大簡化）、彈簧/半剛性接頭（以額外元素或修正勁度模擬）

## 常見陷阱

- 自由度編號與元素轉換矩陣方向不一致，導致組裝時正負號錯誤
- 忽略非節點載重的固端力修正，直接用節點力向量求解會漏掉跨間載重效應
- 半剛性接頭/旋轉彈簧未正確視為獨立元素（或修正勁度）疊加
- 忽略軸向變形的假設未在所有相關元素一致套用，造成自由度數目誤判

## 適用範圍與限制

系統化程度最高、最適合電腦化與自由度數目多的複雜結構（如格床、多層剪力牆-框架系統），但手算時矩陣組裝與求逆計算量大；自由度少（1–2 個）且結構規則時，傾角變位法（`slope-deflection-method.md`）或彎矩分配法（`moment-distribution-method.md`）手算更快。
