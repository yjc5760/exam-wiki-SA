# 考題編號：RC-2014-3

**主分類：** `RC-U2-1` RC 剪力強度分析與設計
**副分類：** （無）
**設計法：** USD 強度設計法
**標籤：** `臨界斷面` `Vc最大混凝土剪力強度` `軸力影響` `等效彎矩Mm` `詳細公式` `軸壓梁` `軸拉梁` `剪力三種條件比較`

---

## 1. 原始題目重述 (Problem Restatement)

有一**簡支矩形梁**，跨距 $L = 5\ \text{m}$，斷面尺寸：

$$b = 30\ \text{cm},\quad d = 60\ \text{cm},\quad h = 70\ \text{cm}$$

材料與載重：
- $f'_c = 280\ \text{kgf/cm}^2$
- 均布設計載重 $w_u = 5\ \text{tf/m}$（不計梁自重）
- 受拉鋼筋：$A_s = 3\text{-D32} = 24.42\ \text{cm}^2$

在下列三種軸力條件下，求**臨界斷面**（距支承面距離 $d$ 處）之**最大混凝土剪力計算強度 $V_c$**：（25 分）

| 案例 | 軸力條件 |
|------|---------|
| Case 1 | 無軸力（$N_u = 0$） |
| Case 2 | 設計軸壓力 $N_u = 25\ \text{tf}$（壓力） |
| Case 3 | 設計軸拉力 $N_u = 25\ \text{tf}$（拉力） |

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心精神：**
本題考查軸力對混凝土剪力強度 $V_c$ 的影響，使用**等效彎矩法**（$M_m$ 修正）——以 $M_m = M_u - N_u(4h-d)/8$ 代入詳細 $V_c$ 公式，統一處理三種軸力情況。

**關鍵物理直覺：**
- **軸壓**使斷面受壓增加、抑制裂縫，$V_c$ 提升
- **軸拉**使斷面更易開裂，$V_c$ 降低
- $M_m$ 公式通過等效彎矩放大或縮小 $V_u \cdot d / M_m$ 比值，間接反映此效果

**出題者意圖：**
- 測試詳細 $V_c$ 公式（非簡化）的正確應用
- 考查 $M_m$ 公式的物理意義與 $N_u$ 符號規定
- 觀察 $M_m < 0$ 時的邊界條件處理（壓力大到超過彎矩）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

```
共用前置（全三案例相同）：
  → 計算 ρw、√f'c、臨界斷面 Vu、Mu

Case 1（無軸力）：Mm = Mu
  → 詳細公式：Vc = (0.50√f'c + 175ρw × Vu·d/Mu) × bw·d ≤ 0.93√f'c × bw·d

Case 2（軸壓）：計算 Mm = Mu - Nu(4h-d)/8
  → 若 Mm < 0 → 公式失效 → Vc = 0.93√f'c × bw·d（最大值）

Case 3（軸拉）：計算 Mm = Mu - Nu(4h-d)/8，Nu < 0
  → Mm > Mu（拉力使等效彎矩增大）
  → 詳細公式代入，Vc 低於無軸力
```

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | **誤用簡化公式**（$V_c = 0.53\sqrt{f'_c}\,b_w d$）：只能用在「無軸力」簡算，本題要求「最大」$V_c$ | 題目要求最大剪力強度 → 用詳細公式（通常更高） |
| 2 | **$N_u$ 符號搞混**：壓力為正或負取決於規範版本 | 本題規定（土木401-100）：**壓力 $N_u > 0$，拉力 $N_u < 0$** |
| 3 | **$M_m < 0$ 時繼續套公式**：得出 $V_c < 0$ 或極大值 | $M_m < 0$ → 公式不適用 → 取上限 $V_c = 0.93\sqrt{f'_c}\,b_w d$ |
| 4 | **忘記上限值驗算**：詳細公式結果須檢查 $\leq 0.93\sqrt{f'_c}\,b_w d$ | 每次計算 $V_c$ 後必須與上限比較 |
| 5 | **臨界斷面位置錯誤**：取支承中心 $d$ 而非支承面 $d$ | 臨界斷面在**支承面**（支承邊緣）起算 $d$ 距離 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`三種軸力條件下，臨界斷面之最大混凝土剪力計算強度 Vc`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1:}\quad \rho_w = A_s/(b_w\,d),\quad V_u = w_u(L/2 - d),\quad M_u = w_u\,d\,(L-d)/2$$

$$\text{Step 2:}\quad M_m = M_u - N_u\,(4h-d)/8 \quad \text{（軸壓} N_u>0\text{；軸拉} N_u<0\text{）}$$

$$\text{Step 3（}M_m>0\text{）:}\quad V_c = \!\left(0.50\sqrt{f'_c} + 175\rho_w\,\frac{\boxed{V_u}\,d}{\boxed{M_m}}\right)\!b_w\,d \leq 0.93\sqrt{f'_c}\,b_w\,d$$

$$\text{Step 3（}M_m\leq0\text{）:}\quad V_c = 0.93\sqrt{f'_c}\,b_w\,d \quad \text{（直接取上限）}$$

| $d$ | 60 cm | 有效深度 |

| $N_{u,2}$ | +25,000 kgf | Case 2 軸壓力（正） |
| $N_{u,3}$ | −25,000 kgf | Case 3 軸拉力（負） |

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|:-----:|
| **$M_m$ 的物理意義** | 軸壓力會在梁腹產生附加彎矩，使截面的「等效淨彎矩」減小；$M_m < 0$ 表示軸壓引起的彎矩已超過外力彎矩，截面實際上受的是「壓彎加強」→ Vc 達上限 | |
| **為何軸壓提升 Vc** | 壓力使截面中性軸下移、裂縫傾向被壓閉，抗剪能力增強；過大壓力（$M_m < 0$）→ Vc 達規範上限 0.93√f'c·bw·d | |
| **為何軸拉降低 Vc** | 拉力促進開裂，斷面有效壓力區縮小，Vc 降低；極端拉力（Nu/(3.5Ag) < −1）→ Vc → 0 | |
| **詳細公式比簡化公式高的原因** | 詳細公式中 $175\rho_w V_u d/M_m$ 項在 $V_u d/M_m$ 不為零時均為正，增加 Vc；簡化公式相當於忽略此項 | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 前置計算（三案例共用）

**斷面參數：**

$$\rho_w = \frac{A_s}{b_w\,d} = \frac{24.42}{30\times60} = \frac{24.42}{1800} = \boxed{0.01357}$$

$$\sqrt{f'_c} = \sqrt{280} = \boxed{16.733}\ \text{kgf}^{1/2}/\text{cm}$$

$$A_g = b\times h = 30\times70 = 2100\ \text{cm}^2$$

**臨界斷面（距支承面 $d = 60$ cm 處）之 $V_u$、$M_u$：**

$$\text{支承反力：} R = \frac{w_u\,L}{2} = \frac{50\times500}{2} = 12{,}500\ \text{kgf}$$

$$V_u = R - w_u\,d = 12{,}500 - 50\times60 = 12{,}500 - 3{,}000 = \boxed{9{,}500\ \text{kgf}}$$

$$M_u = R\cdot d - \frac{w_u\,d^2}{2} = 12{,}500\times60 - \frac{50\times3600}{2} = 750{,}000 - 90{,}000 = \boxed{660{,}000\ \text{kgf·cm}}$$

$$\frac{V_u\,d}{M_u} = \frac{9{,}500\times60}{660{,}000} = \frac{570{,}000}{660{,}000} = \boxed{0.8636} \leq 1.0\ ✓$$

**等效彎矩參數：**

$$\frac{4h-d}{8} = \frac{4\times70 - 60}{8} = \frac{280-60}{8} = \frac{220}{8} = \boxed{27.5\ \text{cm}}$$

**上限 $V_{c,\text{max}}$：**

$$V_{c,\max} = 0.93\sqrt{f'_c}\,b_w\,d = 0.93\times16.733\times1800 = 15.561\times1800 = \boxed{28{,}010\ \text{kgf} = 28.01\ \text{tf}}$$

**參考：簡化公式 $V_c$（無軸力）**

$$V_{c,\text{simple}} = 0.53\sqrt{f'_c}\,b_w\,d = 0.53\times16.733\times1800 = 8.869\times1800 = 15{,}964\ \text{kgf} = 15.96\ \text{tf}$$

---

### Case 1：無軸力（$N_u = 0$）

$M_m = M_u = 660{,}000\ \text{kgf·cm}$（軸力為零，$M_m$ 即 $M_u$）

詳細公式：

$$V_c = \!\left(0.50\sqrt{f'_c} + 175\,\rho_w\,\frac{V_u\,d}{M_u}\right)\!b_w\,d$$

$$= \!\left(0.50\times16.733 + 175\times0.01357\times0.8636\right)\!\times1800$$

$$= \!\left(8.366 + 2.375\times0.8636\right)\!\times1800$$

$$= \!\left(8.366 + 2.051\right)\!\times1800 = 10.417\times1800$$

$$= \boxed{V_c = 18{,}751\ \text{kgf} = 18.75\ \text{tf}}$$

上限驗算：$18{,}751 < 28{,}010\ ✓$（詳細公式值有效）

---

### Case 2：設計軸壓力 $N_u = +25\ \text{tf}$（壓力，$N_u > 0$）

等效彎矩：

$$M_m = M_u - N_u\,\frac{4h-d}{8} = 660{,}000 - 25{,}000\times27.5 = 660{,}000 - 687{,}500 = \boxed{-27{,}500\ \text{kgf·cm}}$$

$$M_m = -27{,}500\ \text{kgf·cm} < 0$$

**當 $M_m \leq 0$ 時，詳細公式中 $V_u d/M_m < 0$，公式失效。**
按規範規定，此時 $V_c$ 取規範上限：

$$\boxed{V_c = V_{c,\max} = 0.93\sqrt{f'_c}\,b_w\,d = 28{,}010\ \text{kgf} = 28.01\ \text{tf}}$$

**物理意義**：軸壓力 25 tf 所產生的等效附加彎矩（$687{,}500\ \text{kgf·cm}$）已超過外力彎矩（$660{,}000\ \text{kgf·cm}$），說明臨界斷面實際上處於「壓彎加強」狀態，混凝土剪力強度達到上限。

---

### Case 3：設計軸拉力 $N_u = -25\ \text{tf}$（拉力，$N_u < 0$）

等效彎矩（$N_u$ 取負值代入）：

$$M_m = M_u - N_u\,\frac{4h-d}{8} = 660{,}000 - (-25{,}000)\times27.5 = 660{,}000 + 687{,}500 = \boxed{1{,}347{,}500\ \text{kgf·cm}}$$

$$\frac{V_u\,d}{M_m} = \frac{9{,}500\times60}{1{,}347{,}500} = \frac{570{,}000}{1{,}347{,}500} = \boxed{0.4230} \leq 1.0\ ✓$$

詳細公式：

$$V_c = \!\left(0.50\sqrt{f'_c} + 175\,\rho_w\,\frac{V_u\,d}{M_m}\right)\!b_w\,d$$

$$= \!\left(0.50\times16.733 + 175\times0.01357\times0.4230\right)\!\times1800$$

$$= \!\left(8.366 + 2.375\times0.4230\right)\!\times1800$$

$$= \!\left(8.366 + 1.005\right)\!\times1800 = 9.371\times1800$$

$$= \boxed{V_c = 16{,}868\ \text{kgf} = 16.87\ \text{tf}}$$

上限驗算：$16{,}868 < 28{,}010\ ✓$

---

|------|:-----------:|:--------------:|:-----------:|:-----------:|:----------:|

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 為何軸壓 Case 2 的 $V_c$ 反而達到上限（最大）**

直覺上，軸壓使梁的剪切剛度增加——壓力閉合了潛在的斜裂縫，混凝土整體受壓後的抗剪貢獻更大。但 $M_m$ 公式更深的意涵是：

> 軸壓 $N_u$ 在梁腹部的任意斷面會產生一個「負彎矩效應」（$-N_u(4h-d)/8$ 項為負），**等效上抵銷了外加彎矩**。當此抵銷量超過外加彎矩（$M_m < 0$），即表示軸壓已使斷面接近純壓狀態，$V_c$ 達規範允許上限。

**② 軸拉力的工程意義**

Case 3 中，$V_c = 16.87\ \text{tf}$ 比無軸力（18.75 tf）低約 10%，比簡化公式（15.96 tf）仍略高。軸拉力使斷面開裂趨勢增加，降低混凝土抗剪能力。若繼續增加軸拉力，$V_c$ 將持續降低，最終趨近於零（當 $N_u(3.5A_g) \leq -f'_c$ 時，混凝土完全不提供抗剪貢獻）。

**③ 詳細公式 vs. 簡化公式的取捨**

| 公式 | Case 1 $V_c$ | 適用情境 |
|------|:-----------:|---------|
| 簡化：$0.53\sqrt{f'_c}\,b_w d$ | 15.96 tf | 快速估算、保守設計 |
| 詳細：$0.50\sqrt{f'_c} + 175\rho_w V_u d/M_m$ | 18.75 tf | 最大化 $V_c$ 以減少腹筋用量 |

詳細公式需計算 $V_u \cdot d/M_m$，但可節省約 15% 的腹筋設計需求。考場建議：若題目說「最大混凝土剪力計算強度」→ 用詳細公式；若說「計算 $V_c$」或時間緊迫 → 用簡化公式即可。
