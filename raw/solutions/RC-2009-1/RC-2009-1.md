# 考題編號：RC-2009-1

**主分類：** `RC-U3-3` 韌性要求與耐震設計  
**副分類：** 無  
**設計法：** 概念題（USD 耐震設計）  
**標籤：** `耐震柱` `橫向鋼筋` `密箍區lo` `圍束箍筋` `Ash公式` `Vc=0條件` `特殊矩形框架柱` `剪力設計` `Mpr` `Nu小於0.05Agfc`

---

## 1. 原始題目重述 (Problem Restatement)

試說明 RC 柱耐震設計中，如何配置柱之**橫向鋼筋**？  
假設地震效應之設計軸壓力**小於 $0.05\,A_g f'_c$**（$A_g$ 為柱總斷面積），必要時請繪圖說明。（25 分）

> 本題為概念說明題，無題目附圖，無截圖需求。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 特殊矩形框架柱（SMRF）橫向鋼筋同時承擔三個任務：
1. **圍束（Confinement）**：提升核心混凝土延性
2. **剪力抵抗（Shear）**：在塑性鉸區抵抗地震剪力
3. **縱筋防挫屈（Buckling prevention）**：防止縱向鋼筋在反覆載重下挫屈

**出題重點：** 當 $N_u < 0.05A_gf'_c$ 時，混凝土剪力貢獻 $V_c = 0$，**所有剪力由橫向鋼筋承擔**，此時剪力設計更嚴格。

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 說明耐震柱橫向鋼筋的三大目的
2. 定義密箍區（$l_o$）位置與長度
3. 說明密箍區間距 $s_o$ 規定
4. 推導圍束箍筋面積 $A_{sh}$ 兩公式
5. 說明柱剪力設計（$V_e$）及 $N_u < 0.05A_gf'_c$ 時 $V_c = 0$ 的影響
6. 說明非密箍區的橫向鋼筋要求

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ❶ $N_u < 0.05A_gf'_c$ → $V_c = 0$ | 軸力小時混凝土無剪力貢獻，箍筋須全額承擔 | $V_s = V_e$（無 $V_c$ 分攤） |
| ❷ $A_{sh}$ 兩公式須取大值 | 兩個圍束條件均須滿足 | 均算後取較大者 |
| ❸ $l_o$ 的三個取法取最大 | $l_o = \max(h_n/6, h_{max}, 45cm)$ | 不得省略任何一項 |
| ❹ 密箍區 vs. 非密箍區間距不同 | 非密箍區間距較寬鬆 | 分區說明 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

完整說明耐震柱橫向鋼筋的配置規定，並特別指出 $N_u < 0.05A_gf'_c$ 的影響。

### 關鍵公式（依設計順序）

$$l_o \ge \max\!\left(\frac{h_n}{6},\ h_{\max},\ 45\text{ cm}\right)$$

$$s_o \le \min\!\left(\frac{b_{\min}}{4},\ 6d_b,\ 15\text{ cm}\right)$$

$$A_{sh} \ge \max\!\left(0.09\,s\,h_c\,\frac{f'_c}{f_{yh}},\quad 0.3\,s\,h_c\!\left(\frac{A_g}{A_{ch}}-1\right)\!\frac{f'_c}{f_{yh}}\right)$$

$$V_e = \frac{M_{pr,1}+M_{pr,2}}{h_n}\ (\text{加計重力剪力})$$

$$\text{若 }N_u < 0.05A_gf'_c:\quad V_c = 0,\quad V_s = \frac{V_e}{\phi} = \frac{A_v f_{yh} d}{s}$$

### L1：規範直接規定（不需推導）

| 規定項目 | 內容 |
|---------|------|
| $l_o$ 計算方式 | 三值取大 |
| $s_o$ 計算方式 | 三值取小 |
| $A_{sh}$ 計算方式 | 兩公式取大 |
| $N_u < 0.05A_gf'_c$ 時 $V_c$ | 必須取 0 |

### L3：深層知識

| 知識點 | 說明 | 卡關? |
|--------|------|-------|
| 為何小軸力時 $V_c = 0$ | 軸壓增加 $V_c$；軸力小甚至無法提供圍束效果，保守取 $V_c = 0$ | |
| $A_{sh}$ 第一公式（0.09）來源 | 由 $f'_{cc} = f'_c + 4.1f_L$ 推導：圍束力需補充蓋層脫落的強度損失 | |
| $A_{sh}$ 第二公式（0.3）來源 | 直接由 $P_u \ge P_y$ 條件推導：圍束後核心強度 ≥ 蓋層破壞前全斷面強度 | |
| $M_{pr}$ 的定義 | 可能彎矩強度：$M_{pr} = A_s \times 1.25f_y \times$ 力臂（用 $\alpha=1.25$ 考慮應變硬化） | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

本題為概念說明題，以條列式配合圖說明橫向鋼筋配置。

---

### 一、耐震柱橫向鋼筋的三大目的

| 目的 | 說明 |
|------|------|
| ① 圍束核心混凝土 | 提升混凝土延性，使塑性鉸能承受大變形而不脆斷 |
| ② 抵抗剪力 | 柱端塑性鉸區承受大剪力，需足夠箍筋作為 truss 機構 |
| ③ 防止縱筋挫屈 | 反覆壓縮下縱筋可能挫屈，橫向鋼筋提供側向支撐 |

---

### 二、密箍區（$l_o$）的設置

特殊矩形框架柱在**柱端**必須設置密箍區，以容納塑性鉸形成。

$$\boxed{l_o \ge \max\!\left(\frac{h_n}{6},\quad h_{\max,\text{dim}},\quad 45\text{ cm}\right)}$$

- $h_n$ = 柱淨高
- $h_{\max,\text{dim}}$ = 柱斷面最大邊長
- 45 cm = 絕對最小值

**密箍區位置：** 柱**兩端**（上端及下端），各取 $l_o$；接頭（joint）內全長也需配置（見下圖說明）。

```
┌─────────────────────────────────────────┐
│ 上柱端 ← lo →                           │
│ ┌──────────────────────────────────────┐ │
│ │    密箍區（so間距）                   │ │
│ ├──────────────────────────────────────┤ │
│ │                                      │ │
│ │    非密箍區（較寬間距）               │ │
│ │                                      │ │
│ ├──────────────────────────────────────┤ │
│ │    密箍區（so間距）                   │ │
│ └──────────────────────────────────────┘ │
│              ← lo →  下柱端              │
└─────────────────────────────────────────┘
```

---

### 三、密箍區橫向鋼筋間距（$s_o$）

$$\boxed{s_o \le \min\!\left(\frac{b_{\min}}{4},\quad 6d_b,\quad 15\text{ cm}\right)}$$

- $b_{\min}$ = 柱斷面最小邊長
- $d_b$ = 縱向鋼筋直徑
- 15 cm = 絕對最大間距

---

### 四、密箍區圍束箍筋面積（$A_{sh}$）

在密箍區內，每個方向均需滿足：

$$\boxed{A_{sh} \ge \max\!\left(\underbrace{0.09\,s\,h_c\,\frac{f'_c}{f_{yh}}}_{\text{公式①}},\quad \underbrace{0.3\,s\,h_c\!\left(\frac{A_g}{A_{ch}}-1\right)\!\frac{f'_c}{f_{yh}}}_{\text{公式②}}\right)}$$

- $s$ = 箍筋間距
- $h_c$ = 箍筋中心至中心之核心尺寸（被圍束方向）
- $A_g$ = 全斷面積
- $A_{ch}$ = 核心面積（箍筋外緣至外緣）
- $f_{yh}$ = 箍筋降伏強度

**公式①（0.09）** 來自圍束強度需補充因蓋層剝落造成的強度損失。  
**公式②（0.3）** 直接確保 $P_u$ （圍束後） $\ge P_y$ （蓋層破壞前），取**兩式較大值**。

---

### 五、柱剪力設計及 $N_u < 0.05A_gf'_c$ 的影響

**設計剪力 $V_e$（考量強柱弱梁的實際剪力）：**

$$V_e = \frac{M_{pr,\text{top}} + M_{pr,\text{bot}}}{h_n} + V_{\text{gravity}}$$

- $M_{pr}$ = 可能彎矩強度（以 $1.25f_y$ 計算，考量應變硬化）
- 此 $V_e$ 為最大可能剪力（不折減，考慮超強）

**$N_u < 0.05A_gf'_c$ 時：**

> **規範規定：** 當設計軸壓力小於 $0.05A_gf'_c$ 時，**混凝土剪力貢獻 $V_c = 0$**，剪力全由橫向鋼筋承擔。

物理原因：軸壓力小代表混凝土在反覆地震力下更容易開裂，無法提供穩定的骨料咬合剪力。

$$V_c = 0 \Rightarrow V_e \le \phi V_s = \phi\cdot\frac{A_v\,f_{yh}\,d}{s}$$

所需箍筋間距（由剪力控制）：

$$s \le \frac{A_v\,f_{yh}\,d}{\phi^{-1}V_e}$$

配置時取：$s = \min(s_{\text{剪力}},\ s_o)$（取剪力與圍束條件的更嚴格者）

---

### 六、非密箍區橫向鋼筋

在柱中段（非密箍區），橫向鋼筋間距放寬：

$$s_{\text{non-lo}} \le \min(6d_b,\ 15\text{ cm})$$

仍須提供足夠 $A_{sh}$（可用較大間距但需驗核）。

---

### 七、橫向鋼筋型式

- **閉合箍筋（Closed Hoops）**：圍繞全部縱筋的封閉矩形箍
- **繫筋（Crossties）**：連接對角縱筋，每端 135° 彎鉤，兩端交替配置
- **矩形箍筋組合**：多個矩形箍疊套，確保每個縱筋均受橫向支撐

每根縱向鋼筋在密箍區內間距不得超過 35 cm（防止縱筋挫屈）。

---

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 為何 $0.05A_gf'_c$ 是分界點？**

ACI 318 以此值作為「顯著軸壓」的最低門檻。低於此值：
- 柱可能在地震下承受淨拉力（雙軸地震時）
- 混凝土開裂廣泛，骨料咬合喪失
- $V_c$ 的前提（軸壓克制對角拉力）不成立

因此規範保守地取 $V_c = 0$，確保箍筋能完全承擔剪力。

**② 密箍區範圍延伸規定（柱上方）**

若上方連接梁不能充分限制柱端旋轉（即梁剛度不足），$l_o$ 需從柱端延伸至梁端的塑性鉸外。
