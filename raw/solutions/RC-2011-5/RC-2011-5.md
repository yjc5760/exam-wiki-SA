**主分類：** `RC-U4-1` 預力梁斷面應力分析

**設計法：** WSD 工作應力設計法  
**標籤：** `後拉法` `有黏結腱` `合成斷面` `三斷面分析` `淨斷面` `變換斷面` `鋼腱應力` `上下緣應力` `施預力階段` `全載重階段` `n值` `相容性`

## 1. 原始題目重述 (Problem Restatement)

後拉法（後張法）預力混凝土梁斷面，給定：

**預力與載重：**
- 起始預力：$F_i = 250$ tf，有效預力：$F_0 = 220$ tf
- 梁重彎矩：$M_G = 30$ tf·m = 3,000,000 kgf·cm
- 版重彎矩：$M_S = 15$ tf·m = 1,500,000 kgf·cm
- 活載重彎矩：$M_L = 70$ tf·m = 7,000,000 kgf·cm

**材料：** $n = E_p/E_c = 6$，$A_{sp} = 24$ cm²

**淨斷面（施預力時用）：**
$A_{c1} = 2500$ cm²，$Z_{t1} = 40000$ cm³，$Z_{b1} = 39000$ cm³，$I_1 = 1{,}800{,}000$ cm⁴，$e_{p1} = 25$ cm

**變換斷面（有效預力 + 梁上載重用）：**
$A_{c2} = 2800$ cm²，$Z_{t2} = 41000$ cm³，$Z_{b2} = 43000$ cm³，$I_2 = 1{,}900{,}000$ cm⁴，$e_{p2} = 24$ cm

**合成斷面（活載重用）：**
$A_{c3} = 3900$ cm²，$Z_{t3} = 137000$ cm³，$Z_{b3} = 74000$ cm³，$I_3 = 4{,}300{,}000$ cm⁴，$e_{p3} = 39$ cm

**求：** 施預力時及全部載重作用時之**鋼腱應力**及**上下緣混凝土應力**。（20 分）

> 本題為純文字題，無附圖，無截圖需求。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
- 後拉法複合（合成）斷面預力梁，荷重分三階段施加，各階段用不同斷面：
  1. **施預力時**（灌漿前）：淨斷面 → $F_i + M_G$
  2. **有效預力 + 恆載**（灌漿後）：變換斷面 → $F_0 + M_G + M_S$
  3. **加活載**（合成後）：合成斷面 → $M_L$

- **鋼腱應力計算**：後拉法鋼腱（有黏結）與混凝土相容，鋼腱應力 = 基礎值（$F_0/A_{sp}$）+ $n \times$ 重力引起之混凝土應力變化（在腱位）

**出題者意圖：**
1. 測驗三種斷面的正確使用時機
2. 測驗預力梁彈性應力疊加（壓正拉負）的符號判斷
3. 測驗鋼腱相容性公式求腱應力（不能直接用 $F/A_{sp}$）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. **施預力時**：用淨斷面 + $F_i$ + $M_G$ → 算 fc_top、fc_bot；腱應力 = $F_i/A_{sp}$
2. **全部載重時**：
   - Step A：變換斷面，$F_0 + (M_G + M_S)$ → 算上下緣應力分量
   - Step B：合成斷面，$M_L$ → 算上下緣應力分量
   - 疊加：fc_top_total、fc_bot_total
   - 腱應力：$F_0/A_{sp}$ + $n \times$（腱位混凝土應力變化）

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ❶ 施預力用淨斷面，非全截面 | 後拉法灌漿前，導管孔洞需扣除 | 用 $A_{c1}$, $Z_{t1}$, $Z_{b1}$, $I_1$, $e_{p1}$ |
| ❷ 三個偏心距不同 | $e_{p1}=25$（淨）, $e_{p2}=24$（變換）, $e_{p3}=39$（合成） | 各階段用對應 $e_p$ |
| ❸ 腱應力≠單純 $F/A_{sp}$ | 全載重下腱應力 = $F_0/A_{sp}$ + 重力引起之相容應力 | 用 $n \times \Delta\sigma_c$ 修正 |
| ❹ $M_L$ 僅加在合成斷面 | $M_G$、$M_S$ 在非合成階段已計算；$M_L$ 後加到合成斷面 | 分清楚各載重的「時間點」 |
| ❺ 偏心引起的上下緣應力符號 | 偏心在下 → 上緣拉（負）、下緣壓（正）；重力 → 上緣壓（正）、下緣拉（負） | 逐項判斷，不要記符號，要用物理直覺 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

求梁在**施預力時**（$F_i$, $M_G$, 淨斷面）和**全部載重時**（$F_0 + M_G + M_S$ 在變換斷面、$M_L$ 在合成斷面）的上下緣混凝土應力及鋼腱應力。

### 本題關鍵公式（依計算順序）

$$\text{施預力 — 上緣：} f_{c,top}^{(1)} = \frac{F_i}{A_{c1}} - \frac{F_i e_{p1}}{Z_{t1}} + \frac{M_G}{Z_{t1}}$$

$$\text{施預力 — 下緣：} f_{c,bot}^{(1)} = \frac{F_i}{A_{c1}} + \frac{F_i e_{p1}}{Z_{b1}} - \frac{M_G}{Z_{b1}}$$

$$\text{施預力 — 腱應力：} f_{s}^{(1)} = \frac{F_i}{A_{sp}}$$

$$\text{全載 Step A — 上緣：} f_A^{top} = \frac{F_0}{A_{c2}} - \frac{F_0 e_{p2}}{Z_{t2}} + \frac{M_G+M_S}{Z_{t2}}$$

$$\text{全載 Step A — 下緣：} f_A^{bot} = \frac{F_0}{A_{c2}} + \frac{F_0 e_{p2}}{Z_{b2}} - \frac{M_G+M_S}{Z_{b2}}$$

$$\text{全載 Step B（合成斷面）：} f_B^{top} = +\frac{M_L}{Z_{t3}},\quad f_B^{bot} = -\frac{M_L}{Z_{b3}}$$

$$\text{腱位混凝土應力（重力部分）：} \sigma_{c,p}^{grav} = -\frac{(M_G+M_S)\,e_{p2}}{I_2} - \frac{M_L\,e_{p3}}{I_3}$$

$$\text{全載 — 腱應力：} f_s^{(2)} = \frac{F_0}{A_{sp}} + n \left|\sigma_{c,p}^{grav}\right|$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

> **符號規定：壓力 = 正，拉力 = 負**  
> 單位：kgf·cm（力矩），kgf/cm²（應力）

---

### 施預力時（Initial Prestress Stage）

**使用淨斷面**（$A_{c1}$, $Z_{t1}$, $Z_{b1}$, $I_1$, $e_{p1}$）

作用載重：$F_i = 250{,}000$ kgf + $M_G = 3{,}000{,}000$ kgf·cm

---

#### ① 混凝土上緣應力

$$f_{c,top}^{(1)} = \underbrace{\frac{F_i}{A_{c1}}}_{\text{軸壓}} - \underbrace{\frac{F_i\,e_{p1}}{Z_{t1}}}_{\text{偏心→上緣張}} + \underbrace{\frac{M_G}{Z_{t1}}}_{\text{重力→上緣壓}}$$

$$= \frac{250{,}000}{2500} - \frac{250{,}000\times25}{40{,}000} + \frac{3{,}000{,}000}{40{,}000}$$

$$= 100 - 156.25 + 75 = \boxed{18.75 \text{ kgf/cm}^2 \text{（壓）}}$$

---

#### ② 混凝土下緣應力

$$f_{c,bot}^{(1)} = \frac{F_i}{A_{c1}} + \frac{F_i\,e_{p1}}{Z_{b1}} - \frac{M_G}{Z_{b1}}$$

$$= 100 + \frac{250{,}000\times25}{39{,}000} - \frac{3{,}000{,}000}{39{,}000}$$

$$= 100 + 160.26 - 76.92 = \boxed{183.34 \text{ kgf/cm}^2 \text{（壓）}}$$

---

#### ③ 鋼腱應力（施預力時）

後拉法施拉時，腱應力由千斤頂直接決定：

$$\boxed{f_s^{(1)} = \frac{F_i}{A_{sp}} = \frac{250{,}000}{24} = 10{,}417 \text{ kgf/cm}^2 \text{（拉）}}$$

---

### 全部載重作用時（Full Loading Stage）

需三步疊加：**Step A**（有效預力 + 恆載，變換斷面）＋ **Step B**（活載，合成斷面）

---

#### Step A：變換斷面，$F_0 + (M_G + M_S)$

$(M_G + M_S) = (3{,}000{,}000 + 1{,}500{,}000) = 4{,}500{,}000$ kgf·cm

**上緣（beam section only）：**
$$f_A^{top} = \frac{F_0}{A_{c2}} - \frac{F_0\,e_{p2}}{Z_{t2}} + \frac{M_G+M_S}{Z_{t2}}$$

$$= \frac{220{,}000}{2800} - \frac{220{,}000\times24}{41{,}000} + \frac{4{,}500{,}000}{41{,}000}$$

$$= 78.57 - 128.78 + 109.76 = +59.55 \text{ kgf/cm}^2$$

**下緣（beam section only）：**
$$f_A^{bot} = \frac{F_0}{A_{c2}} + \frac{F_0\,e_{p2}}{Z_{b2}} - \frac{M_G+M_S}{Z_{b2}}$$

$$= 78.57 + \frac{220{,}000\times24}{43{,}000} - \frac{4{,}500{,}000}{43{,}000}$$

$$= 78.57 + 122.79 - 104.65 = +96.71 \text{ kgf/cm}^2$$

---

#### Step B：合成斷面，$M_L$（僅活載重）

**上緣：**
$$f_B^{top} = +\frac{M_L}{Z_{t3}} = +\frac{7{,}000{,}000}{137{,}000} = +51.09 \text{ kgf/cm}^2 \text{（壓）}$$

**下緣：**
$$f_B^{bot} = -\frac{M_L}{Z_{b3}} = -\frac{7{,}000{,}000}{74{,}000} = -94.59 \text{ kgf/cm}^2 \text{（拉）}$$

---

#### 全部載重時合計混凝土應力

$$\boxed{f_{c,top}^{(2)} = f_A^{top} + f_B^{top} = 59.55 + 51.09 = 110.64 \text{ kgf/cm}^2 \text{（壓）}}$$

$$\boxed{f_{c,bot}^{(2)} = f_A^{bot} + f_B^{bot} = 96.71 - 94.59 = 2.12 \text{ kgf/cm}^2 \text{（壓）}}$$

> **合理性驗證：** 下緣在全載下幾乎為零壓，符合「預力量剛好平衡最大使用荷重下底纖維張力」之良好設計原則。

---

#### 全部載重時鋼腱應力（相容性法）

有黏結後拉法灌漿後，腱與混凝土相容。重力使腱位混凝土**受張**，腱隨之**拉伸**，腱應力增加。

**基礎腱應力（有效預力）：**
$$f_{s,base} = \frac{F_0}{A_{sp}} = \frac{220{,}000}{24} = 9{,}167 \text{ kgf/cm}^2$$

**腱位混凝土因重力產生之張力（$e_p$ 以下為張力區）：**

在變換斷面（腱位 $e_{p2} = 24$ cm below beam centroid）：
$$\sigma_{c,p}^{A} = -\frac{(M_G+M_S)\,e_{p2}}{I_2} = -\frac{4{,}500{,}000\times24}{1{,}900{,}000} = -56.84 \text{ kgf/cm}^2 \text{（張）}$$

在合成斷面（腱位 $e_{p3} = 39$ cm below composite centroid）：
$$\sigma_{c,p}^{B} = -\frac{M_L\,e_{p3}}{I_3} = -\frac{7{,}000{,}000\times39}{4{,}300{,}000} = -63.49 \text{ kgf/cm}^2 \text{（張）}$$

總重力張力（腱位）：
$$\sigma_{c,p}^{grav} = -56.84 + (-63.49) = -120.33 \text{ kgf/cm}^2 \text{（張）}$$

混凝土受張 → 拉伸 → 腱也拉伸 → 腱應力**增加**：
$$\Delta f_s = n \times \left|\sigma_{c,p}^{grav}\right| = 6 \times 120.33 = +722 \text{ kgf/cm}^2$$

$$\boxed{f_s^{(2)} = f_{s,base} + \Delta f_s = 9{,}167 + 722 = 9{,}889 \text{ kgf/cm}^2 \text{（拉）}}$$

---

### 彙整成果表

| 時間點 | 上緣 $f_c$ | 下緣 $f_c$ | 鋼腱 $f_s$ |
|--------|-----------|-----------|-----------|
| **施預力時**（$F_i + M_G$，淨斷面） | 18.75 kgf/cm²（壓） | 183.34 kgf/cm²（壓） | 10,417 kgf/cm²（拉） |
| **全部載重時**（$F_0+M_G+M_S+M_L$，疊加） | 110.64 kgf/cm²（壓） | 2.12 kgf/cm²（壓） | 9,889 kgf/cm²（拉） |

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 三種斷面的使用邏輯**

| 斷面 | 組成 | 使用時機 |
|------|------|---------|
| 淨斷面（淨 $A_{c1}$） | 混凝土 - 導管孔 | 後拉法施拉時（灌漿前） |
| 變換斷面（$A_{c2} = A_{c1} + n A_{sp}$） | 淨混凝土 + 轉換鋼腱 | 灌漿後，合成前（恆載作用） |
| 合成斷面（$A_{c3}$） | 梁變換斷面 + 樓版（部分合成） | 版澆置並達強度後（活載作用） |

**② 腱應力的物理直覺**

| 狀態 | 腱應力 | 原因 |
|------|--------|------|
| 施拉時 $f_s = F_i/A_{sp}$ | 10,417 kgf/cm² | 千斤頂直接拉伸 |
| 有效預力 $f_{s,0} = F_0/A_{sp}$ | 9,167 kgf/cm² | 長期損失（潛變/收縮/鬆弛）後 |
| 全載重 $f_s^{(2)}$ | 9,889 kgf/cm² | 重力荷重使腱位混凝土拉伸，腱應力回升 |

**③ 下緣應力幾乎為零的設計意義**

$f_{c,bot}^{(2)} = 2.12$ kgf/cm²（壓）≈ 0

這表示此梁設計為「在最大活載重下，底纖維恰好接近零壓」，是預力梁 WSD 設計中常見的「四控制條件」之一：
- 施拉時底纖維壓應力 ≤ $0.6f'_{ci}$
- 施拉時頂纖維拉應力 ≤ $0.25\sqrt{f'_{ci}}$
- 使用時底纖維拉應力 ≤ $0.5\sqrt{f'_c}$（甚至取 0）
- 使用時頂纖維壓應力 ≤ $0.45f'_c$

**④ 考場速解提示**

本題計算量大，建議：
1. 先整理各分量表格（每行對應一個效應）
2. 符號方向判斷後統一方向再代入，不要每項都重推
3. 腱應力計算只需：$F_0/A_{sp}$ + $n \times$ 重力在腱位的張應力
4. 最後彙整成果表（上下緣 + 腱應力），得分率最高
