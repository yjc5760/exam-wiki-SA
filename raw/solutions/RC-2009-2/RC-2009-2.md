# 考題編號：RC-2009-2

**主分類：** `RC-U3-1` 梁工作性要求（含撓度、裂縫）  
**副分類：** 無  
**設計法：** 工作性分析（非強度設計）  
**標籤：** `懸臂梁` `即時撓度` `降伏彎矩` `裂縫斷面慣性矩` `開裂彎矩` `轉換斷面法` `曲率積分` `雙段EI` `kd中性軸` `Icr`

---

## 1. 原始題目重述 (Problem Restatement)

35 cm 寬 × 60 cm 深單筋矩形懸臂梁，自由端受單點荷重，懸臂長度 $L = 4$ m = 400 cm。

材料：$f'_c = 210$ kgf/cm²，$f_y = 2800$ kgf/cm²  
配筋：$A_s = 10.13$ cm²（拉力筋，在頂部），$d = 50$ cm（由底纖維量至鋼筋中心）

給定：
$$E_c = 15000\sqrt{f'_c} \text{ kgf/cm}^2,\quad E_s = 2.04\times10^6 \text{ kgf/cm}^2,\quad f_r = 2.0\sqrt{f'_c} \text{ kgf/cm}^2$$

**問：** 當荷重使固定端受降伏彎矩（拉力筋應變恰達降伏點），不考慮剪力影響，求**自由端即時撓度**。（25 分）

> **斷面說明：** 懸臂梁向下撓曲 → 頂纖維受拉（鋼筋在頂部）→ 底纖維受壓（壓力面）。$d = 50$ cm 從底部（壓力面）量至鋼筋中心。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
- 降伏時固定端為裂縫斷面，梁沿長度並非全部開裂（近自由端 M 較小，尚未開裂）
- 需分**裂縫區**（$I_{cr}$）與**非裂縫區**（$I_g$）分段計算曲率，再積分求撓度

**出題者意圖：**
1. 測驗轉換斷面法求裂縫中性軸 $kd$ 與 $I_{cr}$
2. 測驗開裂彎矩 $M_{cr}$ 的計算（需用 $f_r$）
3. 測驗非均一 EI 下的撓度積分（雙段 EI 法）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 計算材料常數（$E_c$、$n$、$\varepsilon_y$）
2. 用轉換斷面法求裂縫中性軸 $kd$ 和 $I_{cr}$
3. 計算全斷面慣性矩 $I_g$ 和開裂彎矩 $M_{cr}$（用 $f_r$）
4. 求降伏彎矩 $M_y$ 和對應的載重 $P$
5. 確定裂縫從固定端延伸的距離 $x_{cr}$
6. 分段積分求撓度 $\delta$

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ❶ 懸臂梁頂部受拉 | 鋼筋在頂部，$d$ 從底部（壓力面）量 | $d - kd$ = NA 到鋼筋距離 = 從 NA 往上量 |
| ❷ $yt = h/2 = 30$ cm（到頂部拉力纖維） | 矩形斷面 $M_{cr} = f_r I_g/y_t$，$y_t$ 是到**拉力**纖維距離 | 拉力纖維在頂部，$y_t = 30$ cm |
| ❸ 不能用整根梁統一的 $I_{cr}$ | 近自由端 $M < M_{cr}$，為非裂縫斷面 | 分段：裂縫區用 $I_{cr}$，非裂縫區用 $I_g$ |
| ❹ 撓度積分方法 | 用虛功法（彎矩面積法）：$\delta = \int \phi(x)(L-x)\,dx$ | 對懸臂梁成立 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

求懸臂梁在固定端降伏時的**自由端即時撓度** $\delta$。

### 本題關鍵公式（依計算順序）

$$\text{Step 1：} n = E_s/E_c,\quad \varepsilon_y = f_y/E_s$$

$$\text{Step 2（轉換斷面法）：} \frac{b\,(kd)^2}{2} = n\,A_s\,(d-kd) \quad\Rightarrow\quad kd$$

$$\text{Step 3：} I_{cr} = \frac{b\,(kd)^3}{3} + n\,A_s\,(d-\boxed{kd})^2$$

$$\text{Step 4：} M_y = A_s\,f_y\!\left(d - \frac{\boxed{kd}}{3}\right),\quad P = \frac{\boxed{M_y}}{L}$$

$$\text{Step 5：} M_{cr} = \frac{f_r\,I_g}{y_t},\quad x_{cr} = L - \frac{\boxed{M_{cr}}}{P}$$

$$\text{Step 6：} \delta = \frac{P}{E_c}\!\left[\frac{L^3-(L-x_{cr})^3}{3\,\boxed{I_{cr}}} + \frac{(L-x_{cr})^3}{3\,\boxed{I_g}}\right]$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：材料常數

$$E_c = 15000\sqrt{210} = 15000\times14.491 = 217{,}371 \text{ kgf/cm}^2$$

$$n = \frac{E_s}{E_c} = \frac{2{,}040{,}000}{217{,}371} = 9.38$$

$$\varepsilon_y = \frac{f_y}{E_s} = \frac{2800}{2{,}040{,}000} = 1.373\times10^{-3}$$

---

### Step 2：裂縫中性軸 $kd$（轉換斷面法）

裂縫後斷面：壓力區（三角形混凝土）+ 轉換鋼筋

$$\frac{b\,(kd)^2}{2} = n\,A_s\,(d-kd)$$

$$17.5\,(kd)^2 = 9.38\times10.13\times(50-kd) = 95.0\times(50-kd)$$

$$17.5\,(kd)^2 + 95.0\,(kd) - 4750 = 0$$

$$kd = \frac{-95.0 + \sqrt{95.0^2 + 4\times17.5\times4750}}{2\times17.5} = \frac{-95.0 + \sqrt{9025 + 332{,}500}}{35} = \frac{-95.0 + 584.5}{35} = \boxed{14.0 \text{ cm}}$$

> 混凝土受壓區在**底部**（壓力面）$0$ 到 $kd = 14$ cm；鋼筋在**頂部**，離底 $d - kd$... 

**斷面配置確認（懸臂梁）：**  
- 壓力面 = 底纖維（距 $kd = 14$ cm 處為 NA）  
- 拉力面 = 頂纖維  
- 鋼筋在頂部：距壓力面 $d = 50$ cm，距 NA = $d - kd = 50 - 14 = 36$ cm

---

### Step 3：裂縫斷面慣性矩 $I_{cr}$

$$I_{cr} = \frac{b\,(kd)^3}{3} + n\,A_s\,(d-kd)^2$$

$$= \frac{35\times14^3}{3} + 9.38\times10.13\times36^2$$

$$= \frac{35\times2744}{3} + 95.0\times1296$$

$$= 32{,}013 + 123{,}120 = \boxed{155{,}133 \text{ cm}^4}$$

---

### Step 4：全斷面慣性矩 $I_g$ 與開裂彎矩 $M_{cr}$

$$I_g = \frac{b\,h^3}{12} = \frac{35\times60^3}{12} = \frac{35\times216{,}000}{12} = \boxed{630{,}000 \text{ cm}^4}$$

$$f_r = 2.0\sqrt{210} = 2.0\times14.491 = 28.98 \text{ kgf/cm}^2$$

拉力纖維距斷面形心：$y_t = h/2 = 30$ cm（頂纖維）

$$M_{cr} = \frac{f_r\,I_g}{y_t} = \frac{28.98\times630{,}000}{30} = 28.98\times21{,}000 = \boxed{608{,}580 \text{ kgf·cm}}$$

---

### Step 5：降伏彎矩 $M_y$ 與載重 $P$

在降伏狀態，鋼筋應力 = $f_y$：
$$T = A_s\,f_y = 10.13\times2800 = 28{,}364 \text{ kgf}$$

彈性壓力合力 $C = T = 28{,}364$ kgf，作用於 $kd/3 = 14/3 = 4.67$ cm 處（從底部）

力臂：
$$a = d - \frac{kd}{3} = 50 - 4.67 = 45.33 \text{ cm}$$

降伏彎矩：
$$M_y = T\times a = 28{,}364\times45.33 = \boxed{1{,}285{,}700 \text{ kgf·cm} = 12.86 \text{ tf·m}}$$

**驗核混凝土應力（確認仍在彈性範圍）：**
$$\varepsilon_{c,top} = \varepsilon_y\times\frac{kd}{d-kd} = 1.373\times10^{-3}\times\frac{14}{36} = 5.34\times10^{-4} \ll \varepsilon'_{cu} \approx 2\times10^{-3} \quad \checkmark$$

自由端載重：
$$P = \frac{M_y}{L} = \frac{1{,}285{,}700}{400} = \boxed{3{,}214 \text{ kgf}}$$

---

### Step 6：確定裂縫起始位置 $x_{cr}$

裂縫從固定端（$x=0$）向自由端延伸，到 $x_{cr}$ 處彎矩降至 $M_{cr}$：

$$M(x_{cr}) = P\,(L-x_{cr}) = M_{cr}$$

$$L - x_{cr} = \frac{M_{cr}}{P} = \frac{608{,}580}{3{,}214} = 189.4 \text{ cm}$$

$$x_{cr} = 400 - 189.4 = 210.6 \text{ cm}$$

**分段：**
- $0 \le x \le 210.6$ cm：裂縫斷面，使用 $I_{cr}$
- $210.6 \le x \le 400$ cm：完整斷面，使用 $I_g$

---

### Step 7：自由端即時撓度（曲率積分）

懸臂梁自由端撓度（用虛功法，虛單位載重在自由端）：

$$\delta = \int_0^L \varphi(x)\,(L-x)\,dx$$

其中 $\varphi(x) = M(x)/[E_c\,I(x)] = P(L-x)/[E_c\,I(x)]$，故：

$$\delta = \frac{P}{E_c}\int_0^L \frac{(L-x)^2}{I(x)}\,dx = \frac{P}{E_c}\!\left[\frac{1}{I_{cr}}\int_0^{x_{cr}}(L-x)^2\,dx + \frac{1}{I_g}\int_{x_{cr}}^L (L-x)^2\,dx\right]$$

**計算各積分：**

令 $\beta = (L-x_{cr})/L = 189.4/400 = 0.4735$

$$\int_0^{x_{cr}}(L-x)^2\,dx = \frac{L^3 - (L-x_{cr})^3}{3} = \frac{400^3 - 189.4^3}{3} = \frac{64{,}000{,}000 - 6{,}796{,}100}{3} = \frac{57{,}203{,}900}{3} = 19{,}067{,}967 \text{ cm}^3$$

$$\int_{x_{cr}}^L (L-x)^2\,dx = \frac{(L-x_{cr})^3}{3} = \frac{189.4^3}{3} = \frac{6{,}796{,}100}{3} = 2{,}265{,}367 \text{ cm}^3$$

**代入數值：**

$$\delta_1 = \frac{P}{E_c\,I_{cr}}\times19{,}067{,}967 = \frac{3{,}214}{217{,}371\times155{,}133}\times19{,}067{,}967$$

$$= \frac{3{,}214}{3.373\times10^{10}}\times1.907\times10^7 = 9.533\times10^{-8}\times1.907\times10^7 = 1.818 \text{ cm}$$

$$\delta_2 = \frac{P}{E_c\,I_g}\times2{,}265{,}367 = \frac{3{,}214}{217{,}371\times630{,}000}\times2{,}265{,}367$$

$$= \frac{3{,}214}{1.369\times10^{11}}\times2.265\times10^6 = 2.348\times10^{-8}\times2.265\times10^6 = 0.0532 \text{ cm}$$

**自由端即時撓度：**

$$\boxed{\delta = \delta_1 + \delta_2 = 1.818 + 0.053 = 1.87 \text{ cm}}$$

---

### 彙整成果

| 項目 | 值 |
|------|-----|

| 裂縫斷面慣性矩 $I_{cr}$ | 155,133 cm⁴ |
| 全斷面慣性矩 $I_g$ | 630,000 cm⁴ |

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 為何不用 ACI 有效慣性矩 $I_e$ 公式？**

ACI $I_e = (M_{cr}/M_a)^3 I_g + [1-(M_{cr}/M_a)^3]I_{cr}$ 是針對**服務載重**的近似方法，將全梁的非均勻開裂效果轉換成一個均一的等效慣性矩。本題採用**分段積分**（雙段EI），物理意義更清晰且更精確。

若採用 ACI $I_e$（在固定端計算）：
$$I_e = (608{,}580/1{,}285{,}700)^3\times630{,}000 + [1-(0.4733)^3]\times155{,}133 = 0.106\times630{,}000 + 0.894\times155{,}133 = 205{,}447 \text{ cm}^4$$
$$\delta_{ACI} = PL^3/(3E_cI_e) = 1.53 \text{ cm（較小，偏保守）}$$

**② 驗算裂縫比例**

$x_{cr}/L = 210.6/400 = 52.6\%$，即梁有約 52.6% 的長度已開裂。近自由端的 47.4% 尚未開裂，此部分剛度大（$I_g$），對撓度貢獻很小（只有 0.053 cm）。

**③ 考場速解策略**

若時間緊，可用**全裂縫斷面近似**：
$$\delta_{approx} = PL^3/(3E_cI_{cr}) = 3214\times400^3/(3\times217{,}371\times155{,}133) = 2.03 \text{ cm（偏高，保守）}$$

但精確解 1.87 cm 需分段積分，是本題的考核重點。
