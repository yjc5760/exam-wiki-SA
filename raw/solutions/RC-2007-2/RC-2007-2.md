### 最終目標
`分別求 DL 撓度、持續 LL 撓度、全 LL 撓度，再疊加長期效應得 1 年與 5 年總撓度`

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } M_{cr} = \frac{f_r \cdot I_g}{y_t}, \quad f_r = 2\sqrt{f'_c}$$

$$\text{Step 2 (Icr): } \frac{b(\boxed{kd})^2}{2} + (n-1)A_s'(\boxed{kd}-d') = nA_s(d-\boxed{kd})$$

$$\text{Step 3: } I_e = \left(\frac{M_{cr}}{M_a}\right)^3\!I_g + \left[1-\left(\frac{M_{cr}}{M_a}\right)^3\right]\!\boxed{I_{cr}}$$

$$\text{Step 4: } (\Delta_i) = \frac{5wL^4}{384 E_c \boxed{I_e}}$$

$$\text{Step 5: } \lambda = \frac{\xi}{1+50\rho'}, \quad \Delta_{total} = \left[(\Delta_i)_D + (\Delta_i)_{sus}\right](1+\lambda) + \left[(\Delta_i)_l - (\Delta_i)_{sus}\right]$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：幾何與荷載

有效深度（8-D25 兩排各 4 根，保護層 4.5 cm，D10 箍筋，D25 直徑 = 2.54 cm）：

$$y_1 = 4.5 + 1.0 + 1.27 = 6.77 \text{ cm（第一排中心，距底）}$$
$$y_2 = 6.77 + 2.54 + 2.54 = 11.85 \text{ cm（第二排中心，距底，取淨間距} = d_b)$$
$$\bar{y} = \frac{4 \times 6.77 + 4 \times 11.85}{8} = 9.31 \text{ cm} \Rightarrow \boxed{d = 60 - 9.31 = 50.69 \approx 51 \text{ cm}}$$

$$w_{SW} = 2.4 \times 0.30 \times 0.60 = 0.432 \text{ tf/m} \Rightarrow w_{DL} = 0.432 + 0.1 = 0.532 \text{ tf/m} = 5.32 \text{ kgf/cm}$$
$$w_{sus} = 0.20 \times 2.0 = 0.40 \text{ tf/m} = 4.0 \text{ kgf/cm}$$
$$w_{LL} = 2.0 \text{ tf/m} = 20.0 \text{ kgf/cm}$$

**跨中彎矩（$L=800$ cm）：**
$$M_{DL} = \frac{5.32 \times 800^2}{8} = 425{,}600 \text{ kgf·cm} = 4.256 \text{ tf·m}$$
$$M_{sus} = \frac{4.0 \times 800^2}{8} = 320{,}000 \text{ kgf·cm} = 3.2 \text{ tf·m}$$
$$M_{LL} = \frac{20.0 \times 800^2}{8} = 1{,}600{,}000 \text{ kgf·cm} = 16.0 \text{ tf·m}$$

---

### Step 2：斷面性質

$$E_c = \frac{E_s}{n} = \frac{2{,}040{,}000}{8} = 255{,}000 \text{ kgf/cm}^2, \quad n = 8$$

$$I_g = \frac{30 \times 60^3}{12} = 540{,}000 \text{ cm}^4$$

$$f_r = 2\sqrt{280} = 33.47 \text{ kgf/cm}^2, \quad y_t = 30 \text{ cm}$$

$$M_{cr} = \frac{f_r I_g}{y_t} = \frac{33.47 \times 540{,}000}{30} = \boxed{602{,}460 \text{ kgf·cm} = 6.025 \text{ tf·m}}$$

**荷載水準與開裂判斷：**

$$M_{DL} = 4.256 \text{ tf·m} < M_{cr} = 6.025 \text{ tf·m} \Rightarrow \text{DL 水準未開裂，} I_e = I_g$$

$$M_{DL} + M_{sus} = 7.456 \text{ tf·m} > M_{cr} \Rightarrow \text{持續載重水準開裂}$$

$$M_{DL} + M_{LL} = 20.256 \text{ tf·m} \gg M_{cr} \Rightarrow \text{全載重水準深裂}$$

---

### Step 3：開裂轉換斷面 $I_{cr}$（$d=51$ cm，$d'=6.5$ cm）

中性軸方程式：
$$\frac{b(kd)^2}{2} + (n-1)A_s'(kd-d') = nA_s(d-kd)$$
$$15(kd)^2 + 7 \times 8.55 \times (kd-6.5) = 8 \times 40.56 \times (51-kd)$$
$$15(kd)^2 + 59.85(kd) - 389.0 = 16{,}548 - 324.5(kd)$$
$$15(kd)^2 + 384.3(kd) - 16{,}937 = 0$$
$$(kd)^2 + 25.62(kd) - 1{,}129.1 = 0$$

$$kd = \frac{-25.62 + \sqrt{25.62^2 + 4 \times 1{,}129.1}}{2} = \frac{-25.62 + \sqrt{5{,}173}}{2} = \frac{-25.62 + 71.93}{2} = \boxed{23.15 \text{ cm}}$$

$$I_{cr} = \frac{30 \times 23.15^3}{3} + 7 \times 8.55 \times (23.15-6.5)^2 + 8 \times 40.56 \times (51-23.15)^2$$

$$= \frac{30 \times 12{,}406}{3} + 59.85 \times 277.2 + 324.5 \times 776.2$$

$$= 124{,}060 + 16{,}591 + 251{,}827 = \boxed{392{,}478 \approx 392{,}500 \text{ cm}^4}$$

---

### Step 4：各水準有效慣性矩 $I_e$

**持續荷載水準** ($M_a = 745{,}600$ kgf·cm)：
$$\left(\frac{M_{cr}}{M_a}\right)^3 = \left(\frac{602{,}460}{745{,}600}\right)^3 = (0.8080)^3 = 0.5278$$

$$I_{e,sus} = 0.5278 \times 540{,}000 + 0.4722 \times 392{,}500 = 285{,}012 + 185{,}240 = \boxed{470{,}252 \text{ cm}^4}$$

**全活載重水準** ($M_a = 2{,}025{,}600$ kgf·cm)：
$$\left(\frac{M_{cr}}{M_a}\right)^3 = \left(\frac{602{,}460}{2{,}025{,}600}\right)^3 = (0.2974)^3 = 0.02626$$

$$I_{e,LL} = 0.02626 \times 540{,}000 + 0.97374 \times 392{,}500 = 14{,}180 + 382{,}242 = \boxed{396{,}422 \text{ cm}^4}$$

---

### Step 5：三項即時撓度（$\Delta_i = \frac{5wL^4}{384E_cI_e}$）

共同因子：$\frac{5L^4}{384} = \frac{5 \times 800^4}{384} = \frac{5 \times 4.096 \times 10^{11}}{384} = 5.333 \times 10^9$ cm

**(a) $({\Delta_i})_D$（DL 未開裂，$I_e = I_g = 540{,}000$ cm⁴）：**
$$EI_D = 255{,}000 \times 540{,}000 = 1.377 \times 10^{11} \text{ kgf·cm}^2$$

$$\boxed{({\Delta_i})_D = \frac{5.32 \times 5.333 \times 10^9}{1.377 \times 10^{11}} = \frac{2.837 \times 10^{10}}{1.377 \times 10^{11}} = 0.206 \text{ cm}}$$

**(b) $({\Delta_i})_{sus}$（持續 LL = 4 kgf/cm，$I_e = 470{,}252$ cm⁴）：**
$$EI_{sus} = 255{,}000 \times 470{,}252 = 1.199 \times 10^{11}$$

$$\boxed{({\Delta_i})_{sus} = \frac{4.0 \times 5.333 \times 10^9}{1.199 \times 10^{11}} = \frac{2.133 \times 10^{10}}{1.199 \times 10^{11}} = 0.178 \text{ cm}}$$

**(c) $({\Delta_i})_l$（全 LL = 20 kgf/cm，$I_e = 396{,}422$ cm⁴）：**
$$EI_{LL} = 255{,}000 \times 396{,}422 = 1.011 \times 10^{11}$$

$$\boxed{({\Delta_i})_l = \frac{20.0 \times 5.333 \times 10^9}{1.011 \times 10^{11}} = \frac{1.067 \times 10^{11}}{1.011 \times 10^{11}} = 1.055 \text{ cm}}$$

---

### Step 6：長期修正係數 $\lambda$

$$\rho' = \frac{A_s'}{b \cdot d} = \frac{8.55}{30 \times 51} = 0.005588$$

| 時間 | $\xi$ | $\lambda = \xi/(1+50\rho')$ |
|------|-------|---------------------------|
| 1 年 | 1.4 | $1.4/(1+0.2794) = 1.094$ |
| 5 年 | 2.0 | $2.0/1.2794 = 1.563$ |

---

### Step 7：長期總撓度

持續載重之即時撓度：
$$(\Delta_i)_{sustained} = ({\Delta_i})_D + ({\Delta_i})_{sus} = 0.206 + 0.178 = 0.384 \text{ cm}$$

非持續 LL 之即時撓度：
$$(\Delta_i)_{non-sus} = ({\Delta_i})_l - ({\Delta_i})_{sus} = 1.055 - 0.178 = 0.877 \text{ cm}$$

長期總撓度公式：
$$\Delta_{total} = (\Delta_i)_{sustained} \times (1+\lambda) + (\Delta_i)_{non-sus}$$

**(d) 一年後總撓度：**
$$\Delta_{total,1yr} = 0.384 \times (1 + 1.094) + 0.877 = 0.384 \times 2.094 + 0.877 = 0.804 + 0.877 = \boxed{1.681 \approx 1.68 \text{ cm}}$$

**(e) 五年後總撓度：**
$$\Delta_{total,5yr} = 0.384 \times (1 + 1.563) + 0.877 = 0.384 \times 2.563 + 0.877 = 0.984 + 0.877 = \boxed{1.861 \approx 1.86 \text{ cm}}$$

### 爭議：DL 未裂但加上持續 LL 後才裂

本題有一特殊現象：DL 單獨作用時 $M_{DL} = 4.26 < M_{cr} = 6.02$ tf·m（未裂），但一旦加上 20%LL = 3.2 tf·m，合計 7.46 tf·m > 6.02 tf·m（開裂）。

這表示：
- 計算 $({\Delta_i})_D$ 時使用 $I_g$（正確）
- 計算 $({\Delta_i})_{sus}$ 時使用 $I_{e,sus}$（基於 $M_{DL}+M_{sus}$），此 $I_e$ 已反映裂縫狀態

若改用：$({\Delta_i})_{sus} = \Delta_{DL+sus} - ({\Delta_i})_D$ 的差值法（同一 $I_{e,sus}$）：
$\Delta_{DL+sus} = 9.32 \times 5.333 \times 10^9/1.199 \times 10^{11} = 0.415$ cm
$({\Delta_i})_{sus,diff} = 0.415 - 0.206 = 0.209$ cm

這與直接法（0.178 cm）略有差異，是因為不同 $I_e$ 的非線性。兩種方法在考試中均可接受，本解採直接法。

### $\xi = 1.4$（1 年）vs $\xi = 2.0$（≥5 年）

$\xi$ 值對應關係：3 個月 = 1.0，6 個月 = 1.2，1 年 = 1.4，5 年以上 = 2.0。
