## 1. 原始題目重述 (Problem Restatement)

**題目（10 分）：**

四週有邊梁的 RC 雙向樓版，其角隅處在配筋上應作何特殊考慮？試繪圖說明之。

**已知條件：**

*(本題為純文字概念題，無需題目附圖)*

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
雙向版在均佈載重作用下，版面趨向形成「碟形」（dish shape）向下撓曲。四邊有梁支撐時，梁會抵抗版邊的撓曲，但角隅因兩方向梁的交匯，若梁無法完全阻止，版角隅在版面上的投影呈「翹起」趨勢（corner lifting）。此翹起趨勢會在角隅版面產生：

1. **對角線方向（45°）** 的負彎矩：版角翹起 → 對角線方向上緣受拉
2. **垂直對角線方向（135°）** 的正彎矩：版角翹起 → 上述垂直方向下緣受拉

這兩組彎矩的存在，使角隅版面同時在上下緣都需要配置鋼筋，形成「角隅特殊配筋」的要求。

**出題者意圖：**
- 測驗考生是否理解雙向版角隅「翹起效應」的力學機理
- 測驗是否知道需在角隅同時配置**上層**與**下層**鋼筋
- 測驗鋼筋布置範圍與數量的規範規定（$l_n/5$）
- 有繪圖說明要求，需要具體展示配筋細部

**關鍵陷阱：**

| # | 陷阱 | 應對 |
|---|------|------|
| ① | 只說「上層加筋」，忘記下層也需要配置 | 角隅翹起同時產生上下緣拉力，上下層都需配筋 |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：角隅翹起效應說明

均佈載重下，雙向版趨向向下撓曲呈碟形。四邊若設有梁支撐，梁提供線支撐力，但**版角隅無直接向下的支撐反力**。分析可知，四邊梁以扭矩（扭轉反力）來維持版角的位置，若無此扭矩（例如角落沒有束制），版角會**上翹**。

當四週均有梁（即角隅上方有梁的束制），版角雖無法自由上翹，但在版角附近仍存在**二維彎矩場**：

- 沿對角線 45° 方向：**負彎矩**（上緣受拉，下緣受壓）
- 垂直對角線方向：**正彎矩**（下緣受拉，上緣受壓）

$$M_{45°}^{-} \text{（上緣拉力）} \qquad M_{135°}^{+} \text{（下緣拉力）}$$

因此，角隅版面同時需要**上層**與**下層**鋼筋。

### Step 2：規範規定（ACI 318 §13.3.6 / CNS 1480）

配筋要求如下：

**(a) 配置位置：**
角隅上方需設置**特殊角隅配筋**，包含：
- 上層配筋：平行 x 軸（一層）+ 平行 y 軸（一層），共兩組
- 下層配筋：平行 x 軸（一層）+ 平行 y 軸（一層），共兩組

**(b) 延伸範圍：**
每組鋼筋各自從角點沿版面延伸，向兩側各延伸：

$$\boxed{L = \frac{\ell_n}{5}}$$

其中 $\ell_n$ 取兩方向中**較短跨度**的淨跨。

**(c) 鋼筋量：**
每層每方向的鋼筋量，等於版在**最大正彎矩區段**的設計鋼筋量：

$$A_{s,corner} = A_{s,max+}$$

### Step 3：角隅配筋示意圖說明

```
         ← ℓn/5 →                  ← ℓn/5 →
    ┌────┬──────────────────────────┬────┐
    │▒▒▒│                          │▒▒▒│  ← 上層筋（x＋y 兩組）
    │▒▒▒│                          │▒▒▒│     各延伸 ℓn/5
  ↑ ├────┤                          ├────┤
ℓn/5│    │                          │    │
  ↓ │    │   版面中央區（一般配筋）  │    │
    │    │                          │    │
  ↑ ├────┤                          ├────┤
ℓn/5│▒▒▒│                          │▒▒▒│  ← 下層筋（x＋y 兩組）
  ↓ │▒▒▒│                          │▒▒▒│     各延伸 ℓn/5
    └────┴──────────────────────────┴────┘

  ▒▒▒ = 角隅特殊配筋區（同時有上層筋 + 下層筋）
```

**斷面細部說明（角隅區斷面）：**
```
         上層筋（x 方向）
         ▓▓▓▓▓▓▓▓▓▓
  ═══════════════════════  版頂面
  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   上層鋼筋層（x 向）
  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   上層鋼筋層（y 向）
                           版心
  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   下層鋼筋層（x 向）
  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   下層鋼筋層（y 向）
  ═══════════════════════  版底面
```

**結論：**

> 四週有邊梁之雙向版，角隅處因版角翹起效應，在對角線方向產生負彎矩、垂直對角線方向產生正彎矩。應在角隅區域上、下各配置 x 及 y 兩方向鋼筋，每方向延伸距離為 $\ell_n/5$，鋼筋量等於版面最大正彎矩區之設計鋼筋量。

$$\boxed{L_{延伸} = \frac{\ell_n}{5},\quad A_{s,corner} = A_{s,max+},\quad \text{上下層各設 x、y 兩方向鋼筋}}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**1. 邊梁「完全束制」vs「部分束制」：**
若版角上方有梁，但梁的抗扭勁度很低，則版角仍會有部分翹起，此時角隅配筋更顯重要。ACI 318 明確規定當版由梁在兩方向支撐時，角隅筋為必要要求。

**2. 可否用 45° 斜向配筋？**
規範允許採用平行版邊（x/y 方向）配筋或 45°/135° 對角線配筋。
- 45° 對角線配筋：理論上直接對應彎矩方向，鋼筋效率最高
- 平行版邊配筋：施工方便，為實務中常用做法
兩者均符合規範，考場上以「平行版邊 + 說明」最為完整安全。

**3. 角隅特殊配筋範圍是否含頂層普通筋？**
角隅配筋為**附加**配筋，置於一般版筋之上（或合併設計時需確認），不可以一般版筋替代角隅特殊筋的要求。

**4. 若版角為自由端（無梁）：**
若版角僅有牆支撐或完全自由，此條款不適用。角隅特殊配筋僅適用於**版邊由梁支撐**的情形。
