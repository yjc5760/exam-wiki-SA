## 1. 原始題目重述 (Problem Restatement)

於**建築物評估**時，已知一梁：

- $b = 30\ \text{cm}$，$d = 50\ \text{cm}$
- $A_s = 6\ \text{根 D25} = 30.4\ \text{cm}^2$
- $f'_c = 140\ \text{kgf/cm}^2$，$f_y = 4{,}200\ \text{kgf/cm}^2$

假設規範之計算公式仍然適用，**試求 $\phi M_n$**。

若因該梁強度不足，將其寬度向兩側各擴大 $15\ \text{cm}$，成為：

- $b = 60\ \text{cm}$（$d$ 不變，仍為 $50\ \text{cm}$）
- 新增混凝土強度 $f'_{c,\text{新}} = 350\ \text{kgf/cm}^2$（原核心 $30\ \text{cm}$ 仍為 $f'_{c,\text{原}} = 140\ \text{kgf/cm}^2$）
- 鋼筋量 $A_s = 8\ \text{根 D25} = 40.54\ \text{cm}^2$

**試求加寬後之 $\phi M_n$**。（25 分）

> **題目說明**：本題為建築物評估情境，全題採用 ACI/CNS USD 強度設計法。題目無附圖，所有幾何與材料條件均以文字給定。

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`Part 1：過筋梁應變相容求 φMn（壓力控制）｜Part 2：複合斷面平衡求 φMn（拉力控制）`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

**Part 1（過筋梁）：**

$$\text{Step 1:}\quad \beta_1 = 0.85 \quad (f'_c = 140 \leq 280)$$

$$\text{Step 2:}\quad \rho_b = \frac{0.85\,\beta_1\,f'_c}{f_y} \cdot \frac{6120}{6120 + f_y}$$

$$\text{Step 3:}\quad a_1 = \beta_{1,1}\,\boxed{c},\quad a_2 = \beta_{1,2}\,\boxed{c}$$

$$\text{Step 4:}\quad M_n = C_1\!\left(d - \frac{\boxed{a_1}}{2}\right) + C_2\!\left(d - \frac{\boxed{a_2}}{2}\right)$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Part 1：原梁（$b=30$ cm，$f'_c=140$ kgf/cm²，$A_s=30.4$ cm²）

**Step 1：確認 $\beta_1$**

$$\beta_1 = 0.85 \quad (f'_c = 140\ \text{kgf/cm}^2 \leq 280\ \text{kgf/cm}^2)$$

**Step 2：計算平衡鋼筋比，判斷過筋**

$$\rho_b = \frac{0.85\,\beta_1\,f'_c}{f_y} \cdot \frac{6120}{6120 + f_y} = \frac{0.85 \times 0.85 \times 140}{4200} \cdot \frac{6120}{6120 + 4200}$$

$$= \frac{101.15}{4200} \times \frac{6120}{10320} = 0.02408 \times 0.5930 = 0.01428$$

$$\rho = \frac{A_s}{b\,d} = \frac{30.4}{30 \times 50} = \frac{30.4}{1500} = 0.02027$$

$$\rho = 0.02027 > \rho_b = 0.01428 \quad \Rightarrow \quad \textbf{過筋梁，鋼筋不降伏，須應變相容}$$

**Step 3：應變相容求中性軸深度 $c$**

假設混凝土到達極限應變 $\varepsilon_{cu} = 0.003$，鋼筋應力 $f_s = E_s \varepsilon_s$：

力平衡（$C = T$）：

$$0.85\,f'_c\,b\,(\beta_1 c) = A_s\,E_s\,\varepsilon_{cu}\,\frac{d-c}{c}$$

$$0.85 \times 140 \times 30 \times 0.85\,c = 30.4 \times 2.04\times10^6 \times 0.003 \times \frac{50-c}{c}$$

$$3034.5\,c = 186{,}048\,\frac{50-c}{c}$$

$$3034.5\,c^2 = 186{,}048\,(50 - c) = 9{,}302{,}400 - 186{,}048\,c$$

$$\boxed{3034.5\,c^2 + 186{,}048\,c - 9{,}302{,}400 = 0}$$

使用二次公式（取正根）：

$$c = \frac{-186{,}048 + \sqrt{186{,}048^2 + 4 \times 3034.5 \times 9{,}302{,}400}}{2 \times 3034.5}$$

$$= \frac{-186{,}048 + \sqrt{34{,}613{,}858{,}304 + 112{,}912{,}531{,}200}}{6069}$$

$$= \frac{-186{,}048 + \sqrt{147{,}526{,}389{,}504}}{6069} = \frac{-186{,}048 + 384{,}092}{6069}$$

$$\boxed{c = 32.63\ \text{cm}}$$

等效壓力塊深度：$a = \beta_1\,c = 0.85 \times 32.63 = \boxed{27.74\ \text{cm}}$

驗算（確認確為過筋）：$c_b = \frac{6120}{6120+4200} \times 50 = 29.65\ \text{cm}$，$c = 32.63 > c_b$ ✓

**Step 4：計算 $\varepsilon_t$、$f_s$、$\phi$**

$$\varepsilon_t = \varepsilon_{cu} \cdot \frac{d - c}{c} = 0.003 \times \frac{50 - 32.63}{32.63} = 0.003 \times 0.5323 = \boxed{0.001597}$$

$$\varepsilon_t = 0.001597 < 0.002 \quad \Rightarrow \quad \textbf{壓力控制斷面，} \phi = 0.65$$

$$f_s = E_s\,\varepsilon_t = 2.04\times10^6 \times 0.001597 = \boxed{3{,}258\ \text{kgf/cm}^2} < f_y = 4200\ \text{kgf/cm}^2\ ✓$$

**Step 5：計算 $M_n$ 與 $\phi M_n$**

$$M_n = A_s\,f_s\!\left(d - \frac{a}{2}\right) = 30.4 \times 3258 \times \left(50 - \frac{27.74}{2}\right)$$

$$= 30.4 \times 3258 \times 36.13 = 30.4 \times 117{,}672 = 3{,}577{,}229\ \text{kgf·cm}$$

（驗算：$C = 0.85\times140\times30\times27.74 = 119\times30\times27.74 = 99{,}054\ \text{kgf} \approx T = 30.4\times3258 = 99{,}043\ \text{kgf}$ ✓）

$$\phi M_n = 0.65 \times 3{,}577{,}229 = 2{,}325{,}199\ \text{kgf·cm}$$

$$\boxed{\phi M_n \approx 23.25\ \text{tf·m}} \quad \text{（壓力控制，} \phi = 0.65\text{）}$$

---

### Part 2：加寬後複合斷面（$b=60$ cm，$d=50$ cm，$A_s=40.54$ cm²）

**斷面配置：**
- 中央核心 $b_1 = 30\ \text{cm}$，$f'_{c,1} = 140\ \text{kgf/cm}^2$，$\beta_{1,1} = 0.85$
- 兩側新增 $b_2 = 2\times15 = 30\ \text{cm}$，$f'_{c,2} = 350\ \text{kgf/cm}^2$，$\beta_{1,2} = ?$

**Step 1：計算 $\beta_{1,2}$**

$$\beta_{1,2} = 0.85 - 0.05 \times \frac{f'_{c,2} - 280}{70} = 0.85 - 0.05 \times \frac{350 - 280}{70} = 0.85 - 0.05 = \boxed{0.80}$$

**Step 2：複合斷面力平衡求 $c$（假設鋼筋降伏）**

兩種混凝土各貢獻壓力，共享同一中性軸深度 $c$，等效應力塊深度分別為 $a_1 = 0.85c$ 和 $a_2 = 0.80c$：

$$C_1 = 0.85\,f'_{c,1}\,b_1\,a_1 = 0.85\times140\times30\times0.85\,c = 3034.5\,c$$

$$C_2 = 0.85\,f'_{c,2}\,b_2\,a_2 = 0.85\times350\times30\times0.80\,c = 7140\,c$$

$$T = A_s\,f_y = 40.54 \times 4200 = 170{,}268\ \text{kgf}$$

力平衡：

$$(3034.5 + 7140)\,c = 170{,}268$$

$$10{,}174.5\,c = 170{,}268 \quad \Rightarrow \quad \boxed{c = 16.74\ \text{cm}}$$

**Step 3：驗算鋼筋降伏 & $\phi$**

$$\varepsilon_t = 0.003 \times \frac{50 - 16.74}{16.74} = 0.003 \times 1.987 = \boxed{0.005961}$$

$$\varepsilon_t = 0.00596 \gg \varepsilon_y = \frac{4200}{2.04\times10^6} = 0.00206 \quad \Rightarrow\ \text{鋼筋降伏假設成立} ✓$$

$$\varepsilon_t > 0.005 \quad \Rightarrow \quad \textbf{拉力控制斷面，} \phi = 0.90$$

**Step 4：計算壓力合力 $C_1$、$C_2$ 及其力臂**

$$a_1 = \beta_{1,1}\,c = 0.85 \times 16.74 = 14.23\ \text{cm}$$

$$a_2 = \beta_{1,2}\,c = 0.80 \times 16.74 = 13.39\ \text{cm}$$

$$C_1 = 3034.5 \times 16.74 = 50{,}798\ \text{kgf} \quad \text{（作用點距頂面}\ a_1/2 = 7.115\ \text{cm）}$$

$$C_2 = 7140 \times 16.74 = 119{,}524\ \text{kgf} \quad \text{（作用點距頂面}\ a_2/2 = 6.695\ \text{cm）}$$

驗算：$C_1 + C_2 = 170{,}322\ \text{kgf} \approx T = 170{,}268\ \text{kgf}$ ✓

**Step 5：計算 $M_n$ 與 $\phi M_n$（力矩對拉力鋼筋取矩）**

$$M_n = C_1\!\left(d - \frac{a_1}{2}\right) + C_2\!\left(d - \frac{a_2}{2}\right)$$

$$= 50{,}798 \times (50 - 7.115) + 119{,}524 \times (50 - 6.695)$$

$$= 50{,}798 \times 42.885 + 119{,}524 \times 43.305$$

$$= 2{,}178{,}499 + 5{,}175{,}626 = 7{,}354{,}125\ \text{kgf·cm}$$

$$\phi M_n = 0.90 \times 7{,}354{,}125 = 6{,}618{,}713\ \text{kgf·cm}$$

$$\boxed{\phi M_n \approx 66.19\ \text{tf·m}} \quad \text{（拉力控制，} \phi = 0.90\text{）}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 過筋梁評估的 φ 值爭議**

本題 Part 1 中，$\varepsilon_t = 0.00160$，屬**壓力控制**（$\varepsilon_t < 0.002$），故 $\phi = 0.65$。

考場上常見的爭議是：「梁怎麼會用柱的 $\phi = 0.65$？」

答：現行 CNS/ACI 的 $\phi$ 規定是依**應變狀態**而非**構件類型**決定。壓力控制斷面（無論梁或柱）均用 $\phi = 0.65$（箍筋柱）。本題既有梁因過筋而形成壓力控制斷面，合理適用 $\phi = 0.65$。這正是規範禁止過筋梁的設計哲學所在——強迫設計者在張拉控制區工作，確保 $\phi = 0.90$。

**② 複合斷面的簡化爭議**

Part 2 的加寬斷面在考場上有兩種常見做法：

| 做法 | 說明 | 合理性 |
|------|------|--------|
| **嚴格法（本解）** | 兩種 $f'_c$ 各自計算 $\beta_1$ 和 $a$，分開計算壓力合力 | 理論正確，但較繁瑣 |
| **保守簡化法** | 整個 60 cm 斷面均採用較低的 $f'_c = 140$ | 過度保守，低估加寬效益 |

**考場建議**：採用嚴格法。步驟清晰，不難計算，且顯示對規範的精確掌握。

**③ 加寬補強效益**

比較兩種斷面的結果：
