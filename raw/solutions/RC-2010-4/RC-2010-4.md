## 1. 原始題目重述 (Problem Restatement)

簡支預力混凝土梁，跨度 $l = 15$ m，斷面 $b = 50$ cm，$h = 90$ cm  
有效預力比：$f_{pe}/f_{pi} = 0.85$，初終混凝土強度：$f'_{ci} = 280$ kgf/cm²，$f'_c = 350$ kgf/cm²  
除自重外，均佈活重 $W_l = 1.5$ tf/m

**(一)（5 分）** 採用**曲線型鋼腱**時，其可承受之載重能力可較**直線型鋼腱**提升多少？

**(二)（15 分）** 若初始預力 $P_i = 200$ tf，試求**梁中央點**鋼腱配置之**偏心距限制**。

> 本題為純文字計算題，無附圖，無截圖需求。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**
- **(一)** 曲線型鋼腱透過「等效載重」產生向上分佈力，可平衡重力載重；直線型（水平）鋼腱無此效果
- **(二)** 偏心距過小 → 服務載重下底纖維受拉；偏心距過大 → 施拉時頂纖維受拉；四控制條件確定允許範圍

**出題者意圖：**
1. 測驗等效載重法概念（曲線腱 vs 直線腱）
2. 測驗四個應力控制條件的設定與求解偏心距範圍

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

確定梁中央點鋼腱偏心距 $e_p$ 的允許範圍 $e_{p,\min} \le e_p \le e_{p,\max}$。

### 本題關鍵公式（依計算順序）

$$\text{Step 1：} W_G = \gamma_c \cdot b \cdot h,\quad M_G = \frac{W_G l^2}{8},\quad M_L = \frac{W_l l^2}{8}$$

$$\text{Step 2：} P_e = 0.85\,P_i,\quad A = b\,h,\quad Z = \frac{b\,h^2}{6}$$

$$\text{Step 3（傳遞）：} \sigma_{t,i} = \frac{P_i}{A} - \frac{P_i\,e_p}{Z} + \frac{M_G}{Z},\quad \sigma_{b,i} = \frac{P_i}{A} + \frac{P_i\,e_p}{Z} - \frac{M_G}{Z}$$

$$\text{Step 4（服務）：} \sigma_{t,s} = \frac{P_e}{A} - \frac{P_e\,e_p}{Z} + \frac{M_G+M_L}{Z},\quad \sigma_{b,s} = \frac{P_e}{A} + \frac{P_e\,e_p}{Z} - \frac{M_G+M_L}{Z}$$

$$\text{Step 5（四條件）：上限從} \sigma_{t,i} \ge 0 \to e_{p,\max};\quad \text{下限從} \sigma_{b,s} \ge 0 \to e_{p,\min}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Part (一)：曲線型 vs 直線型鋼腱之載重提升

---

**直線型鋼腱（水平，偏心距為零）：**
- 僅提供軸向壓力 $P_e/A$
- 無等效向上分佈載重

**曲線型（拋物線）鋼腱，中央偏心 $e_p$，端部偏心為零：**
- 透過等效載重法，產生向上均佈等效載重：

$$W_e = \frac{8\,P_e\,e_p}{l^2}$$

- 此向上力平衡部分重力載重，等同於梁可多承受 $W_e$ 的均佈載重

**載重提升量：**

$$\boxed{\Delta W = W_e = \frac{8\,P_e\,e_p}{l^2}}$$

使用 $P_e = 170$ tf 及 Part (二) 求得之 $e_p \approx 29$ cm = 0.29 m（代表性取值）：

$$W_e = \frac{8 \times 170 \times 0.29}{15^2} = \frac{394.4}{225} \approx 1.75 \text{ tf/m}$$

由於 $W_e = 1.75 \text{ tf/m} > W_l = 1.5 \text{ tf/m}$，曲線型鋼腱可完全平衡活載重，尚有餘裕。

> 結論：曲線型鋼腱的等效向上載重 $W_e = 8P_e e_p/l^2$，此即較直線型鋼腱額外能承受的均佈載重量。

---

### Part (二)：梁中央點偏心距限制

---

#### Step 1：截面性質

$$A = 50 \times 90 = 4{,}500 \text{ cm}^2$$

$$Z = Z_t = Z_b = \frac{50 \times 90^2}{6} = \frac{50 \times 8100}{6} = 67{,}500 \text{ cm}^3$$

$$P_e = 0.85 \times 200{,}000 = 170{,}000 \text{ kgf}$$

---

#### Step 2：載重計算

$$W_G = 2400 \times 0.5 \times 0.9 = 1{,}080 \text{ kgf/m} = 10.80 \text{ kgf/cm}$$

$$M_G = \frac{10.80 \times 1500^2}{8} = 10.80 \times 281{,}250 = 3{,}037{,}500 \text{ kgf·cm}$$

$$M_L = \frac{15.0 \times 1500^2}{8} = 15.0 \times 281{,}250 = 4{,}218{,}750 \text{ kgf·cm}$$

$$M_G + M_L = 7{,}256{,}250 \text{ kgf·cm}$$

---

#### Step 3：中央斷面纖維應力公式

**符號規定：壓力 = 正，拉力 = 負；$e_p > 0$ 表示鋼腱在形心以下。**

**傳遞階段（$P_i = 200{,}000$ kgf，$M_G$）：**

$$\sigma_{t,i} = \frac{P_i}{A} - \frac{P_i\,e_p}{Z} + \frac{M_G}{Z} = \frac{200{,}000}{4500} - \frac{200{,}000\,e_p}{67{,}500} + \frac{3{,}037{,}500}{67{,}500}$$

$$= 44.44 - 2.963\,e_p + 45.00 = \boxed{89.44 - 2.963\,e_p}$$

$$\sigma_{b,i} = \frac{P_i}{A} + \frac{P_i\,e_p}{Z} - \frac{M_G}{Z} = 44.44 + 2.963\,e_p - 45.00 = \boxed{-0.556 + 2.963\,e_p}$$

**服務階段（$P_e = 170{,}000$ kgf，$M_G + M_L$）：**

$$\sigma_{t,s} = \frac{P_e}{A} - \frac{P_e\,e_p}{Z} + \frac{M_G+M_L}{Z} = \frac{170{,}000}{4500} - \frac{170{,}000\,e_p}{67{,}500} + \frac{7{,}256{,}250}{67{,}500}$$

$$= 37.78 - 2.519\,e_p + 107.5 = \boxed{145.28 - 2.519\,e_p}$$

$$\sigma_{b,s} = \frac{P_e}{A} + \frac{P_e\,e_p}{Z} - \frac{M_G+M_L}{Z} = 37.78 + 2.519\,e_p - 107.5 = \boxed{-69.72 + 2.519\,e_p}$$

---

#### Step 4：容許應力條件（四控制條件）

| 編號 | 條件 | 公式 | 解 $e_p$ | 是否控制 |
|------|------|------|---------|---------|
| ① | 傳遞，頂壓 ≤ 168 | $89.44-2.963e_p \le 168$ | $e_p \ge -26.5$ cm | ❌ 不控制（$e_p > 0$） |
| ② | 傳遞，底壓 ≤ 168 | $-0.556+2.963e_p \le 168$ | $e_p \le 56.9$ cm | ❌ 不控制 |
| ③ | **傳遞，頂拉 ≥ 0** | $89.44-2.963e_p \ge 0$ | $e_p \le 30.18$ cm | ✅ **上限** |
| ④ | 傳遞，底拉 ≥ 0 | $-0.556+2.963e_p \ge 0$ | $e_p \ge 0.19$ cm | ❌ 不控制 |
| ⑤ | 服務，頂壓 ≤ 157.5 | $145.28-2.519e_p \le 157.5$ | $e_p \ge -4.85$ cm | ❌ 不控制 |
| ⑥ | 服務，頂拉 ≥ 0 | $145.28-2.519e_p \ge 0$ | $e_p \le 57.7$ cm | ❌ 不控制 |
| ⑦ | 服務，底壓 ≤ 157.5 | $-69.72+2.519e_p \le 157.5$ | $e_p \le 89.8$ cm | ❌ 不控制 |
| ⑧ | **服務，底拉 ≥ 0** | $-69.72+2.519e_p \ge 0$ | $e_p \ge 27.68$ cm | ✅ **下限** |

---

#### Step 5：偏心距允許範圍

$$\boxed{27.68 \text{ cm} \le e_p \le 30.18 \text{ cm}}$$

**驗算：**

取 $e_p = 27.68$ cm（下限）：

| 纖維 | 應力 | 驗核 |
|------|------|------|
| 傳遞頂 | $89.44 - 2.963\times27.68 = +7.44$ kgf/cm² | ≥ 0 ✅ |
| 傳遞底 | $-0.556 + 2.963\times27.68 = +81.44$ kgf/cm² | ≤ 168 ✅ |
| 服務頂 | $145.28 - 2.519\times27.68 = +75.55$ kgf/cm² | ≤ 157.5 ✅ |
| 服務底 | $-69.72 + 2.519\times27.68 = +0.01 \approx 0$ kgf/cm² | ≥ 0 ✅ |

取 $e_p = 30.18$ cm（上限）：

| 纖維 | 應力 | 驗核 |
|------|------|------|
| 傳遞頂 | $89.44 - 2.963\times30.18 \approx 0$ kgf/cm² | ≥ 0 ✅ |
| 傳遞底 | $-0.556 + 2.963\times30.18 = +88.86$ kgf/cm² | ≤ 168 ✅ |
| 服務頂 | $145.28 - 2.519\times30.18 = +69.26$ kgf/cm² | ≤ 157.5 ✅ |
| 服務底 | $-69.72 + 2.519\times30.18 = +6.30$ kgf/cm² | ≥ 0 ✅ |

---

**物理解釋：**
- $e_{p,\min} = 27.68$ cm：若偏心距再小，服務載重下底纖維產生拉應力（預壓不足抵抗 $M_G + M_L$）
- $e_{p,\max} = 30.18$ cm：若偏心距再大，傳遞時頂纖維產生拉應力（偏心彎矩過大，$M_G$ 不足抵消）

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 控制條件分析（為何只有③和⑧控制）**

| 控制條件 | 物理成因 | 所受關鍵力 |
|---------|---------|----------|
| ③ 傳遞頂拉（e_p 上限） | 傳遞時活載未存在，$M_G$ 較小 → 偏心大時頂纖維被拉 | $P_i$（大）、$M_G$（中） |
| ⑧ 服務底拉（e_p 下限） | 服務時活載很大，$M_G+M_L$ 很大 → 偏心小時底纖維被拉 | $P_e$（小，損失後）、$M_G+M_L$（大） |

**② 為何允許範圍很窄（僅約 2.5 cm）？**

矩形截面的 $Z_t = Z_b$（上下緣對稱），使得：
$$e_{p,\max} - e_{p,\min} = \frac{(P_i/A)(f_{ci,c} + f_{ci,t}) + (P_e/A)(f_{c,t} + f_{c,c})...}{...}$$

本題預力損失大（fpe/fpi = 0.85），加上活載 $W_l$ 大（1.5 tf/m），導致 $P_e$ 與 $M_G+M_L$ 之間的差距使範圍縮窄。

**③ 若允許小量拉應力（寬鬆標準）**

若採用：
- 傳遞頂拉 ≥ $-0.25\sqrt{f'_{ci}} = -4.18$ kgf/cm²（小量拉力許可）
- 服務底拉 ≥ $-0.5\sqrt{f'_c} = -9.35$ kgf/cm²

則：
- 上限：$89.44 - 2.963e_p \ge -4.18 \Rightarrow e_p \le 31.60$ cm
- 下限：$-69.72 + 2.519e_p \ge -9.35 \Rightarrow e_p \ge 24.00$ cm

允許範圍擴大為：$24.00 \le e_p \le 31.60$ cm（寬鬆約 7.6 cm）

**④ 鋼腱實際偏心限制（幾何約束）**

實際安裝時，鋼腱中心與底面之間需有最小保護層（如 5-8 cm）：
$$e_{p,\text{max, practical}} = \frac{h}{2} - \text{底蓋層} = 45 - 8 = 37 \text{ cm}$$

本題計算的 30.18 cm < 37 cm，故幾何限制不控制。

**⑤ Part (一) 的等效載重公式推導**

拋物線鋼腱在端點為 0、中央為 $e_p$ 的曲率下：
鋼腱曲率半徑 $R = l^2/(8e_p)$（近似值）
等效向上均佈力 = $P_e / R = 8P_e e_p / l^2$ ✓
