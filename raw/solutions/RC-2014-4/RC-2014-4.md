## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心精神：**
預力混凝土設計的工作應力法（WSD）核心是「**疊加**」——在梁中點，斷面承受三個來源的應力（均布預力 + 偏心引起的彎矩預應力 + 外力彎矩），依據彈性理論逐項計算後代數疊加。

**兩階段的本質差異：**
- **Stage 1（施加預力時）**：預力最大（$P_i$），外力彎矩最小（僅自重）。頂纖維可能受拉，底纖維大壓。
- **Stage 2（服務載重後）**：預力已損失（$P_e$），外力彎矩最大（自重＋活載重）。底纖維可能出現過大拉應力。

**出題者意圖：**
1. 測試兩階段的正確「換料」（Stage 1 用 $P_i$、$f'_{ci}$；Stage 2 用 $P_e$、$f'_c$）
2. 驗算規範容許應力（尤其是 Stage 2 底纖維是否拉應力超限）

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

```
前置計算：
  → S = I_c/c₁，各力項 (P/A)、(Pe/S)、(M/S) 逐一算出

Stage 1（施加預力時）：P = Pi，M = Md（僅自重）
  f_top_1 = Pi/Ac - Pi×e/S + Md/S
  f_bot_1 = Pi/Ac + Pi×e/S - Md/S

Stage 2（服務載重後）：P = Pe，M = MT（自重＋活載重）
  f_top_2 = Pe/Ac - Pe×e/S + MT/S
  f_bot_2 = Pe/Ac + Pe×e/S - MT/S

規範驗算：分別對照 Stage 1（f'ci）和 Stage 2（f'c）的容許壓/拉應力
```

**關鍵陷阱：**

| # | 陷阱 | 應對策略 |
|---|------|---------|
| 1 | **Stage 1 和 Stage 2 混用 P 或 f'c** | Stage 1 用 $P_i$、$f'_{ci}$；Stage 2 用 $P_e$、$f'_c$——兩組數據完全分開 |
| 2 | **忘記 Stage 1 只有自重（無活載重）** | "施加預力時"表示在此時刻，活載重尚未施加；$M = w_d L^2/8$ only |
| 3 | **偏心項符號搞錯**：e 在形心下方 | 偏心在下方 → 底纖維受壓（+）、頂纖維受拉（−）；再加 M 對頂面是壓、底面是拉 |
| 4 | **S 數值計算錯誤**：直接用題目誤植的 $S$ | 須由 $I_c/c$ 計算 $S$，驗算與 $r^2$ 一致性 |
| 5 | **容許應力用錯**：Stage 1 拿 $f'_c$ 當基準 | Stage 1 用 $f'_{ci} = 300$；Stage 2 用 $f'_c = 420$ |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 前置：斷面與彎矩計算

$$S = \frac{I_c}{c_1} = \frac{4.69\times10^5}{30} = \boxed{15{,}633\ \text{cm}^3}$$

驗算：$r^2 = I_c/A_c = 469{,}000/1100 = 426.4 \approx 426\ \text{cm}^2\ ✓$

$$M_d = \frac{w_d\,L^2}{8} = \frac{0.264\times144}{8} = \boxed{4.752\ \text{tf·m} = 475{,}200\ \text{kgf·cm}}$$

$$M_T = \frac{(w_d + w_l)\,L^2}{8} = \frac{(0.264+1.100)\times144}{8} = \frac{1.364\times144}{8} = \boxed{24.552\ \text{tf·m} = 2{,}455{,}200\ \text{kgf·cm}}$$

---

### 應力公式說明

對稱簡支預力梁，以**壓應力為正**，鋼腱偏心 $e$ 在形心**下方**，中點彎矩為正（下緣受拉）：

$$f_{\text{頂}} = \frac{P}{A_c} - \frac{P\,e}{S} + \frac{M}{S}$$

$$f_{\text{底}} = \frac{P}{A_c} + \frac{P\,e}{S} - \frac{M}{S}$$

---

### Stage 1：施加預力時（$P = P_i = 70\ \text{tf}$，$M = M_d$）

$$\frac{P_i}{A_c} = \frac{70{,}000}{1100} = 63.64\ \text{kgf/cm}^2$$

$$\frac{P_i\,e}{S} = \frac{70{,}000\times19.8}{15{,}633} = \frac{1{,}386{,}000}{15{,}633} = 88.66\ \text{kgf/cm}^2$$

$$\frac{M_d}{S} = \frac{475{,}200}{15{,}633} = 30.40\ \text{kgf/cm}^2$$

$$\boxed{f_{\text{頂},1} = 63.64 - 88.66 + 30.40 = +5.38\ \text{kgf/cm}^2 \quad \text{（壓C）}}$$

$$\boxed{f_{\text{底},1} = 63.64 + 88.66 - 30.40 = +121.90\ \text{kgf/cm}^2 \quad \text{（壓C）}}$$

---

### Stage 2：服務載重後（$P = P_e = 60\ \text{tf}$，$M = M_T$）

$$\frac{P_e}{A_c} = \frac{60{,}000}{1100} = 54.55\ \text{kgf/cm}^2$$

$$\frac{P_e\,e}{S} = \frac{60{,}000\times19.8}{15{,}633} = \frac{1{,}188{,}000}{15{,}633} = 76.00\ \text{kgf/cm}^2$$

$$\frac{M_T}{S} = \frac{2{,}455{,}200}{15{,}633} = 157.05\ \text{kgf/cm}^2$$

$$\boxed{f_{\text{頂},2} = 54.55 - 76.00 + 157.05 = +135.60\ \text{kgf/cm}^2 \quad \text{（壓C）}}$$

$$\boxed{f_{\text{底},2} = 54.55 + 76.00 - 157.05 = -26.50\ \text{kgf/cm}^2 \quad \text{（拉T）}}$$

---

### 兩階段彙總表

| 纖維位置 | Stage 1 ($P_i$, $M_d$) | Stage 2 ($P_e$, $M_T$) |
|---------|:--------------------:|:--------------------:|
| **頂纖維** | $+5.38\ \text{kgf/cm}^2$（C） | $+135.60\ \text{kgf/cm}^2$（C） |
| **底纖維** | $+121.90\ \text{kgf/cm}^2$（C） | $-26.50\ \text{kgf/cm}^2$（**T**） |

（C = 壓；T = 拉）

---

### 規範容許應力驗算

#### Stage 1（$f'_{ci} = 300\ \text{kgf/cm}^2$）

| 限制類型 | 計算限值 | 頂纖維 | 底纖維 | 判定 |
|---------|:--------:|:------:|:------:|:----:|
| 壓應力 ≤ $0.60\,f'_{ci}$ | $0.60\times300 = 180\ \text{kgf/cm}^2$ | $5.38 \leq 180$ | $121.90 \leq 180$ | ✅ |
| 拉應力 ≤ $0.25\sqrt{f'_{ci}}$ | $0.25\times\sqrt{300} = 4.33\ \text{kgf/cm}^2$ | 無拉應力 | 無拉應力 | ✅ |

$$\sqrt{f'_{ci}} = \sqrt{300} = 17.32\ \text{kgf/cm}^2$$

**→ Stage 1 兩纖維均符合規範 ✅**

---

#### Stage 2（$f'_c = 420\ \text{kgf/cm}^2$）

$$\sqrt{f'_c} = \sqrt{420} = 20.49\ \text{kgf/cm}^2$$

| 限制類型 | 計算限值 | 頂纖維 | 底纖維 | 判定 |
|---------|:--------:|:------:|:------:|:----:|
| 壓應力 ≤ $0.45\,f'_c$（持續載重） | $0.45\times420 = 189\ \text{kgf/cm}^2$ | $135.60 \leq 189$ | — | ✅ |
| 拉應力 ≤ $0.5\sqrt{f'_c}$（無黏結筋） | $0.5\times20.49 = 10.25\ \text{kgf/cm}^2$ | — | $26.50 > 10.25$ | ❌ |
| 拉應力 ≤ $1.0\sqrt{f'_c}$（有黏結筋） | $1.0\times20.49 = 20.49\ \text{kgf/cm}^2$ | — | $26.50 > 20.49$ | ❌ |

$$\boxed{\text{Stage 2 底纖維拉應力 } 26.50\ \text{kgf/cm}^2 > 1.0\sqrt{f'_c} = 20.49\ \text{kgf/cm}^2 \Rightarrow \textbf{不符合規範} ❌}$$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① Stage 2 底纖維不符合規範的工程分析**

底纖維拉應力不足（$-26.50\ \text{kgf/cm}^2$）的根本原因是：

$$\frac{M_T}{S} = 157.05 \gg \frac{P_e}{A_c} + \frac{P_e\,e}{S} = 54.55 + 76.00 = 130.55\ \text{kgf/cm}^2$$

外力彎矩項 $(M_T/S)$ 超過預力總效果 $(P_e/A_c + P_e\,e/S)$ 約 **26.5 kgf/cm²**，說明預力量或偏心距不足以抵抗服務載重。

**補救方向（工程設計角度）：**

| 方法 | 效果 | 備註 |
|------|------|------|
| 增加有效預力 $P_e$ | 提升 $P_e/A_c$ 和 $P_e\,e/S$ | 受鋼腱許可應力限制 |
| 增加偏心距 $e$ | 提升 $P_e\,e/S$ | 受斷面幾何限制（$e < c_2$） |
| 增加斷面尺寸（增大 $S$） | 降低 $M_T/S$ | 自重也增加 |
| 提高混凝土強度 $f'_c$ | 放寬容許拉應力 | $0.5\sqrt{f'_c}$ 隨 $f'_c$ 增加 |
| 配置黏結輔助鋼筋 | 允許 $1.0\sqrt{f'_c}$ 的拉應力 | 本題仍不足（26.5 > 20.49） |

**② Stage 1 頂纖維低壓應力（5.38 kgf/cm²）的意涵**

Stage 1 頂纖維僅 5.38 kgf/cm² 壓應力，說明偏心預力對頂部的「反彎矩」效果（88.66 kgf/cm² 拉）幾乎抵銷了均布預壓（63.64 kgf/cm² 壓），再由自重彎矩（30.40 kgf/cm² 壓）拉回正值。

這是典型的「**大偏心預力設計**」—— 偏心率 $e/c = 19.8/30 = 0.66$，幾乎接近容許最大偏心。

**③ 兩階段預力損失的影響**

預力從 $P_i = 70\ \text{tf}$ 損失至 $P_e = 60\ \text{tf}$（損失率 14.3%），應力影響：

$$\Delta(P/A_c) = (63.64 - 54.55) = 9.09\ \text{kgf/cm}^2 \quad \Delta(Pe/S) = (88.66 - 76.00) = 12.66\ \text{kgf/cm}^2$$

底纖維因預力損失，壓應力貢獻減少 $9.09 + 12.66 = 21.75\ \text{kgf/cm}^2$。搭配活載重引入的大量拉應力（$M_l/S = 1.1\times18\times10^5/15{,}633 = 126.65\ \text{kgf/cm}^2$），底纖維從 Stage 1 的 +121.9 kgf/cm² 壓應力「翻轉」至 Stage 2 的 -26.5 kgf/cm² 拉應力，反差達 **148.4 kgf/cm²**。
