## 1. 原始題目重述 (Problem Restatement)

**規範依據：** 中國土木水利工程學會「混凝土工程設計規範與解說」（土木401-112）

一方形柱 $40\text{ cm} \times 40\text{ cm}$，承受靜載重 $P_D = 80\text{ tf}$ 及活載重 $P_L = 75\text{ tf}$，其下由一位於地表下 $1.5\text{ m}$ 處之正方形獨立基腳支持。

**基腳資料：**
- 尺寸：$B \times B = 3\text{ m} \times 3\text{ m}$
- 有效深度：$d = 50\text{ cm}$
- 雙向鋼筋量（各方向）：$A_s = 50\text{ cm}^2$
- $f_y = 4200\text{ kgf/cm}^2$（D25以上）

**材料與土壤：**
- $f'_c = 280\text{ kgf/cm}^2$（全卷通用）
- RC 單位重：$\gamma_{RC} = 2.4\text{ tf/m}^3$
- 土壤單位重：$\gamma_{soil} = 2\text{ tf/m}^3$
- 地基土壤承載力：$q_a = 25\text{ tf/m}^2$

**要求：** 檢核基腳承受彎矩與穿孔剪力之適當性。（25分）

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 獨立基腳是「倒置懸臂梁」——柱傳來的集中力透過基腳底板向四周懸挑，底部承受向上的均布土壓力。設計者需同時確認：
1. **彎矩強度**：底板在柱面處的懸臂彎矩 $\leq \phi M_n$
2. **穿孔剪力（雙向衝剪）**：柱四周臨界周面的剪力 $\leq \phi V_c$

**出題者意圖：**
- 測試能否正確區分「服務載重（驗地基）」vs.「因數載重（驗混凝土強度）」
- 測試穿孔剪力臨界周長 $b_o$ 的計算（距柱面 $d/2$ 處，四面形成封閉矩形）
- 測試在 $\beta_c = 1$（方形柱）時，三個 $V_c$ 公式中哪個控制

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 算因數化載重 $P_u$，得淨因數化土壓力 $q_u = P_u/A$
2. 彎矩：找懸臂長 $L$，算 $M_u$；算 $\phi M_n$（Whitney 應力塊）
3. 穿孔剪力：算 $b_o$，算 $V_u$；用 ACI/土木401 三公式取最小 $V_c$，乘 $\phi=0.75$

**關鍵陷阱：**

| # | 陷阱 | 正確做法 |
|---|------|---------|
| 1 | **弄混服務載重與因數載重**：地基承載力用服務載重（$P_D+P_L$），混凝土強度設計用因數載重（$1.2P_D+1.6P_L$） | 地基驗算用 $P=155\text{ tf}$；混凝土驗算用 $P_u=216\text{ tf}$ |
| 2 | **穿孔剪力臨界周長位置**：在距柱面 $d/2$ 處，不是在柱面 | $b_o = 4(c+d) = 4\times90 = 360\text{ cm}$ |
| 3 | **$q_u$ 的計算方式**：用淨因數化土壓力（僅用 $P_u$），不另加基腳自重因數化值 | $q_u = P_u/B^2 = 24\text{ tf/m}^2$（標準作法） |
| 4 | **穿孔剪力 $V_c$ 公式的單位**：土木401-112（SI制）係數為 0.33，若用 kgf/cm² 必須換算 | 建議統一換算至 SI 制計算 |

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標
`驗算 φMn ≥ Mu（彎矩）且 φVc ≥ Vu（穿孔剪力）`

### 本題關鍵公式（依計算順序）

> $\boxed{\cdot}$ = 需由前步驟推導，非題目直接給定的變數

$$\text{Step 1: } P_u = 1.2P_D + 1.6P_L \quad\Rightarrow\quad q_u = \frac{\boxed{P_u}}{B^2}$$

$$\text{Step 2: } M_u = q_u \cdot B \cdot \frac{L^2}{2}, \quad L = \frac{B-c}{2}$$

$$\text{Step 3: } a = \frac{A_s f_y}{0.85 f'_c B} \quad\Rightarrow\quad \phi M_n = \phi\, A_s f_y\!\left(d - \frac{\boxed{a}}{2}\right) \geq M_u$$

$$\text{Step 4: } b_o = 4(c+d) \quad\Rightarrow\quad V_u = \boxed{q_u}(B^2-(c+d)^2)$$

$$\text{Step 5: } V_c = 0.33\sqrt{f'_c[\text{MPa}]}\cdot b_o\cdot d \quad\Rightarrow\quad \phi V_c \geq \boxed{V_u}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 【前置】材料參數

$$f'_c = 280\text{ kgf/cm}^2 = 27.46\text{ MPa},\quad \beta_1 = 0.85$$
$$f_y = 4200\text{ kgf/cm}^2,\quad \phi_{flex}=0.9,\quad \phi_{shear}=0.75$$

---

### 【Step 1】因數化載重與淨因數化土壓力

$$P_u = 1.2P_D + 1.6P_L = 1.2(80) + 1.6(75) = 96 + 120 = \boxed{216\text{ tf}}$$

淨因數化上壓力（標準簡化法：忽略基腳自重與覆土重之相互抵消）：

$$q_u = \frac{P_u}{B^2} = \frac{216}{3\times3} = \boxed{24\text{ tf/m}^2}$$

> 📝 **策略註解：** 基腳自重與其上覆土的因數化重量，與其所佔面積上的因數化地基反力相抵銷，工程實務中常直接用 $P_u/A$ 計算淨上壓力，此為保守且合理的簡化。

---

### 【Step 2】地基承載力驗核（服務載重）

假設基腳厚度 $h_f \approx 60\text{ cm}$：

$$W_f = 3\times3\times0.60\times2.4 = 12.96\text{ tf}$$
$$W_{soil} = 3\times3\times(1.50-0.60)\times2.0 = 16.20\text{ tf}$$
$$q_{service} = \frac{P_D+P_L+W_f+W_{soil}}{B^2} = \frac{155+12.96+16.20}{9} = 20.46\text{ tf/m}^2 < q_a = 25\text{ tf/m}^2 \quad \checkmark$$

---

### 【Step 3】彎矩驗核（臨界截面：柱面）

**懸臂長度：**
$$L = \frac{B - c}{2} = \frac{300 - 40}{2} = 130\text{ cm} = 1.30\text{ m}$$

**因數化彎矩（柱面處懸臂）：**
$$M_u = q_u \times B \times \frac{L^2}{2} = 24 \times 3.0 \times \frac{1.30^2}{2} = 24 \times 3.0 \times 0.845 = \boxed{60.84\text{ tf·m}}$$

**Whitney 應力塊深度：**
$$a = \frac{A_s f_y}{0.85 f'_c b} = \frac{50 \times 4200}{0.85 \times 280 \times 300} = \frac{210{,}000}{71{,}400} = 2.94\text{ cm}$$

**延性確認（拉力應變）：**
$$c = \frac{a}{\beta_1} = \frac{2.94}{0.85} = 3.46\text{ cm}$$
$$\varepsilon_t = 0.003 \times \frac{d - c}{c} = 0.003 \times \frac{50 - 3.46}{3.46} = 0.003 \times 13.45 = 0.0404 \geq 0.005 \Rightarrow \phi = 0.9$$

**設計彎矩強度：**
$$M_n = A_s f_y \left(d - \frac{a}{2}\right) = 50 \times 4200 \times \left(50 - \frac{2.94}{2}\right) = 210{,}000 \times 48.53 = 10{,}191\text{ tf·cm}$$

$$\phi M_n = 0.9 \times 101.91 = \boxed{91.7\text{ tf·m}} \geq M_u = 60.84\text{ tf·m} \quad \checkmark$$

> 📝 **策略註解：** $a = 2.94\text{ cm}$ 僅佔有效深度 50 cm 的 5.9%，鋼筋量遠低於平衡鋼筋比，延性完全無虞。

---

### 【Step 4】穿孔剪力驗核（臨界截面：距柱面 $d/2$ 處）

**臨界周長（方形柱）：**
$$b_o = 4(c + d) = 4 \times (40 + 50) = 4 \times 90 = 360\text{ cm} = 3600\text{ mm}$$

**臨界周線內面積：**
$$(c+d)^2 = 90^2 = 8100\text{ cm}^2 = 0.81\text{ m}^2$$

**因數化穿孔剪力：**
$$V_u = q_u \times (B^2 - (c+d)^2) = 24 \times (9.00 - 0.81) = 24 \times 8.19 = \boxed{196.6\text{ tf}}$$

**計算 $\phi V_c$（土木401-112 三公式，SI 單位）：**

換算：$f'_c = 27.46\text{ MPa}$，$\sqrt{f'_c} = 5.24\text{ MPa}^{0.5}$，$b_o = 3600\text{ mm}$，$d = 500\text{ mm}$

| 公式 | 計算 | $V_c$ (tf) |
|------|------|:----------:|
| ① $0.17(1+2/\beta_c)\sqrt{f'_c}\cdot b_o\cdot d$，$\beta_c=1$ | $0.17\times3\times5.24\times3600\times500 = 4{,}810{,}000\text{ N}$ | **490.5** |
| ② $0.083(\alpha_s d/b_o+2)\sqrt{f'_c}\cdot b_o\cdot d$，$\alpha_s=40$ | $0.083\times7.56\times5.24\times1{,}800{,}000 = 5{,}914{,}000\text{ N}$ | **603.1** |
| ③ $0.33\sqrt{f'_c}\cdot b_o\cdot d$ | $0.33\times5.24\times3600\times500 = 3{,}113{,}000\text{ N}$ | **317.3** ← 控制 |

> 📝 **策略註解：** 方形柱（$\beta_c=1$）時公式①與③相同結構，公式③通常為控制條件。內柱（$\alpha_s=40$）的公式②通常不控制。

$$\phi V_c = 0.75 \times 317.3 = \boxed{238.0\text{ tf}} \geq V_u = 196.6\text{ tf} \quad \checkmark$$

---

### 【最小鋼筋量驗核】

$$\rho_{min} = \max\!\left(\frac{0.25\sqrt{f'_c}}{f_y},\ \frac{1.4}{f_y}\right) = \max\!\left(\frac{0.25\times16.73}{4200},\ \frac{1.4}{4200}\right) = \max(0.000996,\ 0.000333) = 0.000996$$

$$\rho = \frac{A_s}{b\cdot d} = \frac{50}{300\times50} = 0.00333 > 0.000996 \quad \checkmark$$

---

### 【結論】

| 驗核項目 | 需求 | 供給 | 結果 |
|---------|------|------|:----:|
| 彎矩 | $M_u = 60.84\text{ tf·m}$ | $\phi M_n = 91.7\text{ tf·m}$ | ✅ 合格（裕度 51%） |
| 穿孔剪力 | $V_u = 196.6\text{ tf}$ | $\phi V_c = 238.0\text{ tf}$ | ✅ 合格（裕度 21%） |
| 最小鋼筋比 | $\rho_{min} = 0.000996$ | $\rho = 0.00333$ | ✅ 合格 |

> **∴ 基腳彎矩與穿孔剪力均符合土木401-112設計規範要求，基腳設計適當。**

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**爭議1：基腳自重是否納入因數化設計載重？**
- 嚴格作法：$q_u$ 應考慮 $1.2(W_f + W_{soil})$ 的因數化，再扣除後得淨 $q_u$
- 工程簡化：$q_u = P_u/B^2$（本題採用）
- 考場建議：若題目不特別要求，採簡化法並加以說明即可

**爭議2：穿孔剪力公式中的 $\alpha_s$ 值**
- $\alpha_s = 40$：內柱（四面均有板）
- $\alpha_s = 30$：邊柱
- $\alpha_s = 20$：角柱
- 本題為柱下基腳（視同內柱），$\alpha_s = 40$

**進階思考：若穿孔剪力不足怎麼辦？**
- 方案①：增加有效深度 $d$（最有效）
- 方案②：加大基腳平面尺寸（降低 $q_u$）
- 方案③：設置剪力頭（shear head）或剪力柱（stud rail）
