# 考題編號：RC-2011-1

**主分類：** `RC-U1-2` RC 柱強度分析與設計  
**副分類：** 無  
**設計法：** 概念題（USD）  
**標籤：** `P-M互制圖` `求算步驟` `平衡點` `應變相容` `φ值分區` `純軸壓` `純彎` `壓力控制` `過渡區`

---

## 1. 原始題目重述 (Problem Restatement)

請簡要說明鋼筋混凝土柱斷面之 $P_u$-$M_u$ 關係曲線圖（P-M 互制圖）之求算步驟，其中 $P_u$ 及 $M_u$ 分別為柱斷面之**設計軸力強度**（$\phi P_n$）及**設計彎矩強度**（$\phi M_n$）。（10 分）

> 本題為純概念說明題，無附圖，無截圖需求。

---

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** P-M 互制圖是描述「柱斷面在各種偏心距下最大承載能力」的包絡線。任何 $(P_u, M_u)$ 落在曲線內部，表示斷面安全；落在外部表示斷面不足。

**出題者測驗能力：**
1. 是否理解應變相容（strain compatibility）與力平衡（force equilibrium）兩原則的聯合應用
2. 是否能正確識別三個關鍵特徵點：純壓點、平衡點、純彎點
3. 是否理解 $\phi$ 值如何隨 $\varepsilon_t$ 過渡（壓力控制 → 過渡區 → 拉力控制）

---

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**
1. 先確立座標系（縱軸 $\phi P_n$，橫軸 $\phi M_n$）
2. 求三個邊界特徵點（純壓、平衡、純彎）
3. 在特徵點之間取若干假設 $c$ 值，逐點計算
4. 對各點套用對應的 $\phi$ 值後描點連線

**關鍵陷阱：**
| 陷阱 | 說明 | 應對 |
|------|------|------|
| ❶ $\phi P_{n,\text{max}}$ 上限 | ACI 規定：純軸壓設計上限 = $0.80\phi P_{n0}$（矩形箍），非 $\phi P_{n0}$ | 記住 0.80（矩形）/ 0.85（螺旋）修正係數 |
| ❷ $c_b$ 公式單位 | $c_b = \frac{6120}{6120+f_y}\cdot d$（kgf/cm² 單位），SI 制為 $\frac{600}{600+f_y}\cdot d$ | 確認單位制再代公式 |
| ❸ 壓力鋼筋應力 | $c$ 大時，$A'_s$ 所在應變未必達 $\varepsilon_y$，需應變相容驗算 | $f'_s = \min(E_s\varepsilon'_s,\ f_y)$ |
| ❹ $\phi$ 值判斷 | $\varepsilon_t < \varepsilon_y$ → 壓力控制 $\phi=0.65$；$\varepsilon_t > 0.005$ → 拉力控制 $\phi=0.9$；中間線性插值 | 以 $\varepsilon_t$ 而非 $c/d$ 判斷 |

---

## 3.5 變數層次分析 (Variable Hierarchy Analysis)

### 最終目標

求出柱斷面在不同偏心距下對應的設計強度點集合 $(\phi M_n,\ \phi P_n)$，繪成封閉包絡曲線。

### 本題關鍵公式（依計算順序）

$$\text{Step 1（純壓）：} P_{n0} = 0.85f'_c(A_g - A_{st}) + f_y A_{st}$$

$$\text{Step 2（平衡中性軸）：} c_b = \frac{6120}{6120 + f_y}\cdot d \quad \Rightarrow \quad a_b = \beta_1 \cdot \boxed{c_b}$$

$$\text{Step 3（平衡點力）：} P_{nb} = 0.85f'_c \cdot \boxed{a_b} \cdot b + C_s - T_b$$

$$\text{Step 4（任意點應變）：} \varepsilon'_s = \varepsilon_{cu}\cdot\frac{c - d'}{c},\quad \varepsilon_t = \varepsilon_{cu}\cdot\frac{d - c}{c}$$

$$\text{Step 5（任意點力矩）：} M_n = \sum F_i \cdot y_i \quad \text{（各力對斷面形心取矩）}$$

$$\text{Step 6（φ值）：} \phi = \begin{cases}0.65 & \varepsilon_t \le \varepsilon_y \\ 0.65 + 0.25\dfrac{\varepsilon_t - \varepsilon_y}{0.005 - \varepsilon_y} & \varepsilon_y < \varepsilon_t < 0.005 \\ 0.90 & \varepsilon_t \ge 0.005\end{cases}$$

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

> 本題為概念說明，以下依步驟條列 P-M 互制圖完整求算流程。

---

### 步驟一：確立基本材料常數

$$\beta_1 = 0.85 - 0.05\cdot\frac{f'_c - 280}{70} \quad (f'_c > 280 \text{ kgf/cm}^2)$$

$$\varepsilon_y = \frac{f_y}{E_s} = \frac{f_y}{2.04\times10^6}$$

$$\varepsilon_{cu} = 0.003 \text{（ACI 規定混凝土極限壓應變）}$$

---

### 步驟二：求純軸壓點（Point A，e = 0）

$$P_{n0} = 0.85f'_c(A_g - A_{st}) + f_y A_{st}$$

ACI 設計上限（考慮最小偏心）：

$$\boxed{\phi P_{u,\text{max}} = 0.80\phi P_{n0}} \quad (\phi = 0.65 \text{，矩形箍筋柱})$$

此點座標：$(\phi M_n = 0,\ \phi P_n = \phi P_{u,\text{max}})$

---

### 步驟三：求平衡點（Point B，$\varepsilon_t = \varepsilon_y$）

**中性軸深度：**

$$c_b = \frac{\varepsilon_{cu}}{\varepsilon_{cu} + \varepsilon_y}\cdot d = \frac{6120}{6120 + f_y}\cdot d \quad \text{（kgf/cm² 單位）}$$

$$a_b = \beta_1 \cdot c_b$$

**各力：**

$$C_c = 0.85f'_c \cdot a_b \cdot b$$

$$\varepsilon'_s = \varepsilon_{cu}\cdot\frac{c_b - d'}{c_b} \quad \Rightarrow \quad f'_s = \min(E_s\varepsilon'_s,\ f_y)$$

$$C_s = A'_s(f'_s - 0.85f'_c),\quad T = A_s\cdot f_y$$

**標稱強度：**

$$P_{nb} = C_c + C_s - T$$

$$M_{nb} = C_c\!\left(\frac{h}{2} - \frac{a_b}{2}\right) + C_s\!\left(\frac{h}{2} - d'\right) + T\!\left(d - \frac{h}{2}\right)$$

**φ 值判斷：** $\varepsilon_t = \varepsilon_y$，屬壓力控制過渡邊界，$\phi = 0.65$（保守取）

$$\boxed{(\phi M_{nb},\ \phi P_{nb}) = (0.65 M_{nb},\ 0.65 P_{nb})}$$

---

### 步驟四：求純彎點（Point C，$P_n = 0$）

令 $P_n = 0$，以中性軸 $c$ 為未知數求解（同單筋/雙筋梁分析）：

$$0.85f'_c\cdot\beta_1 c\cdot b + A'_s(f'_s - 0.85f'_c) - A_s f_s = 0$$

其中 $f_s = \min(E_s\varepsilon_t, f_y)$，$f'_s = \min(E_s\varepsilon'_s, f_y)$，解出 $c$ 後：

$$M_{nc} = \text{各力對形心之矩}$$

此區段 $\varepsilon_t \gg 0.005$（通常），$\phi = 0.90$。

$$\boxed{(\phi M_{nc},\ \phi P_{nc}) = (0.90 M_{nc},\ 0)}$$

---

### 步驟五：求若干中間點（假設一系列 $c$ 值）

在 $0 < c < c_b$ 及 $c_b < c < h$ 範圍內各取 3–5 個 $c$ 值，對每個 $c$：

| 計算項目 | 公式 |
|----------|------|
| 壓力鋼筋應變 | $\varepsilon'_s = \varepsilon_{cu}(c - d')/c$ |
| 拉力鋼筋應變 | $\varepsilon_t = \varepsilon_{cu}(d - c)/c$ |
| 鋼筋應力（限制 $\le f_y$） | $f'_s = \min(E_s\varepsilon'_s, f_y)$，$f_s = \min(E_s\varepsilon_t, f_y)$ |
| 各力 | $C_c = 0.85f'_c\cdot\beta_1 c\cdot b$，$C_s = A'_s(f'_s-0.85f'_c)$，$T = A_s f_s$ |
| 標稱強度 | $P_n = C_c + C_s - T$，$M_n = \sum F_i y_i$（對形心取矩） |
| φ 值 | 依 $\varepsilon_t$ 線性插值（見 §3.5） |
| 設計強度 | $(\phi M_n,\ \phi P_n)$ |

---

### 步驟六：描點繪製互制圖

將所有計算點 $(\phi M_n,\ \phi P_n)$ 描在座標系上，平滑連線後形成封閉曲線，並標示：

- 曲線上方/右方 → 斷面**不足**（unsafe）
- 曲線內部 → 斷面**安全**（safe）
- 壓力控制區（$\varepsilon_t \le \varepsilon_y$）、過渡區、拉力控制區三段

$$\boxed{\text{P-M 互制圖繪製完成}}$$

---

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**① 對稱斷面 vs. 不對稱斷面的形心位置**
- 雙向對稱斷面（$A_s = A'_s$，$d + d' = h$）：塑性形心 = 幾何形心 = $h/2$
- 不對稱配筋：需由 $x_p = \sum(F_i \cdot y_i)/\sum F_i$ 計算塑性形心後再取矩

**② $\phi$ 值的折減邏輯**
- 拉力控制（$\varepsilon_t \ge 0.005$）：$\phi = 0.90$——斷面延性足，允許較高利用率
- 壓力控制（$\varepsilon_t \le \varepsilon_y$）：$\phi = 0.65$（矩形箍）——壓力破壞突然，需更保守
- 螺旋柱壓力控制 $\phi = 0.75$，因圍束效果較好
