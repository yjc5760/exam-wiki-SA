## 4. 完整論述 (Detailed Answer)

---

### 【第一部分】施加預力之目的

施加預力（Prestress）的主要目的：

**（一）控制混凝土拉應力，防止裂縫產生**

混凝土抗拉強度僅約為抗壓強度的 1/10，在彎矩作用下受拉區極易開裂。施加預壓力後，可令構件全截面（或大部分截面）處於受壓狀態，有效抑制或消除拉應力，達到防止裂縫的目的。

**（二）提升構件跨度能力與剛度**

由於預力避免了受拉裂縫，有效利用全截面面積傳遞力，構件剛度大幅提升，撓度減小，可大幅延長跨距（與普通 RC 相比，預力梁可達 2~3 倍跨距）。

**（三）充分利用高強度材料**

普通 RC 中高強鋼絲（$f_{pu} = 18{,}000\text{ kgf/cm}^2$）若不預張，裂縫控制要求使其應力僅能達到 $f_y$（約 4,000~5,000 kgf/cm²），遠低於其實際強度。施加預力後，可讓鋼絞線在工作狀態下維持高應力，充分發揮材料潛能。

---

### 【第二部分】預力損失原因與計算公式

預力損失依發生時序分為兩大類：

$$\text{預力損失} = \underbrace{\text{即時損失}}_{\text{（施工時即發生）}} + \underbrace{\text{時間相關損失}}_{\text{（長期效應）}}$$

---

#### ─ 先拉法（Pre-tensioning）預力損失 ─

先拉法：先張拉鋼腱（固定於台座）→ 澆置混凝土、養護 → 混凝土達強度後放張（Release）→ 鋼腱藉握裹力將預力傳入混凝土。

**先拉法無管道，故無摩擦損失；無主要錨固系統，端部滑移量小。**

---

**① 彈性縮短損失（Elastic Shortening, ES）**

| 項目 | 內容 |
|------|------|
| **原因** | 放張後混凝土受壓彈性縮短，鋼腱隨之縮短，預力降低 |
| **計算公式** | $$\Delta f_{pES} = n \cdot f_{cir} = \frac{E_s}{E_c}\cdot f_{cir}$$ |
| **符號說明** | $n = E_s/E_c$：模數比（通常 $E_s=2.04\times10^6$，$E_c=15{,}000\sqrt{f'_c}$）；$f_{cir}$：鋼腱重心處混凝土初始壓應力（$= P_i/A + P_i e^2/I - M_D e/I$） |
| **先拉法特點** | 放張時一次彈性縮短，所有鋼腱均受影響 |

---

**② 乾縮損失（Shrinkage, SH）**

| 項目 | 內容 |
|------|------|
| **原因** | 混凝土澆置後水分散逸，體積收縮，預力鋼腱縮短 |
| **計算公式（ACI 估算法）** | $$\Delta f_{pSH} = 8.2\times10^{-6} K_{sh} \cdot E_s\!\left(1-0.06\frac{V}{S}\right)(100 - RH)$$ |
| **符號說明** | $K_{sh}$：時間修正係數；$V/S$：體積/表面積比（cm）；$RH$：環境相對濕度（%）；台灣 $RH \approx 70\sim80\%$ |
| **特點** | 先拉法乾縮量較大（因放張前已有部分乾縮），後拉法則因混凝土較成熟，乾縮量略小 |

---

**③ 潛變損失（Creep, CR）**

| 項目 | 內容 |
|------|------|
| **原因** | 混凝土在持續壓力下因分子蠕動而發生時間相關應變，壓縮量持續增加，鋼腱應力降低 |
| **計算公式（ACI 估算法）** | $$\Delta f_{pCR} = K_{cr}\!\left(\frac{E_s}{E_c}\right)\!(f_{cir} - f_{cds})$$ |
| **符號說明** | $K_{cr}$：潛變係數（先拉法 $K_{cr}=2.0$，後拉法 $K_{cr}=1.6$）；$f_{cds}$：使用載重在鋼腱重心處產生的混凝土壓應力 |

---

**④ 鬆弛損失（Relaxation, RE）**

| 項目 | 內容 |
|------|------|
| **原因** | 高強鋼絞線在高應力、恆定應變下，因材料黏彈性性質導致應力隨時間降低（蠕變） |
| **計算公式** | $$\Delta f_{pR} = \left[K_{re} - J(\Delta f_{pSH}+\Delta f_{pCR}+\Delta f_{pES})\right]C$$ |
| **符號說明** | $K_{re}$：初始鬆弛係數（依鋼種查表）；$J$：折減係數；$C$：鬆弛修正係數（依 $f_{pi}/f_{pu}$ 比值查表） |
| **低鬆弛鋼腱** | 採低鬆弛（Low-relaxation）鋼絞線時，$K_{re}$ 可大幅降低（鬆弛損失僅約 1~2%） |

---

#### ─ 後拉法（Post-tensioning）預力損失 ─

後拉法：先澆置混凝土（含預留導管）→ 混凝土硬化 → 穿入鋼腱張拉 → 錨定 → 灌漿（若黏結型）。

後拉法除與先拉法共有的乾縮、潛變、鬆弛、彈性縮短損失外，**額外增加摩擦損失與錨固損失**。

---

**① 摩擦損失（Friction Loss, F）**【後拉法特有】

| 項目 | 內容 |
|------|------|
| **原因** | 後拉法鋼腱置於彎曲導管中；張拉時鋼腱與管壁之間因（a）**曲率效應**（Curvature Effect）及（b）**波紋效應**（Wobble Effect）產生摩擦，導致從千斤頂端到錨固端應力遞減 |
| **計算公式** | $$f_{px} = f_{pj}\cdot e^{-(\mu\alpha + kL)} \approx f_{pj}(1-\mu\alpha - kL)$$ （小量近似） |
| **符號說明** | $f_{pj}$：千斤頂端張拉應力；$\mu$：曲率摩擦係數（鋼-金屬管 $\mu=0.15\sim0.25$）；$\alpha$：導管偏折角（rad，$=$ 各段曲率角度和）；$k$：波紋係數（$/\text{m}$，$k=0.0030\sim0.0066$）；$L$：從千斤頂至計算點的距離（m） |
| **設計考量** | 可採雙端張拉減少損失；也可先張拉到較高值（Overstressing）再回拉 |

---

**② 錨固（端部滑移）損失（Anchorage Set Loss, A）**【後拉法較顯著】

| 項目 | 內容 |
|------|------|
| **原因** | 鋼腱張拉完成後夾片鎖緊（Wedge Seating）時，鋼腱有微小回縮（ $\Delta_a$ = 通常 3~6 mm），造成預力降低 |
| **計算公式** | $$\Delta f_{pA} = \frac{\Delta_a}{L_{pA}} \cdot E_s$$ |
| **符號說明** | $\Delta_a$：滑移量（mm）；$L_{pA}$：受影響長度（對應滑移影響範圍，m）；$E_s = 2.0\times10^5\text{ MPa}$（鋼絞線） |
| **特點** | 影響範圍隨距千斤頂距離增加而縮小（超過某一長度後，摩擦抵抗了回縮影響） |

---

**③ 彈性縮短損失（後拉法）**

| 項目 | 內容 |
|------|------|
| **特點** | 若僅一根鋼腱：張拉時混凝土已存在，張拉完即鎖固，**無彈性縮短損失**；若有多根鋼腱**依序**張拉：後張拉的鋼腱引起彈性縮短，使先張拉的鋼腱損失預力 |
| **計算公式（n根鋼腱依序張拉）** | $$\Delta f_{pES} = \frac{n-1}{2n}\cdot\frac{E_s}{E_c}\cdot f_{cir}$$ （平均值法） |

---

**④⑤⑥ 潛變、乾縮、鬆弛損失（後拉法）**

公式結構與先拉法相同（參見上方），但係數略有不同：

| 損失 | 先拉法係數 | 後拉法係數 |
|------|:----------:|:----------:|
| 潛變 $K_{cr}$ | 2.0 | 1.6（因混凝土較成熟） |
| 乾縮 | 較大（放張前已乾縮） | 較小 |
| 鬆弛 | 相同（取決於鋼種） | 相同 |

---

### 【第三部分】規範建議之 Lump-Sum 損失總量

依 ACI 318 及土木401-100 規範，對設計良好之預力梁（正常環境、無過大偏心），規範建議的估算值：

$$\boxed{\text{最終有效預力} \approx \text{初始預力} \times (80\sim85\%)}$$

即：**預力損失總量約為初始預力的 $15\sim20\%$**（lump-sum）

| 工法 | 損失估計（lump-sum） |
|------|:--------------------|
| 先拉法（Pre-tensioning） | 總損失 ≈ **15~20%**（主要為彈性縮短 + 乾縮 + 潛變 + 鬆弛） |
| 後拉法（Post-tensioning） | 總損失 ≈ **15~20%**（主要為摩擦 + 錨固 + 乾縮 + 潛變 + 鬆弛） |

> 📝 **記憶口訣：「八成留下」** — 良好設計的預力梁，長期有效預力應至少為初始預力的 **80%**；若低於此值，代表設計不佳或材料問題。

**個別損失量參考（典型值）：**

| 損失種類 | 先拉法（%） | 後拉法（%） |
|---------|:----------:|:----------:|
| 摩擦損失 | — | 5~10% |
| 錨固滑移 | 1~2% | 1~3% |
| 彈性縮短 | 3~5% | 1~3% |
| 乾縮 | 2~4% | 2~3% |
| 潛變 | 5~8% | 4~7% |
| 鬆弛 | 2~4% | 2~4% |
| **合計** | **13~23%** | **15~30%** |

> 設計良好且使用低鬆弛鋼絞線，通常控制在 15~20% 範圍內。

## 附錄：關鍵公式速查表

$$\text{摩擦損失：} \quad \Delta f_{pF} = f_{pj}\!\left(1 - e^{-(\mu\alpha+kL)}\right)$$

$$\text{彈性縮短（先拉）：} \quad \Delta f_{pES} = \frac{E_s}{E_c}f_{cir}$$

$$\text{乾縮（ACI）：} \quad \Delta f_{pSH} = 8.2\times10^{-6} K_{sh} E_s\!\left(1-0.06\frac{V}{S}\right)(100-RH)$$

$$\text{潛變（ACI）：} \quad \Delta f_{pCR} = K_{cr}\frac{E_s}{E_c}(f_{cir}-f_{cds})$$

$$\text{錨固損失：} \quad \Delta f_{pA} = \frac{\Delta_a}{L_{pA}}E_s$$

$$\text{鬆弛（查表）：} \quad \Delta f_{pR} = \left[K_{re} - J(\Delta f_{pSH}+\Delta f_{pCR}+\Delta f_{pES})\right]C$$

$$\boxed{\text{總損失} \approx 15\sim20\% \text{ of } f_{pi} \quad\Rightarrow\quad f_{pe} \approx 0.80\sim0.85\, f_{pi}}$$
