## 1. 原始題目重述 (Problem Restatement)

一鋼筋混凝土簡支矩形梁，斷面條件如下：

| 參數 | 數值 |
|------|------|
| 梁寬 $b$ | 40 cm |
| 梁深 $h$ | 60 cm |
| 有效深度 $d$ | 53 cm |
| 拉力鋼筋 | 4 根 D25（$A_s = 4 \times 5.07 = 20.28 \text{ cm}^2$） |
| 箍筋 | D13@15 cm |

試求混凝土強度分別為 $f'_c = 210 \text{ kgf/cm}^2$ 及 $f'_c = 280 \text{ kgf/cm}^2$ 時，斷面的**延展比**（曲率延展比 $\mu_\phi$）及兩者比值。（25 分）

> **假設：** 題目未明示 $f_y$，依 SD420 鋼筋標準值取 $f_y = 4200 \text{ kgf/cm}^2$，$E_s = 2{,}040{,}000 \text{ kgf/cm}^2$，$\varepsilon_{cu} = 0.003$（依土木401-110）。

## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：**「**延性**」是結構在超過彈性範圍後仍能持續變形、消散能量的能力，是耐震設計的靈魂。曲率延展比 $\mu_\phi = \phi_u / \phi_y$ 是衡量斷面層次延性的標準指標。

**出題者要測驗的能力：**
1. 能正確定義並計算**極限曲率** $\phi_u$（Whitney 應力塊）
2. 能正確計算**降伏曲率** $\phi_y$（彈性轉換斷面法）
3. 理解 $f'_c$ 對延性的影響：更高強度的混凝土 → 更小的 $c_u$ → 更高的 $\phi_u$ → 更大的 $\mu_\phi$

**常見陷阱：** 考生容易對兩種狀態（降伏 vs. 極限）套用同一種分析方法，導致公式錯誤。

## 3. 解題戰略地圖與陷阱分析 (Strategic Roadmap & Trap Analysis)

**作戰計畫：**

```
Step 1：查材料常數（β₁、Ec、n、εy）
     ↓
Step 2：彈性轉換斷面法 → 求降伏中性軸比 ky → 求 φy
     ↓
Step 3：Whitney 應力塊 → 求極限中性軸深度 cu → 求 φu
     ↓
Step 4：μφ = φu / φy → 分別計算兩個 f'c → 求比值
```

**關鍵陷阱：**

| 陷阱 | 說明 | 應對策略 |
|------|------|---------|
| ⚠ 降伏曲率用 Whitney 應力塊 | **錯誤**！Whitney 應力塊是極限狀態假設；降伏時混凝土仍近似線彈性 | 降伏曲率務必用彈性轉換斷面法求 $k_y$ |
| ⚠ 混淆 $k_y d$ 與 $c_u$ | 兩者物理意義不同：前者是彈性降伏時的中性軸；後者是極限破壞時的中性軸 | 分清兩種分析模型 |
| ⚠ f'c = 280 的 β₁ | 土木401-110：$f'_c \leq 280 \Rightarrow \beta_1 = 0.85$（280 是臨界值，仍取 0.85）| 不要誤算 β₁ 折減 |
| ⚠ 忽略 fy 假設 | 題目未給 fy，需明確說明假設 SD420 | 答題開頭寫「假設 fy = 4200 kgf/cm²」 |

## 3.5 變數層次分析（Variable Hierarchy Analysis）

> 複習提示：第一次解題後，在每個卡住的知識點旁標記 `⚠`；第二次複習時只看有 `⚠` 的項目。

### 最終目標

`求兩個 f'c 下的曲率延展比 μφ = φu / φy，及比值 μ(280)/μ(210)`

### 本題關鍵公式（依計算順序）

$$\text{Step 1: } k_y = -n\rho + \sqrt{(n\rho)^2 + 2n\rho}$$

$$\text{Step 2: } \phi_y = \frac{\varepsilon_y}{d(1 - \boxed{k_y})}$$

$$\text{Step 3: } c_u = \frac{a}{\beta_1} = \frac{A_s f_y}{0.85 f'_c b \cdot \beta_1}$$

$$\text{Step 4: } \phi_u = \frac{\varepsilon_{cu}}{\boxed{c_u}}$$

$$\text{Step 5: } \mu_\phi = \frac{\boxed{\phi_u}}{\boxed{\phi_y}}$$

### L1：題目直接給定

_看到題目就能讀出的數字，不需要任何公式。_

| 符號 | 數值 | 說明 |
|------|------|------|
| $b$ | 40 cm | 梁寬 |
| $d$ | 53 cm | 有效深度 |
| $A_s$ | 20.28 cm² | 4×D25，查表 $4 \times 5.07$ |
| $f'_c$ | 210 / 280 kgf/cm² | 兩種情況分別計算 |
| $\varepsilon_{cu}$ | 0.003 | 規範常數（土木401-110） |
| $f_y$ | 4200 kgf/cm² | **假設** SD420 標準值 |
| $E_s$ | 2,040,000 kgf/cm² | 規範常數 |

### L2：需知識點推導

_需要知道公式名稱與適用條件，套入 L1 即可算出。_

**Step 1：材料常數**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|
| $\beta_1$ | $f'_c \leq 280 \Rightarrow 0.85$；每超 70 減 0.05，下限 0.65 | |
| $E_c$ | $15{,}000\sqrt{f'_c}$（kgf/cm²）| |
| $n$ | $E_s / E_c$ | |
| $\varepsilon_y$ | $f_y / E_s$ | |

**Step 2：彈性轉換斷面求降伏中性軸**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|
| $\rho$ | $A_s / (b \cdot d)$ | |
| $k_y$ | $-n\rho + \sqrt{(n\rho)^2 + 2n\rho}$ | |

**Step 3：Whitney 應力塊求極限中性軸**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|

**Step 4：延展比**

| 符號 | 公式／來源 | 卡關? |
|------|-----------|:-----:|

### L3：深層知識（不懂就卡住）

| 知識點 | 說明 | 卡關? |
|--------|------|:-----:|
| 為何降伏曲率用彈性分析？ | 鋼筋剛降伏時，混凝土仍在線彈性範圍，Whitney 矩形應力塊假設只適用於極限狀態 | |
| 轉換斷面法的核心假設 | 平截面假設 + 混凝土與鋼筋彈性行為 → $n \cdot A_s$ 等效為混凝土面積 | |
| $k_y$ 公式推導邏輯 | 令彈性中性軸以上混凝土壓力 = 以下等效鋼筋拉力，解二次方程 → $k_y$ 公式 | |
| 為何 $f'_c$ 越高延展比越大？ | $f'_c$ 越高 → $a = A_s f_y / (0.85 f'_c b)$ 越小 → $c_u$ 越小 → $\phi_u = \varepsilon_{cu}/c_u$ 越大；而 $\phi_y$ 變化有限 → $\mu_\phi$ 提升 | |
| β₁ 的分界線 | 土木401-110：$f'_c \leq 280$ kgf/cm² → $\beta_1 = 0.85$（280 kgf/cm² 本身不折減） | |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 共用條件

$$A_s = 4 \times 5.07 = 20.28 \text{ cm}^2,\quad b = 40 \text{ cm},\quad d = 53 \text{ cm}$$

$$\varepsilon_{cu} = 0.003,\quad f_y = 4200 \text{ kgf/cm}^2,\quad E_s = 2{,}040{,}000 \text{ kgf/cm}^2$$

$$\varepsilon_y = \frac{f_y}{E_s} = \frac{4200}{2{,}040{,}000} = 2.059 \times 10^{-3}$$

$$\rho = \frac{A_s}{b \cdot d} = \frac{20.28}{40 \times 53} = \frac{20.28}{2120} = 9.566 \times 10^{-3}$$

---

### Case 1：$f'_c = 210 \text{ kgf/cm}^2$

**▸ 材料常數**

$$\beta_1 = 0.85 \quad (f'_c \leq 280)$$

$$E_c = 15{,}000\sqrt{210} = 15{,}000 \times 14.49 = 217{,}371 \text{ kgf/cm}^2$$

$$n = \frac{E_s}{E_c} = \frac{2{,}040{,}000}{217{,}371} = 9.38$$

**▸ 降伏曲率 $\phi_y$（彈性轉換斷面法）**

$$n\rho = 9.38 \times 9.566 \times 10^{-3} = 0.08973$$

$$k_y = -n\rho + \sqrt{(n\rho)^2 + 2n\rho} = -0.08973 + \sqrt{0.08973^2 + 2 \times 0.08973}$$

$$= -0.08973 + \sqrt{0.00805 + 0.17946} = -0.08973 + \sqrt{0.18751} = -0.08973 + 0.4330 = 0.3433$$

$$\phi_y^{(210)} = \frac{\varepsilon_y}{d(1 - k_y)} = \frac{2.059 \times 10^{-3}}{53 \times (1 - 0.3433)} = \frac{2.059 \times 10^{-3}}{53 \times 0.6567} = \frac{2.059 \times 10^{-3}}{34.81} = 5.916 \times 10^{-5} \text{ rad/cm}$$

**▸ 極限曲率 $\phi_u$（Whitney 應力塊）**

$$a = \frac{A_s f_y}{0.85 f'_c b} = \frac{20.28 \times 4200}{0.85 \times 210 \times 40} = \frac{85{,}176}{7{,}140} = 11.93 \text{ cm}$$

$$c_u = \frac{a}{\beta_1} = \frac{11.93}{0.85} = 14.03 \text{ cm}$$

$$\phi_u^{(210)} = \frac{\varepsilon_{cu}}{c_u} = \frac{0.003}{14.03} = 2.138 \times 10^{-4} \text{ rad/cm}$$

**▸ 延展比**

$$\boxed{\mu_{\phi,1} = \frac{\phi_u^{(210)}}{\phi_y^{(210)}} = \frac{2.138 \times 10^{-4}}{5.916 \times 10^{-5}} = 3.61}$$

---

### Case 2：$f'_c = 280 \text{ kgf/cm}^2$

**▸ 材料常數**

$$\beta_1 = 0.85 \quad (f'_c = 280 \text{ kgf/cm}^2\text{，臨界值，仍取 } 0.85)$$

$$E_c = 15{,}000\sqrt{280} = 15{,}000 \times 16.73 = 250{,}998 \text{ kgf/cm}^2$$

$$n = \frac{2{,}040{,}000}{250{,}998} = 8.13$$

**▸ 降伏曲率 $\phi_y$**

$$n\rho = 8.13 \times 9.566 \times 10^{-3} = 0.07775$$

$$k_y = -0.07775 + \sqrt{0.07775^2 + 2 \times 0.07775} = -0.07775 + \sqrt{0.00605 + 0.15550} = -0.07775 + \sqrt{0.16155}$$

$$= -0.07775 + 0.4019 = 0.3242$$

$$\phi_y^{(280)} = \frac{2.059 \times 10^{-3}}{53 \times (1 - 0.3242)} = \frac{2.059 \times 10^{-3}}{53 \times 0.6758} = \frac{2.059 \times 10^{-3}}{35.82} = 5.748 \times 10^{-5} \text{ rad/cm}$$

**▸ 極限曲率 $\phi_u$**

$$a = \frac{85{,}176}{0.85 \times 280 \times 40} = \frac{85{,}176}{9{,}520} = 8.947 \text{ cm}$$

$$c_u = \frac{8.947}{0.85} = 10.53 \text{ cm}$$

$$\phi_u^{(280)} = \frac{0.003}{10.53} = 2.850 \times 10^{-4} \text{ rad/cm}$$

**▸ 延展比**

$$\boxed{\mu_{\phi,2} = \frac{\phi_u^{(280)}}{\phi_y^{(280)}} = \frac{2.850 \times 10^{-4}}{5.748 \times 10^{-5}} = 4.96}$$

---

### 兩者比值

$$\boxed{\frac{\mu_{\phi,2}}{\mu_{\phi,1}} = \frac{4.96}{3.61} = 1.37}$$

---

### 彙整表

| 項目 | $f'_c = 210$ | $f'_c = 280$ |
|------|:---:|:---:|
| $\beta_1$ | 0.85 | 0.85 |
| $n$ | 9.38 | 8.13 |
| $k_y$ | 0.343 | 0.324 |
| $\phi_y$ (rad/cm) | $5.92 \times 10^{-5}$ | $5.75 \times 10^{-5}$ |
| $a$ (cm) | 11.93 | 8.95 |

| $\phi_u$ (rad/cm) | $2.14 \times 10^{-4}$ | $2.85 \times 10^{-4}$ |

**比值：** $\mu_{\phi}(280) / \mu_{\phi}(210) = \mathbf{1.37}$

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

**1. $f_y$ 的假設問題**

本題原文未明示 $f_y$，考場上應在答題開頭寫「假設 $f_y = 4200 \text{ kgf/cm}^2$（SD420）」。因為 $f_y$ 同時影響 $\varepsilon_y$ 和 $a$，若假設不同 $f_y$ 答案會變，但比值趨勢不變。

**2. 為何高強度混凝土有更高延展性？**

從公式可見：
- $\phi_u \propto 1/c_u \propto f'_c$（$f'_c$ 越高 → $a$ 越小 → $c_u$ 越小 → $\phi_u$ 越大）
- $\phi_y$ 受 $n = E_s/E_c \propto 1/\sqrt{f'_c}$ 影響，但變化幅度遠小於 $\phi_u$ 的變化

因此提高 $f'_c$ 可以有效提升斷面的曲率延展比——這也是高強度混凝土在耐震設計中被允許使用的理論依據之一。

**3. 位移延展比 vs. 曲率延展比**

本題求的是**曲率延展比**（斷面層次）；實際結構耐震設計更常用**位移延展比**（整體層次），兩者的關係取決於構架的塑鉸分布型態，是更進階的議題。

**4. 考場最安全作法**

- 寫清楚哪個狀態用哪個方法（降伏 → 彈性分析；極限 → Whitney 應力塊）
- $k_y$ 公式要記住，不能與 $a/d$、$c_u/d$ 混用
