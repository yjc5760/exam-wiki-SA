④ 核心公式速查

傾角變位基本式（順時針為正）

$$M_{AB} = \frac{2EI}{L}\left(2\theta_A + \theta_B - 3\psi\right) + FEM_{AB},\qquad \psi = \frac{\Delta}{L}$$

遠端為鉸／滾支承之修正式：$$M_{AB} = \frac{3EI}{L}(\theta_A - \psi) + FEM'_{AB}$$（該端方程可省略）

⚠ 修正式之固端彎矩需用「一端固定一端鉸」值：均佈 wL²/8、跨中集中 3PL/16。

常用固端彎矩（兩端固定）

$$\text{均佈 } w:\ FEM = \mp\frac{wL^2}{12} \qquad \text{跨中集中 } P:\ FEM = \mp\frac{PL}{8}$$

$$\text{任意位置 } P:\ FEM_{AB} = -\frac{Pab^2}{L^2},\quad FEM_{BA} = +\frac{Pa^2b}{L^2}$$

支承沉陷 δ：以 \(\psi = \delta/L\) 帶入基本式（不另列 FEM）。

⚠ 符號慣例：對桿端順時針為正；左端常為負、右端為正，抄表最易錯。

剪力（側移）方程式

每一獨立側移自由度取水平剪力平衡：$$\sum_{\text{柱}} V_{\text{柱}} = \sum_{\text{柱}} \frac{M_{\text{上}}+M_{\text{下}}}{h} = \sum P_{\text{層}}$$

斜桿或幾何複雜時，改用虛功法建立側移方程：\(\sum M_i\,\psi_i^{virt} = \sum P\,\delta^{virt}\)。

⚠ 斜桿各桿 ψ 不同：以瞬心法或虛位移幾何求 ψ 間關係（SA-2014-3）。

非均勻斷面：勁度與傳遞係數

一般化桿端彎矩：$$M_{AB} = K_A(\theta_A) + C_{BA}K_B(\theta_B) + \cdots$$

以共軛梁（單位轉角法）自行推導：勁度 \(K = M/\theta\)、傳遞係數 \(C = M_{\text{遠}}/M_{\text{近}}\)。

均勻斷面特例：\(K = 4EI/L,\ C = 1/2\)。

⚠ 變斷面、剛域、剛性構件皆不可直接套 4EI/L（SA-2025-3、SA-2006-4）。

⑥ 高頻陷阱由考古題解析歸納

1. 固端彎矩符號抄錯

傾角變位法以「順時針為正」：均佈載重左端 −wL²/12、右端 +wL²/12。與彎矩分配法表格混用最容易錯號。 （SA-2017-3、SA-2022-2）

2. 遠端鉸支未用修正式

遠端為鉸／滾支承時用 3EI/L 修正式可少一個未知數；硬用基本式須補該端 M=0 方程，兩者擇一，混用必錯。 （SA-2021-1）

3. 獨立側移數目判定錯誤

忽略軸向變形下，每層通常一個側移；但垂直滾支承、斜桿、剛性構件會改變側移模式，先畫「可能的剛體位移圖」再數。 （SA-2020-1）

4. 斜桿 ψ 關係式算錯

斜桿側移時各桿旋轉角不同：以垂直桿軸之位移分量計 ψ=Δ⊥/L，用瞬心或虛功幾何求關係。 （SA-2014-3）

5. 剪力方程式漏項

層剪力=該層以上所有水平外力和；柱剪力 V=(M上+M下)/h 注意符號方向，有斜桿時軸力的水平分量也要進方程。 （SA-2005-2、SA-2024-4）

6. 對稱分解邊界條件用錯

對稱載重：對稱軸上 θ=0、可有垂直位移（滑動支承）；反對稱載重：軸上 M=0（滾支承）。切錯邊界整題重來。 （SA-2020-2、SA-2021-2、SA-2024-3）

7. 彈簧支承與沉陷處理

彈簧力 F=kΔ 為未知位移之函數，需與傾角變位方程聯立；已知沉陷則以 ψ=δ/L 直接帶入。 （SA-2021-1）

8. 非均勻斷面直接套 4EI/L

變斷面桿之勁度、傳遞係數、固端彎矩都要用共軛梁重新推導；剛性構件段 EI→∞ 不變形但會傳遞剛體運動。 （SA-2025-3、SA-2006-4）

⑦ 互動計算器

兩跨連續梁傾角變位法計算器

A—B—C 兩跨連續梁（A、C 簡支），各跨均佈載重。以修正式解 θB 並繪 BMD（EI 為常數，結果與 EI 無關）。
