# 變數層次分析（VHA）Skill

## 目的

鋼結構設計考試的本質是：**從題目給的線索，按正確順序找到所有公式中每個變數的值，代入求解**。

VHA 把這個過程拆解為三層，讓考生能精確回答「我是在哪個環節卡住的？」：

| 層次 | 定義 | 判斷標準 |
|------|------|---------|
| **L1** | 題目直接給的數字或條件 | 不看任何公式就能從題幹讀出 |
| **L2** | 需套公式推導的中間變數 | 知道公式名稱 + 套入 L1 即可算出 |
| **L3** | 深層概念知識 | 不懂這個概念就會選錯公式或犯典型錯誤 |

---

## 執行流程

### Step A：定位檔案

使用者會指定題號（如 SS-2024-1）。檔案路徑為：

```
raw/solutions/SS-YYYY-N/SS-YYYY-N.md
```

讀取該檔案的完整內容。

### Step B：分析三層變數

**從 §1（原始題目重述）提取 L1：**
- 所有題幹直接給出的數字（跨距、載重、尺寸、材料常數、係數）
- 題目明示的設計條件（「充分側向支撐」、「無加勁板」、「已知有效長細比」）
- 題目提供的設計圖表或參考公式（以條目名稱列入，不展開）

**從 §4（步驟化詳細計算）提取 L2，按計算步驟分組：**
- 每個計算步驟（Step/一/二/三...）為一組
- 每列：符號、公式來源（簡述）、卡關?欄（留空）
- 特別注意：若計算中有「策略註解」或步驟說明特別警示某個判斷，標記 `⚠ 常見卡關`

**從 §3（解題戰略地圖與陷阱分析）提取 L3：**
- 每一條陷阱（❶❷❸❹...）對應一個 L3 知識點
- L3 的「說明」欄要說明「不懂這點會怎麼算錯」
- 若 §2（考題核心精神）也有「出題者測驗重點」，補充為 L3
- **「補強頁」欄：** 查下方「L3 → 補強頁對照表」填入 `[[page-name]]`；
  優先填 `methods/`（怎麼算），其次填 `concepts/`（為什麼），
  若同時有兩頁則用 ` · ` 分隔；無對應頁面則留空

### Step C：決定最終目標（一句話）

讀取題目的子問題列表（§1 的結尾），用一句話概括解題的完整鏈條。

### Step D：生成 §3.5 區塊

## 層次判斷指引

判斷一個量屬於哪一層時，問自己：

> **「一個完全不懂這門課的人，拿到題目能找到這個數字嗎？」**
> - 能 → L1
> - 不能，但給我公式我能算 → L2
> - 給公式還不夠，我需要理解這個公式「什麼時候用」「用哪個版本」→ L3

### 常見 L3 知識點（鋼構考試通用）

下列知識點若在題目的陷阱分析中出現，一律歸入 L3：

- LRFD 載重組合係數（1.2D + 1.6L 的來源）
- 使用性 vs 強度設計：撓度用**未因數化**服務載重
- LTB 三段判斷邏輯（L_b 與 L_p、L_r 的比較）
- φ_c = 0.85（壓力）vs φ_b = 0.9（彎曲）的區別
- A_w 定義：I 形用全高 d；箱形只取平行剪力方向的兩片板
- H1-1a vs H1-1b 門檻（P_u/φcPn 與 0.2 比較）
- BOX 斷面 Z 公式（(b_o³ − b_i³)/4）與 I 形的差異
- 0.658^λ 指數計算（e^(x·ln0.658)）
- 撓度公式選擇：均佈 5wL⁴/384EI vs 三分點 23PL³/648EI vs 跨中 PL³/48EI
- 剪力遲滯係數 U 的查表條件
- 塊狀剪力撕裂（BSR）的三項極限狀態

## 輸出品質要求

- L1 表格：題目所有給定數字與條件，**不遺漏、不合併**（寧多勿少）
- L2 分組：與 §4 的步驟一一對應
- L3 條數：通常 4–8 條（太少表示沒有深入分析，太多表示把 L2 混進來了）
- L3「補強頁」欄：查對照表填入；找不到對應頁面時留空，不可捏造頁面名稱
- 所有符號使用 LaTeX 行內格式 `$...$`
- 「卡關?」欄全部留空，不要替考生預填

## 主要公式框的生成規則

「主要公式框」是 §3.5 的第一個視覺元素，目的是讓考生在看 L1/L2 表格之前，先看到「這題從頭到尾需要代入哪些公式、哪些變數是未知的」。

### 什麼要放進公式框？

只放**計算鏈條上的核心公式**，不放定義式或幾何推導細節。通常 3–7 條公式：

- 載重換算（如 LRFD 組合 $P_u = 1.2P_D + 1.6P_L$）
- 強度計算（如 $\phi_bM_p = 0.9 F_y Z_x$）
- 最終檢核（如互制式 $P_u/\phi_cP_n + (8/9)(M_u/\phi_bM_n) \leq 1.0$）

### \boxed{} 的使用規則

| 情況 | 處理方式 | 範例 |
|------|---------|------|
| L1 變數（題目給定） | 直接寫，不加框 | $P_D$、$F_y$、$L$ |
| L2 變數（需推導） | 用 `\boxed{}` 框住 | $\boxed{P_u}$、$\boxed{Z_x}$ |
| 最終目標 | 用 `\boxed{}` 框住 + 可加文字說明 | $\boxed{\phi_bM_n} \geq M_u$ |
| 某變數已在前一行求出，後面又用到 | 繼續用 `\boxed{}` 框住，提示「這個值是算出來的，不是給的」 | |

### 格式範例（SS-2015-1 梁設計）

```latex
**Step 1：需求彎矩**
$$\boxed{P_u} = 1.2 P_D + 1.6 P_L$$
$$\boxed{M_u} = \boxed{P_u} \times \frac{L}{3}$$

**Step 2：選最輕斷面**
$$\boxed{Z_x} \Rightarrow \boxed{\phi_b M_p} = 0.9 F_y \boxed{Z_x} \geq \boxed{M_u}$$

**Step 3：剪力強度**
$$\boxed{\phi_v V_n} = \phi_v \times 0.6 F_y \times \boxed{A_w} \geq V_u$$

**Step 4：活載重撓度**
$$\boxed{\delta} = \frac{23 P_L L^3}{648 E \, \boxed{I_x}} \leq \frac{L}{360}$$
```

### 概念題（無數值計算）的處理

若題目是純說明題（如 SS-2021-2），公式框改為「關鍵公式對照」：

```latex
**塊狀剪力（BSR）雙公式：**
$$\phi R_n = \phi(0.6F_y \boxed{A_{gv}} + F_u \boxed{A_{nt}}) \quad \cdots(1)$$
$$\phi R_n = \phi(0.6F_u \boxed{A_{nv}} + F_y \boxed{A_{gt}}) \quad \cdots(2)$$
$$\text{上限：} \phi(0.6F_u \boxed{A_{nv}} + F_u \boxed{A_{nt}})$$
```

框住的是需要計算面積的幾何量（$A_{gv}$、$A_{nt}$ 等）。

## 舊版格式的處理（邊界案例）

部分較舊的解析（如 SS-2024-1）使用非標準章節結構，沒有 `## 3. 解題戰略地圖` / `## 4. 步驟化詳細計算` 的節號。

遇到此情況時，依以下原則調整插入位置：

| 舊版章節名稱 | 對應 VHA 角色 |
|-------------|-------------|
| `## 題幹摘要` | L1 的來源（≈ §1） |
| `## 核心考點` | L3 的補充（≈ §2） |
| `## 解題關鍵步驟` | L2 的來源（≈ §4） |
| `## 涉及陷阱` | L3 的主要來源（≈ §3） |

**插入位置：** 在 `## 核心考點`（或等效章節）的結尾 `---` 之後，在 `## 解題關鍵步驟`（或等效章節）之前。

**辨認方式：** 若 .md 中搜尋不到 `## 4. 步驟化詳細計算過程`，則為舊版格式，改用上述規則。
