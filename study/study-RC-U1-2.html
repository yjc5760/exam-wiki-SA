③ 解題流程

P-M 互制圖計算標準步驟（A 型為主）

1

確認截面條件

讀取 b, h, f'c, fy, 配筋位置 d1/d2/d3，確認 β₁ 值（f'c ≤ 280→0.85，每增加 70 減 0.05）

2

假設 / 接受中性軸 c

題目若給 c 直接用；否則依題目條件（如 εt=0，εt=εy）求 c；或迭代假設

3

應變相容

$\varepsilon_{si} = \varepsilon_{cu}\dfrac{c - d_i}{c}$
壓為正；拉為負（$\varepsilon_{cu}=0.003$）

4

鋼筋應力

$f_{si} = E_s\varepsilon_{si}$，但 $|f_{si}| \leq f_y$
壓力筋：扣除 $0.85f'_c$（不重複計入混凝土）

5

計算各力

$C_c = 0.85f'_c \cdot a \cdot b$（$a = \beta_1 c$）
各排鋼筋淨力 $F_{si}$

6

求 Pn 與 Mn

$P_n = C_c + \sum F_{si}$
$M_n$ 各力對形心取矩

7

判斷 φ 值

查最外排拉力鋼筋 $\varepsilon_t$，決定 φ（0.65/線性/0.90）；確認 $\phi P_n \leq \phi P_{n,\max}$

8

驗算或計算偏心距

若題目給 $P_u, M_u$：確認點在 P-M 曲線內
若逆推：$e = M_n/P_n$

三個特殊控制點（考試必算）

純軸壓 P₀： $P_0 = 0.85f'_c(A_g - A_{st}) + f_y A_{st}$，$\phi P_{n,\max} = 0.80\phi P_0$（締箍）或 $0.85\phi P_0$（螺旋）

平衡點 Pb： $c_b = \dfrac{6120}{6120+f_y} \cdot d$（最外排拉力筋恰降伏，$\varepsilon_t = \varepsilon_y$）

純彎 Mn： 令 $P_n = 0$，迭代求 c，等同梁設計（少見於柱題）

④ 核心公式速查

A 型：應變相容（基本）

$$\varepsilon_{si} = \varepsilon_{cu}\frac{c - d_i}{c}, \quad \varepsilon_{cu}=0.003$$ $$f_{si} = E_s\varepsilon_{si}, \quad |f_{si}| \leq f_y$$
壓力筋淨力：$F_{si} = A_{si}(f_{si} - 0.85f'_c)$（$d_i \leq a$ 時）
拉力筋淨力：$F_{si} = A_{si} f_{si}$（$d_i > a$ 時）

A 型：軸力與彎矩強度

$$P_n = 0.85f'_c ab + \sum A_{si}f_{si,\text{net}}$$ $$M_n = 0.85f'_c ab\!\left(\frac{h}{2}-\frac{a}{2}\right) + \sum A_{si}f_{si,\text{net}}\!\left(\frac{h}{2}-d_i\right)$$
對斷面形心取矩。$a = \beta_1 c$，$a \leq h$

A 型：φ 值（締箍柱）

$\varepsilon_t$ 範圍φ 值
$\leq 0.002$0.65
$0.002 \sim 0.005$$0.65+\dfrac{\varepsilon_t-0.002}{0.003}\times0.25$
$\geq 0.005$0.90
螺旋柱：壓力控制 φ = 0.70；其餘同

A 型：最大軸力與純軸壓

$$P_0 = 0.85f'_c(A_g - A_{st}) + f_y A_{st}$$ $$\phi P_{n,\max} = \begin{cases}0.80\,\phi P_0 & \text{締箍柱} \\ 0.85\,\phi P_0 & \text{螺旋柱}\end{cases}$$
$\phi = 0.65$（締箍）或 $0.70$（螺旋）代入

A 型：平衡點

$$c_b = \frac{6120}{6120 + f_y}\cdot d \quad (f_y \text{ in kgf/cm}^2)$$ $$\text{或}\quad c_b = \frac{0.003}{0.003+\varepsilon_y}\cdot d, \quad \varepsilon_y = f_y/E_s$$
$d$ = 最外排拉力鋼筋至壓力面距離；$f_y=4200$時 $c_b = 0.593d$

C 型：圓形螺旋柱

$$\rho_{s,\min} = 0.45\left(\frac{A_g}{A_c}-1\right)\frac{f'_c}{f_{yh}}$$ $$f'_{cc} = f'_c + 4.1 f_L, \quad f_L = \frac{2A_{sp}f_{yh}}{s\cdot D_c}$$
$A_c$ = 核心面積（以外徑 $D_c$ 計）；$A_{sp}$ = 螺旋筋截面積；$s$ = 螺距

D 型：細長柱彎矩放大法

$$\delta_{ns} = \frac{C_m}{1 - P_u/(0.75P_c)} \geq 1.0$$ $$P_c = \frac{\pi^2(EI)_e}{(kl_u)^2}, \quad (EI)_e = \frac{0.4E_cI_g}{1+\beta_{dns}}$$ $$C_m = 0.6 + 0.4\frac{M_1}{M_2} \geq 0.4$$
$M_1/M_2$：同向曲度為正，反向為負；$kl_u/r \geq 34 - 12(M_1/M_2)$ 才需放大

E 型：加大柱（組合截面）

$$P_{n,\max} = 0.85f'_{c,new}(A_{g,new}-A_{st,new}) + f_{y,new}A_{st,new}$$ $$+\; 0.85f'_{c,old}(A_{g,old}-A_{st,old}) + f_{y,old}A_{st,old}$$
新舊混凝土分別計算；$\phi P_{n,\max} = 0.80\phi P_n$（締箍）；雙折線應力應變可計算降伏軸力

⑥ 高頻陷阱

考古題歸納，共 8 個核心陷阱

⚠ 壓力筋需扣除混凝土

位於壓縮區（$d_i \leq a$）的鋼筋，淨力必須扣除已計入的混凝土壓力：$F_{si} = A_{si}(f_{si} - 0.85f'_c)$。漏扣此項導致 $P_n$ 偏大。

⚠ 中排鋼筋在形心≠零力矩

$d_2 = h/2$ 時對形心的力矩臂為零，但中排鋼筋仍有對 $P_n$ 的貢獻。RC-2011-3 就考「中排鋼筋在形心」的特殊情形。

⚠ εt 依最外排拉力筋判斷

φ 值由最遠拉力鋼筋（$d_t$）的應變 $\varepsilon_t$ 決定，不是所有鋼筋平均。如果最底排鋼筋在壓縮區，則 $\varepsilon_t \leq 0$，為壓力控制。

⚠ 壓力控制 ≠ εt ≤ 0.002

新版規範（ACI 318-14+）以 $\varepsilon_y$ 為界限（非 0.002）。考試題目若用舊版則以 0.002 為準；注意題目指示的版本。

⚠ 「拉力筋應變為零」≠ 純軸壓

令 $\varepsilon_{s3} = 0$ 即 $c = d_3$（中性軸恰通過最底排鋼筋），此時 $a = \beta_1 d_3$；$\varepsilon_t=0$，$\phi=0.65$，是常見的壓力控制邊界計算題。

⚠ P-M 圖最大彎矩不在平衡點

P-M 曲線的最大 $M_n$ 不一定出現在平衡點。實際上最大彎矩通常在 $\varepsilon_t$ 略大於 0 的位置（中性軸略低於平衡）。

⚠ 細長柱 M1/M2 符號

$C_m = 0.6 + 0.4(M_1/M_2)$：柱兩端同向曲度（S形）$M_1/M_2$ 為正，反向（C形）為負。無側移構架才用此式；有側移則另法。

⚠ 螺旋柱 φPn,max 用 0.85

螺旋柱（spiral column）的最大軸力係數為 $0.85\phi P_0$，而締箍柱為 $0.80\phi P_0$。螺旋筋對韌性有貢獻，故允許較高係數。
