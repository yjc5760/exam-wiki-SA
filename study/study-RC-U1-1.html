④ 核心公式速查

🔵 單筋矩形梁

\[ a = \frac{A_s f_y}{0.85\,f'_c\,b} \]
\[ \phi M_n = \phi\,A_s f_y\!\left(d - \frac{a}{2}\right) \]
最大拉力鋼筋：\(\varepsilon_t \geq 0.004\)（延性要求）
最小配筋：\(A_{s,\min} = \max\!\left(\dfrac{0.8\sqrt{f'_c}}{f_y}b_wd,\, \dfrac{14}{f_y}b_wd\right)\)

🟣 雙筋梁

\[ \varepsilon'_s = 0.003\,\frac{c - d'}{c} \]
\[ f'_s = \min(E_s\varepsilon'_s,\, f_y) \]
\[ 0.85f'_c\,ab + A'_s(f'_s - 0.85f'_c) = A_s f_y \]
\[ M_n = C\!\left(d - \frac{a}{2}\right) + A'_s(f'_s - 0.85f'_c)(d - d') \]
⚠ Cs 必須扣除 \(0.85f'_c\)（壓力筋被應力塊佔據）

🩵 T 形梁（中性軸在腹板）

\[ C_f = 0.85f'_c(b - b_w)h_f \]
\[ A_{s2} = \frac{C_f}{f_y},\quad A_{s1} = A_s - A_{s2} \]
\[ a = \frac{A_{s1}f_y}{0.85f'_c\,b_w} \]
\[ M_n = C_f\!\left(d - \frac{h_f}{2}\right) + 0.85f'_c b_w a\!\left(d - \frac{a}{2}\right) \]
有效翼板寬 b：取 \(l_n/4\)、\(b_w + 8h_f\)、淨間距三者最小

🟠 曲率韌性（μφ）

\[ \phi_y = \frac{\varepsilon_y}{d - c_y} \qquad \phi_u = \frac{\varepsilon_{cu}}{c_u} \]
\[ \mu_\phi = \frac{\phi_u}{\phi_y} \]
降伏曲率：用彈性轉換斷面法求中性軸 \(c_y\)（鋼筋換算為 \(nA_s\)）
極限曲率：用 Whitney 應力塊求 \(c_u = a/\beta_1\)
開裂彎矩：\(M_{cr} = f_r\,S_g\)，\(f_r = 2\sqrt{f'_c}\)（kgf/cm²）
β₁ 折減：\(f'_c \leq 280\)→\(\beta_1=0.85\)；每增 70 kgf/cm² 減 0.05，最低 0.65

φ 值速查

破壞模式 條件 φ（矩形箍） φ（螺旋筋）
拉力控制 \(\varepsilon_t \geq 0.005\) 0.90 0.90
過渡段 \(0.002 < \varepsilon_t < 0.005\) \(0.65 + 0.25\dfrac{\varepsilon_t - 0.002}{0.003}\) \(0.75 + 0.15\dfrac{\varepsilon_t - 0.002}{0.003}\)
壓力控制 \(\varepsilon_t \leq 0.002\) 0.65 0.75

題型篩選：

⑥ 高頻陷阱（考古題歸納）

❌
【雙筋梁 #1】壓力鋼筋應力必須驗算，不可直接用 fy

幾乎所有雙筋梁題目（RC-2002-4、2022-1、2023-2、2024-2）都刻意讓 \(f'_s < f_y\)。 驗算流程：先假設 \(f'_s = f_y\) 求 c → 算 \(\varepsilon'_s = 0.003(c-d')/c\) → 若 \(\varepsilon'_s < \varepsilon_y\) 則 代入 \(f'_s = E_s\varepsilon'_s\) 重解二次方程。

❌
【雙筋梁 #2】Cs 要扣除混凝土佔據的那份：\(C_s = A'_s(f'_s - 0.85f'_c)\)

Whitney 應力塊已涵蓋壓力筋所在位置的混凝土壓力，所以壓力筋的淨貢獻要扣掉 \(0.85f'_c\)。 不扣會高估 Pn 或 Mn，是最常見失分點之一。

❌
【T形梁 #1】先假設 NA 在翼板，驗算 a ≤ hf 後才能用 b

若直接用腹板分解法會增加計算量。正確流程：先以有效翼板寬 b 算 a； 若 \(a > h_f\) 再分解成翼板 \(C_f\) + 腹板 \(C_w\)。

❌
【T形梁 #2】T形梁的最小鋼筋量用 bw（腹板寬），不是 b

\(A_{s,\min} = \max(0.8\sqrt{f'_c}/f_y \cdot b_w d,\; 14/f_y \cdot b_w d)\)。 RC-2021-1 就是考這個細節，誤用 b 會使最小鋼筋量大幅偏高。

❌
【φ 值】過渡段不要忘了線性內插

RC-2019-1 專門考過渡段（εt ≈ 0.003–0.004），若直接取 φ=0.9 會錯誤。 須用公式 \(\phi = 0.65 + 0.25(\varepsilon_t - 0.002)/0.003\) 計算。

❌
【β₁】f'c > 280 kgf/cm² 時 β₁ ≠ 0.85

RC-2002-4（β₁=0.80）和 RC-2014-1（複合斷面）都考了 β₁ 折減。 公式：\(\beta_1 = 0.85 - 0.05(f'_c - 280)/70\)，最低 0.65。

❌
【曲率韌性】降伏曲率用彈性法（n·As），極限曲率用 Whitney 法

RC-2014-2 和 RC-2023-1 都考完整三階段分析。 降伏前：轉換斷面（n = Es/Ec，As → nAs）求彈性中性軸深 ky； 極限時：Whitney 應力塊求 cu = a/β₁。 兩個階段用不同方法，混用會得到錯誤結果。

❌
【有效翼板寬】ln/4 是跨度的 1/4，不是梁高的 1/4

ACI 規定取三值最小：① \(l_n/4\)（淨跨度）；② \(b_w + 8h_f\)（兩側各 8hf）； ③ 相鄰梁淨距/2 + bw。常見錯誤是把 \(l_n\) 誤算為全跨 \(l\)（含支承寬）。
