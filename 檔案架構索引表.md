## 二、資料夾結構

```
exam-wiki-SA/
├── CLAUDE.md                ← 身份層（科目代碼、分工、資料流）
├── CLAUDE-SOLVE.md          ← 解題 Skill（流程、公式框架）
├── CLAUDE-CODE.md           ← 操作指令手冊（16個指令）
├── CLAUDE-SPEC.md           ← 規格層（命名、格式、完成標準）
├── README.md                ← 冷啟動導覽
├── index.html               ← 離線儀表板
├── dashboard-data.js        ← 儀表板資料快照
│
├── study/                   ← 複習講義（study 指令輸出）
│
├── raw/
│   ├── exams/               ← SA-YYYY_結構學.pdf（24份）
│   ├── json/
│   │   ├── question_index.json   ← 題目總索引
│   │   ├── concepts.json         ← 概念定義
│   │   └── syllabus_taxonomy.json ← 命題大綱分類樹
│   └── solutions/
│       ├── SA-YYYY-N/       ← 每題解析資料夾（解析後建立）
│       └── methods/         ← 解題方法論文件
│
└── wiki/
    ├── index.md             ← 主導航
    ├── by-year.md           ← 依年份索引
    ├── log.md               ← 操作日誌
    ├── problems/            ← 題目頁
    ├── concepts/            ← 概念頁
    ├── methods/             ← 方法論頁
    ├── traps/               ← 陷阱頁
    ├── philosophy/          ← 分析哲學頁
    ├── queries/             ← 查詢結果
    ├── diagnosis/           ← 題型診斷
    ├── failure-modes/       ← 常見錯誤
    ├── materials/           ← 材料特性
    └── code-ref/            ← 公式來源
```
