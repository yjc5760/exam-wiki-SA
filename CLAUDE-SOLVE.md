### 【SOLVE】解析一年考卷（在 Cowork 執行）

**觸發方式：** 在 Cowork 中開啟 `exam-wiki-SA/` 資料夾，輸入：
```
解析 [年份] 年考卷
例：解析 2018 年考卷
```

**Cowork 執行前置（自動）：**
```
自動讀取：
  - CLAUDE.md（身份層：分工、資料流、重要規則）
  - CLAUDE-SPEC.md（格式規範、命名規則、完成標準）
  - raw/exams/SA-YYYY_結構學.pdf（YYYY 為對應西元年）
  - raw/json/question_index.json（確認哪些題目已有解析）
不需要手動上傳任何檔案
```

**執行規則：**
```
【階段一：建立資料夾並等待截圖（觸發語句後立即執行）】

1. 讀取 raw/exams/SA-YYYY_結構學.pdf
2. 讀取 raw/json/question_index.json，找出尚無解析的題目
3. 為所有尚無解析的題目建立資料夾：
   → mkdir raw/solutions/SA-YYYY-N/（每題一個，已有解析者跳過）
4. 讀取考卷 PDF，瀏覽每道題目，判斷是否有附圖：
   → 有附圖的題目：列出，提醒使用者截圖並存入對應路徑：
       raw/solutions/SA-YYYY-N/SA-YYYY-N-fig-1.png
   → 無附圖的題目：標注「無需截圖」
5. 輸出截圖清單後停止，等待使用者通知「截圖完成，請開始解題」

【階段二：逐題解析（收到截圖完成通知後執行）】

6. 先讀考卷最後一頁的參考公式，確認適用公式
7. 依序解析尚無解析的題目，但每次只解一題：
   → 解完一題，存檔後停止
   → 使用者確認後說「繼續下一題」，再解下一題
   → 目的：避免單次輸出超過 32,000 token 上限
8. 每道題目輸出一份獨立的 .md 檔案：
   → 直接存入 raw/solutions/SA-XXXX-N/SA-XXXX-N.md
9. 所有數學公式使用 LaTeX 格式
10. 每份解析開頭必須包含標籤區塊
11. 判斷是否需要輸出互動圖（依「圖形觸發條件」）：
    → 直接存入 raw/solutions/SA-XXXX-N/SA-XXXX-N-[內容碼]-viz.html
```

#### 解題前置：三層掃描法（每題解題前先執行）

**核心原則：** 先見林（外在全貌）→ 再見樹（策略路徑）→ 後見土（內在意圖）

**第一層：外在掃描（2分鐘內完成）**
```
[目標] 這題最終要我交出什麼？（彎矩圖？反力？位移量？影響線？矩陣？）
[已知] 題目給了哪些數字和條件？（跨度、EI、載重、支撐條件、截面尺寸）
[待算] 從已知到目標，需要算出哪些中間參數？（固端彎矩、分配因子、不平衡彎矩...）
```

**第二層：策略掃描**
```
[邏輯] 適用哪個分析方法？（見下方「分析哲學框架」）
[陷阱] 考官可能在哪裡挖了坑？（靜不定度計算、符號約定、側移方程式、EI 差異）
[捷徑] 有沒有條件可以簡化步驟？（對稱結構、反對稱載重、EI 相同）
```

**第三層：內在掃描**
```
[核心] 這題最重要的一個觀念是什麼？（力法/位移法選擇哲學、影響線的物理意義）
[線索] 特殊用字給了什麼暗示？（「繪製彎矩圖」→需輸出 sfd-bmd-viz、「影響線」→influence-line-viz）
```

#### 分析哲學框架（解題邏輯對照）

**靜定結構分析（SA-U1-2）**
```
核心：平衡方程式（ΣF=0, ΣM=0）即可完全求解
靜不定度 = 0 → 力法/位移法均可但無需
步驟：① 判斷結構型式（簡支梁/懸臂梁/三鉸拱/桁架）
      ② 求支承反力（取矩法確認）
      ③ 依載重型式求彎矩方程式或節點法/截面法
      ④ 繪製 SFD/BMD（注意符號約定：下受拉為正）
關鍵：集中載重→剪力突變；均布載重→拋物線彎矩；
      彎矩在最大剪力為零處取極值
```

**靜不定度與穩定性判斷（SA-U1-1）**
```
桿件結構靜不定度：i = m + r - 2j（桁架）
                   i = 3m + r - 3j（剛架，含固端、內鉸）
穩定性條件：幾何不穩定（可動機構）≠ 靜定；
           需同時確認靜不定度計算與幾何配置
外部靜不定度：r - 3（平面剛架支承反力超出平衡方程數）
內部靜不定度：含多餘桿件或封閉環
```

**影響線（SA-U1-3）**
```
靜定結構：載重移動法（逐點計算）或 Müller-Breslau 原理（撤除束制 → 虛功）
靜不定結構：必用 Müller-Breslau 原理（施加對應變位，讀取彈性曲線）
關鍵：影響線縱座標 = 該點受單位力時對應量值
      最大彎矩：連續梁用 IL 面積法 × 均布載重密度
      最大剪力：集中載重在 IL 最大縱座標處
```

**最小功法 / 卡式定理（SA-U2-1）**
```
核心：對多餘力 Ri 偏微分 ∂U/∂Ri = 0（∂U/∂Δi = Pi 為載重點位移）
      U = ∫M²/(2EI)dx + ∫N²/(2EA)dx + ∫V²/(2GA)dx
步驟：① 釋放多餘束制，以 Ri 表示內力
      ② 建立 M(x) 的 Ri 函數
      ③ ∂U/∂Ri = ∫M·(∂M/∂Ri)/(EI)dx = 0 → 求 Ri
      ④ 代回求所有反力與內力
虛功法（單位力法）：δ = ∫(m·M)/(EI)dx（求位移最快）
```

**傾角變位法（SA-U2-3）**
```
基本方程（近端）：MAB = (2EI/L)(2θA + θB - 3ψ) + MFAB
                  MBA = (2EI/L)(2θB + θA - 3ψ) + MFBA
ψ = Δ/L（相對側移角）
步驟：① 識別未知量（θ, ψ）
      ② 寫出各桿端彎矩的傾角變位方程式
      ③ 依節點平衡（ΣM=0）建立方程組
      ④ 若有側移：加入層剪力方程（ΣH=0 或層剪力等於外力）
      ⑤ 解方程組 → 代回求所有桿端彎矩 → 繪 BMD
固端彎矩（FEM）速查：
  均布 w：MFAB = -wL²/12，MFBA = +wL²/12
  集中 P（中點）：MFAB = -PL/8，MFBA = +PL/8
  集中 P（距 A 端 a）：MFAB = -Pab²/L²，MFBA = +Pa²b/L²
```

**彎矩分配法（SA-U2-5）**
```
分配因子：DFij = (EI/L)ij / Σ(EI/L)  （固定遠端：剛度 = 4EI/L）
                                         （鉸接遠端：剛度 = 3EI/L，無傳遞）
傳遞因子：遠端固定 = 1/2；遠端鉸接 = 0
步驟：① 計算各桿固端彎矩（FEM）
      ② 計算各節點分配因子（ΣDF=1）
      ③ 釋放節點（分配不平衡彎矩）→ 傳遞至遠端
      ④ 重複至收斂（誤差 < 0.1%）
      ⑤ 各桿端彎矩 = FEM + Σ分配量 + Σ傳遞量
有側移：需加入側移修正（計算假設側移量→校正係數）
```

**矩陣位移法（SA-U2-4）**
```
局部勁度矩陣 [k]（梁元素 4×4）：
  k11=k33=12EI/L³，k22=k44=4EI/L，k12=k21=6EI/L²，k14=k41=-6EI/L²
  k13=k31=-12EI/L³，k23=k32=2EI/L
整體座標：[K] = Σ[T]ᵀ[k][T]（T 為座標轉換矩陣）
解法：{P} = [K]{Δ} → {Δ} = [K]⁻¹{P}（分割已知/未知自由度）
元素內力：{f} = [k]{d} - {fF}（固端力）
```

**諧合變位法（SA-U2-2）**
```
核心：以多餘力為未知量，利用諧合條件（位移諧合）建立方程
三彎矩方程（連續梁）：
  MAL/EI₁ + 2MB(L₁/EI₁ + L₂/EI₂) + MCL₂/EI₂ = -6Aā/EI₁L₁ - 6Bb/EI₂L₂
步驟：① 確定多餘力數目（靜不定度）
      ② 釋放多餘束制，以 Ri 表示
      ③ 計算各 Ri=1 時的影響係數 δij（柔度矩陣）
      ④ 計算外力引起的位移 Δi0
      ⑤ 諧合條件：[δ]{R} = -{Δ0}
      ⑥ 解方程組 → 求出所有 Ri
```

```
2. Cowork 讀取相關 concepts/、problems/、traps/、methods/ 頁面
3. Cowork 合成跨頁面的分析答案
4. 你說「把這個答案存進知識庫」
5. Cowork 存成 wiki/queries/[主題]-[日期].md
```

**wiki/queries/[id].md 格式：**
```markdown
# [查詢主題]

**查詢日期：** YYYY-MM-DD
**關鍵字：** `標籤1` `標籤2`
**參考來源：** [[概念頁]] · [[題目頁]]

## 問題
[你當時問的問題]

## 分析結果
[Cowork 合成的答案]

## 關鍵結論
[2-3 條最重要的洞察]
```

**命名規則：**
```
wiki/queries/moment-distribution-historical-trend-20260702.md
wiki/queries/slope-deflection-traps-20260702.md
wiki/queries/force-method-vs-displacement-method-20260702.md
```
